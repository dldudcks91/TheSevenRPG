> 각 항목은 논의 후 확정하고, 확정 시 관련 문서에 반영 + 이 문서에서 ✅ 처리한다.

## 우선순위 요약

### 즉시 확정 필요 (문서 간 모순 → 구현 시 혼선)
- L-1: 레벨업 자동 성장 유/무
- L-2: 도감 레벨 3단계 vs 5단계
- L-3: 챕터당 3스테이지+별도보스 vs 4스테이지
- L-4~6: 상태이상 효과 정의 (내부 기획 vs 페스티벌 기획서)

### 밸런싱 전 확정 필요 (수치 시뮬레이션 불가)
- A-1~3: 사망 페널티 규칙
- C-1~2: 상태이상 발동확률 + 지속시간
- B-1~3: 경직 기본 시간 + FHR 공식
- F-3: 접두/접미 동일 죄종 허용 여부
- F-4: 폭식 패널티 수치

### 구현 전 확정 필요 (코드 분기에 영향)
- E-1: 코스트 초과 시 차단 vs 페널티
- F-1: 유니크 세트포인트 죄종 소속
- G-1~2: 던전 티어 매핑 + 장비 레벨 결정 방식
- H-3: 웨이브 간 HP 처리
- I-1: 스킬 동시 발동 규칙
