## Set 1 — Beta Set

**출시:** 2019년 6월 (패치 V9.13) | **종료:** 2019년 11월
**테마:** 룬테라 진영(Faction) 정치

### 시즌 특징
- TFT 최초 공개 버전. 공식 명칭은 "Set 1" 또는 "Beta Set"
- 57명의 챔피언, 다양한 룬테라 진영 기반 시너지
- 주요 메타: 6 Noble, 6 Glacial, 9 Sorcerer, 6 Blademaster, Void/Assassin 조합
- 증강체 시스템 없음

### Origins (출신지)

| Origin | 특성 효과 |
|--------|-----------|
| Demon | 기본 공격 시 마나 소각 + 마나 회복 |
| Dragon | 마법 피해 면역 (2개 필요) |
| Exile | 인접 아군 없을 시 최대 체력 100% 보호막 |
| Glacial | 기본 공격 시 % 확률로 2초 기절 |
| Hextech | 아이템 보유 적 유닛 아이템 비활성화 |
| Imperial | 1~2명 랜덤 2배 피해 |
| Ninja | 정확히 1명 또는 4명일 때 AD 보너스 |
| Noble | 방어력+치유 보너스 (3개: 랜덤 1명, 6개: 전체) |
| Phantom | 전투 시작 시 적 1명 HP를 100으로 고정 |
| Pirate | 매 라운드 추가 골드 획득 |
| Robot | 전투 시작 시 마나 풀 충전 |
| Void | 방어력 무시 피해 |
| Wild | 공격 시 공격속도 스택 증가 |
| Yordle | 기본 공격 회피 확률 |

### Classes (직업)

Assassin, Blademaster, Brawler, Elementalist, Guardian, Gunslinger, Knight, Ranger, Shapeshifter, Sorcerer

## Set 2 — Rise of the Elements

**출시:** 2019년 11월 (패치 V9.22) | **종료:** 2020년 3월
**테마:** 원소(Element)

### 시즌 특징
- 원소(Element) 테마. Origin이 "Element"로 재명명
- **원소 헥스 시스템:** 매 게임마다 Inferno/Mountain/Ocean/Wind 4개의 원소 헥스가 랜덤 배치 — 위에 있는 유닛에게 보너스 부여
- 7골드짜리 특수 챔피언 Lux 등장 (원소별로 다른 외형/능력)
- Avatar 오리진: 보드 원소에 따라 능력이 바뀌는 챔피언들
- 주요 메타: Ocean/Mage, Inferno/Sorcerer, 6 Light, 4 Electric, Poison 조합
- 증강체 시스템 없음

### Origins/Elements (원소)

|--------|-----------|
| Cloud | 주문 재사용 대기시간 감소 |

## Set 3 — Galaxies

**출시:** 2020년 3월 (패치 10.6) | **종료:** 2020년 9월
**테마:** 우주/SF (Dark Star, Star Guardian, Mech-Pilot 스킨 세계관)

### 시즌 특징
- **갤럭시 시스템:** 매 게임 랜덤 갤럭시가 선택되어 게임 규칙 변경 (예: 초기 골드 2배, 챔피언 풀 변경 등)

- 증강체 시스템 없음
