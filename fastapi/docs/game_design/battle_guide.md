## 스테이지 런 흐름 ✅확정

### 스테이지 구성

| 스테이지 타입 | 웨이브 구성 | 총 마리 수 |
|--------------|------------|----------|
| 일반 스테이지 | [일반3+정예1] × 3웨이브 + 보스(웨이브4) 1마리 | 13마리 |
| 보스 스테이지 (챕터4번째) | [일반3+정예1] × 3웨이브 + 챕터보스(웨이브4) 1마리 | 12마리 |

- **보스 = 웨이브4** (별도 구간 아님)
- 챕터 보스 스테이지: 일반 3스테이지 클리어 후 개방되는 4번째 스테이지

### 체크포인트 & 재입장

- **체크포인트**: 웨이브 단위로 저장 (웨이브 클리어마다)
- **귀환/사망 시**: 해당 웨이브 처음(1번 몬스터)부터 재시작 — 웨이브 내부 진행은 리셋
- **HP 보존**: 귀환 시 현재 HP 그대로 유지, 재입장 시에도 유지
- **드롭 보존**: 사망 시 해당 웨이브 내 처치한 몬스터 드롭은 인벤토리 지급 유지

### HP 회복 규칙

- **웨이브 클리어 후**: 포션만 사용 가능, **자동 회복 없음**
- **전투 중**: 흡혈(색욕 세트)만 회복 수단
- 포션이 핵심 소모 자원 — 개수 제한, 상인에서 구매 (챕터3 해금)

## 기본 전투 흐름

- **1:1 완전 자동전투** (플레이어 개입 없음)
- 공격속도 쿨타임마다 자동으로 상대 공격
- HP가 0이 되면 패배
- 전투에는 애니메이션 포함

## 경직 시스템 ✅확정

- **경직 발동 조건**: 단타 피해 > 최대 HP ÷ 10 (DoT는 경직 유발 안 함)
- 경직 중에는 **평타 쿨타임이 멈춤** (공격 액션 딜레이)
- **`every_n_seconds` 스킬 타이머**: 경직과 무관하게 독립적으로 작동
- **둔기 Implicit**: 데미지 조건 무시하고 **강제 경직** 발동 + 경직 시간 추가 +N초
- **타격회복속도(FHR)**: 경직 시간을 감소시키는 방어 스탯
  - 공식: `실제 경직 시간 = 기본 경직 시간 × (1 - FHR%)` (수확체감, 100% 면역 불가)
  - 갑옷 나태 죄종 옵션으로 획득 가능
  - 공격 측(경직 시간 추가) vs 방어 측(타격회복속도) 공방 축
- 기본 경직 시간, 둔기 추가 경직 시간: 밸런싱 단계에서 확정

## 카드 스킬 발동 ✅확정 (트리거 7종)

- 카드 슬롯에 카드를 장착하면 전투 중 **조건 충족 시 자동 발동**

| 트리거 | 코드명 | 발동 조건 | 확률 |
|--------|--------|----------|------|
| 공격 시 | `on_attack` | 매 공격 액션 (명중 무관) | n% |
| 타격 시 | `on_hit` | 매 명중 | n% |
| 피격 시 | `on_being_hit` | 매 피격 | n% |
| 쿨타임 후 첫 공격 | `on_cooldown_ready` | 쿨타임 완료 후 다음 공격 | 100% |
| n번째 타격마다 | `every_n_hit` | n번 타격 누적 | 100% |
| 회피 시 | `on_evade` | 내 회피 성공 | n% |
| n초마다 | `every_n_seconds` | n초 간격, 전투 중 반복 | 100% |

> **종족별 스킬 유형**: Demon=공격스킬 / Normal=자기버프 / Undead=디버프

## 상태이상 구조 ✅확정

- **7종**, 각 챕터 테마와 1:1 연동
- 각 상태이상은 특정 빌드를 카운터하는 역할
- **발동**: 매 타격마다 고정 확률로 발동
- **중첩**: 같은 상태이상 중복 시 **갱신** (타이머 리셋). 침식만 스택형 유지
- **동시 적용**: 다른 종류의 상태이상은 동시에 적용 가능

| 상태이상 | 효과 | 지속시간 |
|---------|------|--------|
| 화상 | 체력회복 감소 | 4초 |
| 중독 | 매 초 고정 데미지 | 4초 |
| 스턴 | 공격 불가 | 1초 |
| 빙결 | 공격속도 감소 | 3초 |
| 침식 | 피격마다 방어력 영구 감소 (스택형) | 영구 |
| 매혹 | 명중률 대폭 감소 | 4초 |
| 심판 | 카드 스킬 발동 확률 0% | 2초 |

## 스탯 체계

### 기본 규칙

- **기본 스탯 5종**: 힘(Str), 민첩(Dex), 체력(Life), 운(Luck), 코스트(Cost)
- **스탯 초기값**: 각 5
- **최대 레벨**: 50
- **레벨업 시 스탯 포인트**: 5개 / 총 49 × 5 = **245개**
- **레벨업 자동 성장**: 없음 (스탯·HP 모두 자동 증가 없음)
- **기본 HP**: 100 (체력 스탯 투자로만 증가)
- **레벨 효과**: 명중/회피 판정에 LvDiff로 반영 (레벨 1차이 = ±1%)

### 세부 스탯 공식

| 세부 스탯 | 공식 |
|-----------|------|
| 최대HP | `(100 + 체력×10) × (1 + Σ아이템HP%)` |
| 공격력(ATK) | `무기베이스(min~max) × (1 + 힘×0.005 + Σ아이템공격력%)` |
| 공격속도 | `무기기본공격속도 × (1 + 민첩×0.003) × (1 + Σ아이템공속%)` |
| Acc (명중 수치) | `(민첩×5 + Σ아이템명중수치) × 0.001` |
| Eva (회피 수치) | `(운×5 + Σ아이템회피수치) × 0.001` |
| **명중 판정 확률** | `clamp(Acc/(Acc+Eva) + LvDiff×0.01 + K, 0.05, 0.95)` |
| 치명타확률 | `(운×5 + Σ아이템치확수치) × 0.001` |
| 치명타데미지 | `1.5 + Σ아이템치뎀` |
| 최종방어력 | `Σ(갑옷베이스 + 아이템방어옵션 + 세트보너스 등)` |
| 방어감소율 | `최종방어력 / (최종방어력 + 100)` |
| 마법저항 감소율 | `마법저항력 / (마법저항력 + 100)` |

> **명중 판정 상수**: LvDiff = 공격자Lv - 방어자Lv, K = 0.1
> 동레벨 동스탯 명중률 **60%**, 레벨 1차이 = ±1%, 클램프 5%~95%

> **통합 계수**: 스탯과 아이템이 같은 스케일로 합산 (민첩 1 = 아이템 명중 5)
> 내부 연산은 소수(0~1), 유저 표시는 퍼센트(%)

### 데미지 공식 ✅확정

```
최종 데미지 = ATK × (1 - 방어감소율) × 사이즈보정 × 치명타배율(발동 시)
```

- **ATK**: 합연산 — 모든 공격력% 증가가 하나의 괄호 안에서 합산
- **방어감소율**: `DEF/(DEF+100)` — 수확체감, 절대 100% 불가
- **사이즈보정**: 곱연산 — 대형↔대형 +10%, 대형↔소형 -10%, 중형 보정 없음
- **치명타배율**: 곱연산 — 발동 시 `1.5 + Σ아이템치뎀`
- **데미지 분산**: 무기 베이스 min~max 범위 내 랜덤
- **유효체력(EHP)**: `HP × (100 + DEF) / 100` — DEF 100당 HP만큼 EHP 증가 (선형)

## 전투 리포트 ✅확정

전투 종료 시 통계 리포트를 표시한다. 힌트/추천은 없음 — 숫자만 제공, 판단은 유저가.

### 표시 항목

| 항목 | 설명 |
|------|------|
| 내 총 피해량 / 받은 총 피해량 | 공방 밸런스 판단 |
| 명중률 / 회피율 | 실제 전투 수치 |
| 치명타 횟수 | 운 투자 효용 확인 |
| 경직 횟수 | FHR 필요 여부 판단 |
| 상태이상 적중 횟수 | 세트 효과 발동 확인 |
| 사망 시점 적 잔여 HP | 재도전 동기 ("얼마나 아깝게 졌는가") |

> 서버: 전투 시뮬레이션 결과에 통계 데이터를 포함하여 클라이언트에 반환
> 세부 수치/UI는 구현 단계에서 확정

## 코스트 시스템 ✅확정

장비 착용에 코스트 한도가 필요하며, 좋은 장비일수록 더 많은 코스트를 소비.

### 플레이어 최대코스트

```
최대코스트 = (레벨 × 1) + (코스트스탯 × 2)
```

### 장비 코스트 공식

```
장비 코스트 = 장비레벨 × 옵션레벨 × 등급배율
```

### 코스트 규칙
- **코스트 초과 시 장착 차단** (페널티 방식 아님)
- **스탯 리셋으로 코스트 부족해질 경우** → 장비를 순서대로 무효화 (효과 없음, 장착은 유지)
  - 무효화 순서: 무기 → 갑옷 → 투구 → 장갑 → 신발
- **코스트 감소% 옵션**: 해당 장비 자신의 코스트에만 적용
- **무장비 전투 가능**: 무기 없을 시 기본 ATK = 1~1 (밸런싱 시 확정)

> 장비레벨은 플레이어에게 숨겨지며, 레벨에 따라 뜰 수 있는 옵션 종류가 달라짐
> 크래프트 코스트가 레어보다 높은 이유: 죄종 카테고리 선택권(통제권)에 대한 프리미엄
