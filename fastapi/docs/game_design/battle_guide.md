# 전투 시스템 가이드

> 전투 흐름, 스탯 구조, 코스트 개념의 설계 원칙
> 확정 데이터: [battle_design.md](battle_design.md)

---

## 기본 전투 흐름

- **1:1 완전 자동전투** (플레이어 개입 없음)
- 공격속도 쿨타임마다 자동으로 상대 공격
- HP가 0이 되면 패배
- 전투에는 애니메이션 포함

## 경직 시스템 ✅확정

- **모든 피격**에 기본 경직 발생 (기본 경직 시간은 밸런싱 시 확정)
- 경직 중에는 공격 액션이 딜레이됨
- **둔기 Implicit**: 경직 시간 추가 +N초 (기본 경직에 가산)
- **타격회복속도(FHR)**: 경직 시간을 감소시키는 방어 스탯
  - 갑옷 나태 죄종 옵션으로 획득 가능
  - 공격 측(경직 시간 추가) vs 방어 측(타격회복속도) 공방 축

## 카드 스킬 발동

- 장비 소켓에 카드를 장착하면 전투 중 **조건 충족 시 자동 발동**

| 발동 조건 | 설명 |
|-----------|------|
| 공격 시 n% | 내가 공격 액션을 취할 때 |
| 타격(명중) 시 n% | 내 공격이 실제로 명중했을 때 |
| 피격 시 n% | 내가 공격을 맞았을 때 |

## 상태이상 구조

- **7종**, 각 챕터 테마와 1:1 연동
- 각 상태이상은 특정 빌드를 카운터하는 역할
- 상세 7종 표 → [battle_design.md](battle_design.md)

## 스탯 체계

### 기본 규칙

- **기본 스탯 5종**: 힘(Str), 민첩(Dex), 체력(Life), 운(Luck), 코스트(Cost)
- **최대 레벨**: 50
- **레벨업 시 스탯 포인트**: 5개 / 총 49 × 5 = **245개**
- **기본 HP**: 100 (레벨업 시 자동 증가 없음 → 체력 스탯 투자로만 증가)

### 세부 스탯 공식

| 세부 스탯 | 공식 |
|-----------|------|
| 최대HP | `(100 + 체력×10) × (1 + Σ아이템HP%)` |
| 공격력 | `무기기본공격력 × (1 + 힘×0.005) × (1 + Σ아이템공격력%)` |
| 공격속도 | `무기기본공격속도 × (1 + 민첩×0.003) × (1 + Σ아이템공속%)` |
| Acc (명중 수치) | `민첩×0.005 + Σ아이템명중수치` |
| Eva (회피 수치) | `운×0.001 + Σ아이템회피수치` |
| **명중 판정 확률** | `clamp(Acc/(Acc+Eva) + LvDiff×0.01 + K, 0.05, 0.95)` |
| 치명타확률 | `운×0.001 + Σ아이템치확` |
| 치명타데미지 | `1.5 + 운×0.003 + Σ아이템치뎀` |
| 방어력 감소율 | `방어력 / (방어력 + 100)` |
| 마법저항력 감소율 | `마법저항력 / (마법저항력 + 100)` |

> **명중 판정 상수**: LvDiff = 공격자Lv - 방어자Lv, K = 0.1
> 동레벨 동스탯 명중률 **60%**, 레벨 1차이 = ±1%, 클램프 5%~95%

> 계수(0.005 등)는 전투 시뮬레이션 후 튜닝 예정

## 코스트 시스템

장비 착용에 코스트 한도가 필요하며, 좋은 장비일수록 더 많은 코스트를 소비.

### 플레이어 최대코스트

```
최대코스트 = (레벨 × 1) + (코스트스탯 × 2)
```

### 장비 코스트 공식

```
장비 코스트 = 장비레벨 × 옵션레벨 × 등급배율
```

> 장비레벨은 플레이어에게 숨겨지며, 레벨에 따라 뜰 수 있는 옵션 종류가 달라짐
> 크래프트 코스트가 레어보다 높은 이유: 죄종 카테고리 선택권(통제권)에 대한 프리미엄
> 등급배율 수치 → [battle_design.md](battle_design.md)
