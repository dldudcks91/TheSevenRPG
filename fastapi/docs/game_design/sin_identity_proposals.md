# 7죄종 독창성 시스템 제안서

> 작성일: 2026-03-19
> 상태: 브레인스토밍 단계 (미확정)
> 목적: 7죄종 테마를 게임 메커니즘과 일체화하여 독창성 확보

---

## 핵심 전제

죄악의 본질은 **"달콤하지만 대가가 있는 것"**이다.
모든 게임에서 죄악을 "속성"처럼 쓰지만, 진짜 죄악다운 건 **"유혹 → 탐닉 → 대가"** 구조.
단순한 속성 스킨이 아니라, 죄악의 철학적 의미가 게임플레이로 체험되어야 한다.

### 참고 사례
- **Darkest Dungeon**: 심리 상태 = 전투 메커니즘 (스트레스 → Affliction/Virtue 분기)
- **Hades**: 신의 성격 = 전투 메커니즘 (Boon 시스템, Duo Boon 조합)
- **Slay the Spire (Watcher)**: 분노/평정 스탠스 전환 = 수도승 테마 그 자체
- **Pathfinder Wages of Sin**: 죄악에 빠질수록 강해지지만 행동 제약, 타락 비용 감소 = 중독 구조
- **Fable**: 선악/순수부패 이중 축 → 외형 변화
- **Dark Souls**: 인간성(Humanity) 시스템 — 강해지는 대가로 침입 위험

## 제안 1: 타락도 (Corruption Gauge)

죄종 스킬트리에 투자할수록 해당 죄종의 "타락도"가 올라간다.
타락도가 높을수록 해당 죄종 효과가 강해지지만, 일정 구간마다 대가가 발생한다.

| 타락도 | 효과 | 대가 |
|--------|------|------|
| 0~30% | 기본 효과 | 없음 |
| 30~60% | 효과 강화 | 해당 죄종의 "성향"이 전투에 영향 |
| 60~90% | 대폭 강화 | 다른 죄종 효과 약화 |
| 90%+ | 극한 강화 | 치명적 약점 발생 |

예시: 분노 타락도 90% → 공격력 극강, 하지만 HP가 매턴 감소 (분노에 의해 자기 몸을 태움)

**독창성**: "강해지는 게 곧 위험해지는 것". 단순 스킬트리가 아니라 **얼마나 깊이 빠질 것이냐**의 선택.

---

## 제안 2: 죄종 각성 (Sin Awakening)

타락도가 특정 임계치를 넘으면 전투 중 "각성" 상태 진입.
각성 = 해당 죄종의 극한 형태. 전투 규칙이 바뀐다.

| 죄종 | 각성 효과 | 각성 조건 |
|------|----------|----------|
| 분노 | **폭주** — 공격만 가능, 방어/회피 불가, ATK 3배 | HP 30% 이하 |
| 나태 | **무심** — 5초간 무적, 이후 행동 불가 3초 | 경직 3회 연속 피격 시 |
| 색욕 | **지배** — 다음 3타를 적이 자해 | 흡혈 누적량 일정 이상 |
| 시기 | **질투** — 적 최고 스탯을 자신 것으로 복사 | 적 버프 3개 이상 제거 시 |
| 오만 | **심판** — 다음 1타에 전투 중 누적 피해 전부 반환 | HP 100% 유지 10초 |
| 폭식 | **포식** — 킬 시 적 스탯 영구 흡수 | 킬 스택 10 이상 |
| 탐욕 | **잭팟** — 다음 3타 피해 0~5배 랜덤 | 연속 치명타 3회 |

**독창성**: 죄종마다 진입 조건과 효과가 완전히 다르고, 타락도로 해금. 빌드의 최종 목표가 됨.

---

## 제안 3: 유혹 시스템 (Temptation)

전투 중 랜덤으로 "유혹"이 나타난다.
수락하면 즉시 강해지지만, 대가를 치른다.
거절하면 아무 일도 안 일어난다.

| 유혹 예시 | 보상 | 대가 |
|----------|------|------|
| "이 적의 생명력을 빼앗겠는가?" | HP 전부 회복 | 다음 전투 흡혈 불가 |
| "분노를 해방하겠는가?" | 3턴간 ATK 2배 | 이후 5턴 공격 불가 |
| "적의 보물을 탐하는가?" | 드롭률 3배 | 이번 런 경험치 0 |

**독창성**: 죄악의 본질 = 유혹. 자동전투 게임에서 **플레이어가 능동적으로 개입하는 유일한 순간**. "탐닉할 것인가, 절제할 것인가".

---

## 제안 4: 미덕 대척점 (Virtue Counter-system)

7대 죄악에 대응하는 7대 미덕이 존재.
죄악에 빠지지 않는 플레이(유혹 거절, 타락도 낮게 유지)가 미덕 수치를 올린다.

| 죄악 | 미덕 | 미덕 효과 |
|------|------|----------|
| 분노 | 인내 | 경직 시간 감소, 받는 피해 감소 |
| 나태 | 근면 | 파밍 보상 증가, 경험치 보너스 |
| 색욕 | 순결 | 상태이상 저항, 디버프 면역 |
| 시기 | 친절 | 아군 효과 강화 (PVP/연맹 시) |
| 오만 | 겸손 | 적 레벨 높을수록 명중률 보정 |
| 폭식 | 절제 | 포션 효율 증가, HP 회복 보너스 |
| 탐욕 | 자선 | 상인 가격 할인, NPC 보상 증가 |

**독창성**: 죄악 올인이 정답이 아니게 만드는 장치. "미덕 빌드"라는 완전히 다른 플레이 방향이 생김.

## 제안 5: 죄종 공명 (Sin Resonance)

스킬트리 + 세트 + 카드가 같은 죄종으로 일정 수준 이상 모이면 "공명" 효과 발동.
단순 합산이 아니라 시너지 폭발.

예시: 분노 공명 조건 — 분노 세트 4+ / 분노 트리 Lv5+ / Demon 카드 2+

```
공명 효과: "끝없는 분노"
→ HP가 1% 줄어들 때마다 ATK +0.5% (최대 50%)
→ 분노 세트(잃은체력 비례 공격력)와 곱연산 = 시너지 폭발
```

**독창성**: 단일 죄종 올인에 숨겨진 보너스. 장비+트리+카드를 한 방향으로 모으는 장기 목표. Hades Duo Boon처럼 시스템 간 조합이 새로운 효과를 만듦.

---

## 제안 6: 죄악의 속삭임 (Whispers)

투자한 죄종에 따라 게임 내 텍스트/선택지가 달라진다.

예시:
- 분노 타락도 50%+: 보스 전투 전 *"저놈을 찢어버려. 자비 따위 필요 없어."*
- 탐욕 타락도 50%+: 보스 드롭 후 *"더... 더 가져. 이것으론 부족해."*
- 나태 타락도 50%+: 웨이브 시작 시 *"굳이 싸워야 하나? 그냥 돌아가."*

**독창성**: 메커니즘이 아니라 분위기와 몰입. 캐릭터가 죄악에 물들어가는 과정을 텍스트로 체감. 기능적 영향 없이도 게임 정체성을 강하게 만듦. 구현 비용 낮음.

---

## 제안 7: 죄종 상극 (Sin Opposition)

특정 죄종 조합은 서로 약화시킨다.
둘 다 투자하면 양쪽 다 약해진다.

| 상극 관계 | 이유 |
|----------|------|
| 분노 ↔ 나태 | 폭발 vs 지구전, 양립 불가 |
| 색욕 ↔ 시기 | 흡수 vs 박탈, 방향 반대 |
| 오만 ↔ 탐욕 | 완벽 vs 도박, 철학 반대 |
| 폭식 = 독립 | 누구와도 상극 없음 (대신 자체 패널티) |

**독창성**: 7개 전부 찍는 "만능 빌드"를 구조적으로 불가능하게 만듦. "어디에 안 투자할 것인가"가 빌드 선택의 핵심.

---

## 제안 8: 원죄 선택 (Original Sin)

캐릭터 생성 시 (또는 특정 레벨 도달 시) 7죄종 중 하나를 "원죄"로 선택.
원죄 = 해당 죄종 트리의 최종 노드 해금 조건. 다른 죄종은 최종 노드까지 갈 수 없다.

- 원죄 분노 → 분노 트리 끝까지 가능 + 분노 각성 해금
- 다른 죄종은 중간까지만 투자 가능 (하이브리드용)
- 원죄 변경 = 매우 비싼 비용 또는 불가

**독창성**: "베이직 죄종"의 상위 개념. 단순 +1 세트포인트가 아니라 캐릭터의 근본 방향을 결정. 전직 시스템의 역할까지 흡수 가능.

---

## 제안 9: 죄악 전염 (Sin Contamination)

투자한 죄종이 전투 외 시스템에도 영향을 미친다.

| 죄종 | 전염 효과 |
|------|----------|
| 분노 | 전투 속도↑, 파밍 효율↑, 사망 확률↑ |
| 나태 | 전투 속도↓, 안전성↑, 경험치 드롭 보정↑ |
| 색욕 | 보스전 보상↑, 잡몹 보상↓ |
| 시기 | 다른 유저가 얻은 아이템 정보 표시 (거래소 연동?) |
| 오만 | 자기보다 높은 레벨 상대 보너스↑, 낮은 레벨 상대 보너스↓ |
| 폭식 | 드롭 수량↑, 드롭 품질↓ |
| 탐욕 | 골드↑, 장비↓ |

**독창성**: 빌드 선택이 전투뿐 아니라 파밍 방식까지 바꿈. 게임 템포 자체가 죄종에 따라 달라짐.

---

## 제안 10: 죄악의 시련 (Sin Trial)

각 챕터 보스 클리어 후, 해당 죄종의 "시련"이 해금된다.
시련 = 특수 조건 던전. 클리어하면 해당 죄종 트리의 강력한 노드 해금.

| 시련 | 조건 | 보상 |
|------|------|------|
| 분노의 시련 | HP 회복 불가 상태로 보스 처치 | 분노 트리 궁극기 해금 |
| 나태의 시련 | 10분 안에 클리어 (나태 빌드에게 불리한 시간제한) | 나태 궁극기 해금 |
| 탐욕의 시련 | 드롭 아이템 전부 포기한 채 클리어 | 탐욕 궁극기 해금 |
| 오만의 시련 | 한 대도 안 맞고 클리어 | 오만 궁극기 해금 |

**독창성**: 시련 조건이 해당 죄종의 철학적 의미를 반영. 분노 = 죽을 각오로 싸움, 탐욕 = 소유 포기. 게임플레이로 죄악의 의미를 체험.

## 조합 구조 (전부 채택 시)

```
원죄 선택 (8) → 해당 죄종 트리 풀 해금
    ↓
스킬트리 투자 → 타락도 상승 (1)
    ↓
타락도 높아지면 → 각성 해금 (2) + 속삭임 (6) + 전염 효과 (9)
    ↓
전투 중 유혹 등장 (3) → 수락하면 타락도↑, 거절하면 미덕↑ (4)
    ↓
죄종 올인하면 공명 (5), 혼합하면 상극 주의 (7)
    ↓
챕터 보스 후 시련 도전 (10) → 궁극기 해금
```

전부 넣을 필요는 없으며, 이 중에서 게임에 맞는 것을 골라서 조합한다.
