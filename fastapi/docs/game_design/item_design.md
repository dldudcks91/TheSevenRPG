## 스폰 등급별 드롭 규칙

| 스폰 등급 | 등장 장소 | 장비 드롭 | 기타 드롭 |
|-----------|-----------|-----------|-----------|
| 일반 | 스테이지 전반 | X | 골드, 강화 재료, 카드(극악 확률) |
| 정예 | 스테이지 중반 | O (매직/레어) | 골드, 강화 재료, 카드 |
| 스테이지 보스 | 스테이지 마지막 | O (레어/유니크) | 골드, 강화 재료 |
| 챕터 보스 | 챕터 클리어 후 | O (챕터 전용 유니크) | 골드 |

## 파밍 흐름도

```
일반몹 처치
  ├─ 골드             → 재화 축적
  ├─ 강화 재료 A/B    → 장비 강화, 크래프팅 소모
  └─ 카드 (극악 확률)  → 장기 수집 목표

정예몹 처치
  ├─ 위 일반몹 드롭 +
  ├─ 매직 장비        → 좋은 베이스면 크래프팅, 아니면 분해
  └─ 레어 장비        → 장착 검토 / 불필요하면 분해

던전 보스 처치
  ├─ 레어 장비        → 고품질 완성품
  └─ 유니크 장비      → 빌드 핵심 슬롯
```
