## 파밍 흐름도

```
일반몹 처치
  ├─ 골드             → 재화 축적
  ├─ 포션             → 전투 중 소비
  ├─ 광석             → 크래프팅 재료
  ├─ 퀘스트 재료       → NPC 납품 / 특정 교환
  └─ 카드 (극악 확률)  → 도감 등록 → 스킬 해금 + 패시브

정예몹 처치
  ├─ 위 일반몹 드롭 +
  ├─ 매직 장비        → 좋은 베이스면 크래프팅, 아니면 분해
  └─ 레어 장비        → 장착 검토 / 불필요하면 분해

스테이지 보스 처치
  ├─ 레어 장비        → 고품질 완성품
  └─ 유니크 장비      → 빌드 핵심 슬롯

챕터 보스 처치
  ├─ 전용 유니크 장비  → 챕터 한정 빌드 아이템
  └─ 낙인 (희귀)      → 크래프팅 시 죄종 선택 재료
```

## 아이템 소모처 매핑

```
골드       → 스탯 리셋, 상인 재료 구매, 인벤 확장, 낙인 공방 비용
장비       → 장착 / 분해 → 광석 회수
카드       → 도감 등록 → 스킬 해금 + 패시브
포션       → 전투 중 소비
광석       → 크래프팅 (+ 추후 강화 확장 가능)
낙인       → 크래프팅 시 죄종 선택 재료
퀘스트재료  → NPC 납품 → 보상 (골드/광석/영구 패시브 등)
```
