## 설계 전제 검수

| 항목 | 규칙 |
|------|------|
| 옵션 구성 | 고정 효과 + 수치 범위 1개 (완전 랜덤 아님) |
| 세트포인트 | 1 (고정 접사 1개가 해당 죄종에 기여) |
| 코스트 배율 | ×2.5 (최고) |
| 드롭처 | 스테이지 보스(레어/유니크), 챕터 보스(전용 유니크) |
| 빌드 역할 | "빌드 방향을 정의"하되, 전 슬롯 유니크는 코스트로 차단 |

### 코스트 제약에 의한 자연 밸런싱

만렙(50) + 코스트 스탯 50 기준 → **최대 코스트 = 150**
유니크 코스트 = `장비레벨 × 옵션레벨 × 2.5` → 고레벨 유니크 1~2개가 한계
→ "어떤 유니크를 낄 것인가"가 핵심 빌드 의사결정

## Chapter 1 — 분노 「불타는 전장」

### 1-1. 파멸자의 도끼 (Abaddon's Cleaver)

| 항목 | 내용 |
|------|------|
| **드롭 보스** | 아바돈 (Stage 1 보스, Demon/Orc) |
| **슬롯** | 무기 |
| **베이스** | Executioner (대형 도끼) |
| **죄종** | 분노 (+1) |
| **고유 효과** | **처형자**: 적 HP 30% 이하 시 치명타 확률 100% + 치명타 데미지 +30~50% |
| **수치 범위** | 치명타 데미지 +30~50% |

**설계 의도**
- D2 `Death Cleaver` (치명타 +66%) + D3 `Furnace` (엘리트 피해 +%)
- 아바돈 = 요한계시록의 "파괴자" → 마무리 일격 특화
- 분노 4세트(잃은 체력 비례 공격력)와 조합: 적이 피를 흘릴수록 내가 강해지고, 30% 이하에서 확정 처형
- 도끼 = Implicit 없음(순수 고데미지) + 확정 치명타 → 극적 마무리
