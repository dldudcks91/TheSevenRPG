## 레벨 구간 설계

| 구간 | 레벨 | 방식 | 목표 시간 | 비고 |
|------|------|------|----------|------|
| 1챕터 | Lv1 → Lv10 | 스토리 진행 | 1시간 | |
| 2챕터 | Lv10 → Lv20 | 스토리 진행 | 2시간 | |
| 3챕터 | Lv20 → Lv30 | 스토리 진행 | 4시간 | |
| 4챕터 | Lv30 → Lv40 | 스토리 진행 | 6시간 | 현재 구현 범위 끝 |
| 그라인드 구간 | Lv40 → Lv50 | 파밍 노가다 | 미정 | 하드코어 파밍 핵심 구간 |

- Ch1~Ch4 스토리 합계: **13시간** (Lv1→40)
- 5~7챕터는 Lv40 이상 구간으로 추후 배치
- Lv40 이후 레벨업이 매우 어렵게 설계 → 레벨 1 차이가 PvP에서 체감됨

## 스테이지 구조 ✅확정

### 스테이지 구성

| 스테이지 타입 | 구성 | 마리 수 |
|--------------|------|---------|
| 일반스테이지 | 일반 9 + 정예 3 + 스테이지보스 1 | 13마리 |
| 보스스테이지 | 일반 9 + 정예 1 + 스테이지보스 1 + 챕터보스 1 | 12마리 |

- 챕터당 **4스테이지** (일반 3 + 보스 1)
- 챕터 클리어 = 4스테이지 각 1회 클리어

### 전투 시간 목표 (동레벨 적정장비 기준)

| 등급 | 전투 시간 | 비고 |
|------|----------|------|
| 일반 | **14초** | |
| 정예 | **28초** | 일반 × 2 |
| 스테이지보스 | **70초** | 일반 × 5 |
| 챕터보스 | **300초 (5분)** | 집중전 |

> 전환 시간: 마리당 ~1.5초

## 경험치 설계 ✅확정

### 필요 XP 커브

- **기준 XP** (Lv1→2): **10**
- **스토리 구간** (Lv1~40): 레벨당 **×1.3** 증가
- **그라인드 구간** (Lv40~50): 레벨당 **×2.0** 증가

| 레벨업 | 필요 XP |
|--------|--------|
| Lv1→2 | 10 |
| Lv9→10 | 106 |
| Lv19→20 | 1,461 |
| Lv29→30 | 20,127 |
| Lv39→40 | 213,710 |
| Lv40→41 | 427,420 |
| Lv49→50 | 218,839,040 |

### 몬스터 XP

- **몬스터 XP 증가율**: 몬스터 레벨당 **×1.2**
- **등급별 XP 비율**: 일반 1x / 정예 3x / 스테이지보스 10x / 챕터보스 20x
- 일반스테이지 1런당 XP: `9×1 + 3×3 + 1×10 = 28 단위`
- 보스스테이지 1런당 XP: `9×1 + 1×3 + 1×10 + 1×20 = 42 단위`
- **챕터 1클리어** (4런): `28×3 + 42 = 126 단위`

### 챕터별 런 수 검증

> 몬스터 평균 레벨을 챕터 중간값으로 가정, 1클리어 = 24분 기준

| 챕터 | 레벨 | 필요 XP | 1클리어 XP | 필요 런 수 | 파밍 시간 | 목표 시간 |
|------|------|---------|-----------|----------|----------|----------|
| Ch1 | 1→10 | 320 | 261 | 1.2회 | 29분 | 1시간 |
| Ch2 | 10→20 | 4,519 | 1,618 | 2.8회 | 67분 | 2시간 |
| Ch3 | 20→30 | 62,310 | 10,017 | 6.2회 | 2.5시간 | 4시간 |
| Ch4 | 30→40 | 858,793 | 62,030 | 13.8회 | 5.5시간 | 6시간 |

> 파밍 시간은 순수 전투 시간. 나머지 = 스토리 + 실패/재도전 + 장비 정리.
> 런 수 증가 추이: 챕터당 약 ×2.2배.
