## 1. 연맹 기본 정보

| 항목 | 내용 |
|------|------|
| 생성 조건 | Lv20 + 골드 (수치 추후 확정) |
| 최대 인원 | 15명 |
| 가입 방식 | 공개 / 승인제 / 초대제 |
| 연맹 레벨 | 1~10 |
| 연맹 이름 | 최대 12자, 중복 불가 |
| 탈퇴 패널티 | 24시간 가입 쿨타임 |

### 직급

| 직급 | 인원 | 권한 |
|------|------|------|
| 연맹장 | 1 | 모든 권한 (해산, 위임) |
| 부연맹장 | 1~2 | 가입 승인, 추방 |
| 일반 | 나머지 | 참여 |

## 2. 연맹 레벨 버프

연맹 경험치 누적으로 레벨업. 멤버 전원에게 상시 적용 (PVP 포함).
전투 스탯 직접 상승 없음 — 효율/편의 버프만.

| 연맹 Lv | 해금 버프 | 누적 효과 |
|---------|----------|----------|
| 1 | (기본) | - |
| 2 | 경험치 +2% | 경험치 +2% |
| 3 | 골드 +2% | 경험치+2, 골드+2 |
| 4 | 경험치 +2% | 경험치+4, 골드+2 |
| 5 | 드롭률 +1% | 경험치+4, 골드+2, 드롭+1 |
| 6 | 골드 +2% | 경험치+4, 골드+4, 드롭+1 |
| 7 | 경험치 +2% | 경험치+6, 골드+4, 드롭+1 |
| 8 | 드롭률 +1% | 경험치+6, 골드+4, 드롭+2 |
| 9 | 골드 +2% | 경험치+6, 골드+6, 드롭+2 |
| 10 | 드롭률 +1% | 경험치+6, 골드+6, 드롭+3 |

### 연맹 경험치 획득

| 활동 | 경험치 |
|------|--------|
| 멤버 스테이지 클리어 | 소량 |
| 연맹 레이드 참가/클리어 | 추후 확정 |
| 연맹 PVP 참가/승리 | 추후 확정 |

> 레벨별 필요 경험치, 획득량 세부 수치는 밸런싱 단계에서 확정
