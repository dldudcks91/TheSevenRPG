# TheSevenRPG 기획 변경 로그

---

## 2026-03-11 (12차 기획 대화)

### 1. 챕터 순서 재배치
- **변경 전**: 2번 나태, 4번 시기
- **변경 후**: 2번 시기(Envy), 4번 나태(Sloth)
- 최종 순서: 분노(1) → 시기(2) → 탐욕(3) → 나태(4) → 폭식(5) → 색욕(6) → 오만(7)

### 2. 지역 테마 변경 (3개 챕터)
| 챕터 | 변경 전 | 변경 후 |
|------|---------|---------|
| 3 탐욕 | 폐허 | 황금의 사막 (Golden Desert) |
| 4 나태 | 늪 | 망각의 동토 (Frozen Oblivion) — 겨울/눈보라 |
| 7 오만 | 눈/겨울 | 신의 폐허 (Ruins of God) — 하얀 대리석 궁전 |

### 3. 상태이상 4종 → 7종 확장
| 상태이상 | 영문 | 효과 | 챕터 | 카운터 대상 |
|---------|------|------|------|------------|
| 화상 | Burn | 체력회복 감소 | 1 분노 | 흡혈(색욕) 빌드 견제 |
| 중독 | Poison | 매 초 고정 데미지 | 2 시기 | 탱커 견제 |
| 스턴 | Stun | 일정 시간 공격 불가 | 3 탐욕 | 고공속 빌드 견제 |
| 빙결 | Freeze | 공격속도 감소 | 4 나태 | 공속형 빌드 견제 |
| **침식** | **Corrode** | **피격마다 방어력 영구 감소 (스택형)** | **5 폭식** | **방어형 빌드 견제** |
| **매혹** | **Charm** | **명중률 대폭 감소** | **6 색욕** | **고공격 빌드 견제** |
| **심판** | **Judge** | **카드 스킬 발동 확률 0% (스킬 봉인)** | **7 오만** | **스킬 의존 빌드 견제** |

> 굵은 글씨 = 신규 추가된 상태이상

### 4. 스테이지별 고유 이름 확정 (21스테이지)
- 기존 "N구역" 형식 → 챕터 테마에 맞는 고유 이름으로 교체
- 예: 1-1 파멸의 진영, 1-2 핏빛 교전지대, 1-3 원한의 묘지 등

### 5. 챕터별 몬스터 풀 재배정
- 챕터 순서 변경에 따라 `chapter_monster_pool.csv` 갱신
- 각 챕터에 6종 몬스터 (타입당 2종) 배정 완료

### 6. 배경 프롬프트 작성 (21스테이지)
- `fastapi/docs/prompt/background.md` 전면 재작성
- 기존 챕터별 1개 → 스테이지별 1개 (총 21개 프롬프트)
- 스테이지 진행감: 외곽 → 내부 → 심부 (깊어질수록 어둡고 위험)

### 7. 배경 이미지 분할
- `background_total.png` (2752×1536) → 개별 스테이지 이미지 분할
- 출력: `fastapi/resources/stages/bg_ch{N}_s{N}.png` (각 458×300px)
- 챕터 2 중복 이미지 발견 → `bg_ch2_dup_s1~s3` 별도 보관 (선택 필요)

### 8. 아트 디자인 색채 팔레트 갱신
- 챕터 4: 나태 → "겨울 요소" 섹션 추가 (cold white / ice blue / ash gray)
- 챕터 7: 오만 → "하얀 신전 요소" 섹션 추가 (pure white marble / corrupted gold)

### 다음 기획 우선순위 (분석 완료, 미착수)
```
[1층] 전투 기반: 몬스터 설계 + 스코어 공식 + 경험치/레벨업
[2층] 아이템 경제: 강화 재료 + 크래프팅 재료 + 코스트 수치
[3층] 빌드 시스템: 세트 보너스 상세 + 카드 옵션 + 낙인
[4층] 수치 튜닝: 스탯 계수 + 흔적 + 드롭률
```
> 1층(몬스터 설계)부터 착수 예정
