## 장비 슬롯 (5종)

| 장비 | 베이스 스탯 | 기본옵션 (타입별) | 분류 | 베이스 수 |
|------|-----------|-----------------|------|----------|
| 무기 | 공격력 | 공격력% | 5무기군 × 3사이즈 | 15개 |
| 갑옷 | 방어력 | 중갑: 방어력% / 경갑: 방어력%+마법저항력+ | 2타입 × 3사이즈 | 6개 |
| 투구 | 방어력 | 생존형: 체력% / 공격형: 데미지% | 2타입 × 3티어 | 6개 |
| 장갑 | 방어력 (소량) | 명중형: 명중률% / 속도형: 공격속도% | 2타입 × 3티어 | 6개 |
| 신발 | 방어력 (소량) | 회피형: 회피율% / 마저항형: 마법저항력% | 2타입 × 3티어 | 6개 |

> 총 **39개** 장비 베이스

## 사이즈 상성 (무기 vs 갑옷)

무기 사이즈와 상대 갑옷 사이즈에 따른 데미지 보정:
- 대형 무기: 대형유닛 +10% 데미지 / 소형유닛 -10% 데미지
- 중형 무기: 보정 없음
- 소형 무기: 대형유닛 -10% 데미지 / 소형유닛 +10% 데미지

> 갑옷 사이즈 = PvP에서 유저의 유닛 사이즈를 결정

## 베이스 레벨 시스템

- **베이스 데미지·공격속도·Implicit은 고정** (레벨 무관)
- **무기 레벨**은 기본옵션(공격력%)의 **롤 범위**를 결정
- 높은 스테이지에서 드롭될수록 높은 레벨 → 더 높은 %가 붙을 수 있음
- 성장 = 같은 베이스에서 더 높은 공격력%가 붙은 무기를 파밍
- 플랫 옵션(최소/최대 피해 +N)도 무기 레벨에 따라 티어 상승

```
예시) Gladius Lv3 → 베이스 7~10 (고정) + 공격력 +10~30%
      Gladius Lv30 → 베이스 7~10 (고정) + 공격력 +100~150%
      (공속 1.25, Implicit 치명타+10%는 동일)
```

> 방어구도 동일 패턴: 베이스 수치 고정 + 기본옵션(방어력%/체력%/공속%/회피율%)이 레벨에 따라 범위 확대

## 장비 요구 레벨

장비 착용에 **캐릭터 레벨 제한**이 적용된다. 요구 레벨 미달 시 장착 불가.

### 산정 공식

```
요구 레벨 = item_level + 등급 보정

등급 보정:
  매직:   +0
  레어:   +2
  크래프트: +2
  유니크:  +5
```

- `item_level` = 드롭 몬스터의 mlvl (스테이지 난이도에 비례)
- 등급이 높을수록 요구 레벨이 높아 → 고급 장비일수록 레벨업 동기 강화
- 코스트 시스템과 **역할 분리**: 요구 레벨 = "언제 낄 수 있는가" (수직 성장), 코스트 = "뭘 낄 것인가" (수평 선택)

### 챕터별 예상 요구 레벨 범위

| 챕터 | 추정 mlvl | 매직 요구Lv | 레어 요구Lv | 유니크 요구Lv |
|------|----------|------------|------------|-------------|
| Ch1 | 3~8 | 3~8 | 5~10 | 8~13 |
| Ch2 | 10~18 | 10~18 | 12~20 | 15~23 |
| Ch3 | 18~25 | 18~25 | 20~27 | 23~30 |
| Ch4 | 25~35 | 25~35 | 27~37 | 30~40 |

### 설계 의도

- **"좋은 장비가 떨어졌지만 못 낀다" → 레벨업 갈망 유발**
- 인벤토리에 "대기 중인 장비"가 쌓이면서 레벨업 목표가 구체적으로 생김
- 초반 스타터 무기(요구 Lv1)는 누구나 즉시 착용 가능

## Implicit (무기군 고유 능력)

무기군마다 고정된 특수 능력. 접사(Prefix/Suffix)와 별도.
- 접사는 **수치를 바꾸고**, Implicit은 **전투 규칙을 바꿈**
- 공격속도는 베이스+사이즈별 고정, 사이즈 간 속도-데미지 트레이드오프 존재
- 둔기 경직: 모든 피격에 기본 경직 발생 + 둔기는 경직 시간 추가 (상세: [battle_guide.md](battle_guide.md))

## 크래프팅 시스템

### 개념
매직 아이템(접두 OR 접미 하나)을 베이스로, **낙인**을 소모하여 비어있는 쪽의 죄종 카테고리를 결정.

```
매직 아이템 [접두: 분노] 보유
  + 오만의 낙인 (챕터 7 보스 드롭)
  + 광석 (아이템 레벨에 비례)
  + 골드
  → 크래프트 아이템 [접두: 분노] + [접미: 오만의 랜덤 옵션]
```

### 핵심 규칙
- **베이스 매직 옵션 유지**: 기존 접두/접미는 변경되지 않음
- **낙인으로 죄종 결정**: 사용한 낙인의 죄종이 추가될 접사의 죄종을 결정 → 해당 카테고리 내 옵션은 랜덤
- **재료 비용**: 낙인 1개 + 광석 (아이템 레벨에 비례) + 골드
- **낙인 획득**: 챕터 보스 희귀 드롭 (Ch1→분노, Ch2→시기, ..., Ch7→오만)
- **결과물 코스트**: 레어보다 높음 (통제권에 대한 프리미엄)
- **크래프트 전용 옵션**: 레어/유니크에서 얻을 수 없는 옵션 존재 (세부 미정)

## 세트 보너스 시스템

### 세트포인트 구조

장비의 접두/접미사 각각이 해당 죄종에 세트포인트 1씩 기여.

| 등급 | 세트포인트/슬롯 | 비고 |
|------|---------------|------|
| 매직 | 1 | 접두 OR 접미 하나 |
| 레어 | 2 | 접두 + 접미 각각 1 |
| 크래프트 | 2 | 접두 + 접미 각각 1 |
| 유니크 | 1 | 고정 접사 하나 |

- **5슬롯** 기준 최대 세트포인트: 레어/크래프트 풀착용 시 **10포인트**
- **죄종 레벨 상한**: 7레벨 (초기 구현은 5레벨)
- **유니크 선택 시 트레이드**: 해당 슬롯 세트포인트 -1 + 코스트 증가 → 고유 효과 획득

### 빌드 전략 갈래

```
7레벨 극대화 빌드: 한 죄종 7포인트 집중 → 각성 효과 확보 + 나머지 3포인트 분배
4+3 분산 빌드:    두 죄종의 중간 단계 보너스를 동시 활용 (각성 없음)
유니크 활용 빌드: 유니크 1~2개 + 세트포인트 희생 → 고유 효과와 세트 조합
```

> 각성(6레벨) 보너스의 강도가 "집중 vs 분산" 메타를 결정하는 핵심 변수
