## 장비 슬롯 (5종)

| 장비 | 기본 능력치 | 사이즈 구분 | 베이스 수 |
|------|------------|-----------|----------|
| 무기 | 공격력 | 소형/중형/대형 | 15개 (5무기군 × 3사이즈) |
| 갑옷 | 방어력 | 소형/중형/대형 | 6개 (3사이즈 × 2종) |
| 투구 | 방어력+ | 없음 | 6개 |
| 장갑 | 명중률+ | 없음 | 6개 |
| 신발 | 회피율+ | 없음 | 6개 |

> 총 **39개** 장비 베이스

## 사이즈 상성 (무기 vs 갑옷)

무기 사이즈와 상대 갑옷 사이즈에 따른 데미지 보정:
- 대형 무기: 대형유닛 +10% 데미지 / 소형유닛 -10% 데미지
- 중형 무기: 보정 없음
- 소형 무기: 대형유닛 -10% 데미지 / 소형유닛 +10% 데미지

> 갑옷 사이즈 = PvP에서 유저의 유닛 사이즈를 결정

## 베이스 레벨 시스템

- 같은 베이스라도 **드롭 시 무기 레벨**에 따라 기본 데미지/방어력이 달라짐
- 공격속도, Implicit은 베이스별 고정 (레벨 무관)
- 높은 스테이지에서 드롭될수록 높은 레벨
- 성장 = 더 높은 레벨의 같은 베이스가 드롭되기를 파밍

```
예시) Gladius Lv3 → 기본 데미지 10~15
      Gladius Lv30 → 기본 데미지 60~80
      (공속, Implicit은 동일)
```

## Implicit (무기군 고유 능력)

무기군마다 고정된 특수 능력. 접사(Prefix/Suffix)와 별도.
- 접사는 **수치를 바꾸고**, Implicit은 **전투 규칙을 바꿈**
- 공격속도는 베이스+사이즈별 고정, 사이즈 간 속도-데미지 트레이드오프 존재

## 크래프팅 시스템

### 개념
매직 아이템(접두 OR 접미 하나)을 베이스로, 비어있는 쪽의 **죄종 카테고리를 직접 선택**해서 접사를 추가.

```
매직 아이템 [접두: 분노] 보유
  + 재료 소모 (아이템 레벨에 비례)
  + 접미사 죄종 선택: [오만]
  → 크래프트 아이템 [접두: 분노] + [접미: 오만의 랜덤 옵션]
```

### 핵심 규칙
- **베이스 매직 옵션 유지**: 기존 접두/접미는 변경되지 않음
- **죄종 카테고리 선택**: 7죄종 중 하나 선택 → 해당 카테고리 내 옵션은 랜덤
- **재료 비용**: 아이템 레벨에 비례. 고레벨 크래프팅일수록 재료 많이 소모
- **결과물 코스트**: 레어보다 높음 (통제권에 대한 프리미엄)
- **크래프트 전용 옵션**: 레어/유니크에서 얻을 수 없는 옵션 존재 (세부 미정)

## 세트 보너스 시스템

### 세트포인트 구조

장비의 접두/접미사 각각이 해당 죄종에 세트포인트 1씩 기여.

| 등급 | 세트포인트/슬롯 | 비고 |
|------|---------------|------|
| 매직 | 1 | 접두 OR 접미 하나 |
| 레어 | 2 | 접두 + 접미 각각 1 |
| 크래프트 | 2 | 접두 + 접미 각각 1 |
| 유니크 | 1 | 고정 접사 하나 |

- **5슬롯** 기준 최대 세트포인트: 레어/크래프트 풀착용 시 **10포인트**
- **죄종 레벨 상한**: 7레벨 (초기 구현은 5레벨)
- **유니크 선택 시 트레이드**: 해당 슬롯 세트포인트 -1 + 코스트 증가 → 고유 효과 획득

### 빌드 전략 갈래

```
7레벨 극대화 빌드: 한 죄종 7포인트 집중 → 각성 효과 확보 + 나머지 3포인트 분배
4+3 분산 빌드:    두 죄종의 중간 단계 보너스를 동시 활용 (각성 없음)
유니크 활용 빌드: 유니크 1~2개 + 세트포인트 희생 → 고유 효과와 세트 조합
```

> 각성(6레벨) 보너스의 강도가 "집중 vs 분산" 메타를 결정하는 핵심 변수
