# 초반 콘텐츠 할 일 목록

> 레퍼런스: Diablo 4, Lost Ark, Epic Seven, Path of Exile, MapleStory, Summoners War, AFK Arena, Raid Shadow Legends

## 현재 상태 진단

코어 루프(전투→드롭→장비강화)는 갖춰져 있으나 **파밍 외에 할 게 없음**.
성공한 RPG들은 초반 1~3시간 내에 5개 이상의 서로 다른 활동 축을 열어줌.

---

## A. 즉시 적용 가능 (기존 시스템 확장)

### 1. 초보자 가이드 미션 (7일 미션)

> Epic Seven 30일 미션, Summoners War 초보 미션

"다음에 뭘 해야 하지?"를 항상 알려주는 체크리스트.

| 일차 | 미션 예시 | 보상 |
|------|-----------|------|
| 1일 | 첫 전투 완료, 첫 장비 장착, 스탯 1회 배분 | 골드 + 포션 |
| 2일 | 챕터1 스테이지2 클리어, 장비 1개 판매 | 골드 + 광석 |
| 3일 | 챕터1 보스 처치, 상인에서 포션 구매 | 레어 장비 상자 |
| 4일 | 카드 1장 획득, 카드 스킬 장착 | 카드 소울 ×3 |
| 5일 | 크래프팅 1회, 도감 5종 등록 | 낙인(스티그마) ×1 |
| 6일 | 챕터2 클리어, 세트 보너스 2세트 달성 | 골드 대량 |
| 7일 | 챕터2 보스 처치, 퀘스트 1개 완료 | 유니크 장비 선택권 |

### 2. 업적(Achievement) 시스템

> Diablo 4 명성(Renown), Lost Ark 모험의 서

| 카테고리 | 업적 예시 | 보상 |
|----------|-----------|------|
| 전투 | 몬스터 100/500/1000마리 처치 | 골드, 칭호 |
| 보스 | 챕터보스 첫 클리어 (챕터별) | 스탯 포인트, 낙인 |
| 수집 | 도감 10/25/50종 완성 | 영구 스탯 보너스 |
| 장비 | 레어 장비 첫 획득, 크래프트 장비 첫 제작 | 인벤 확장권 |
| 카드 | 카드 5/10/20종 수집, 카드 레벨업 첫 성공 | 카드 소울, 골드 |
| 경제 | 골드 누적 1000/5000/10000 획득 | 광석, 포션 |
| 세트 | 2세트/4세트/6세트 보너스 첫 발동 | 스탯 포인트 |
| 죽음 | 첫 사망, 10회 사망 | 칭호 (위트있게) |

### 3. 퀘스트 다양화

> Lost Ark 우나의 일일태스크, Diablo 4 사이드퀘스트

현재 재료 납품 퀘스트만 존재 → 조건 다양화 필요.

| 퀘스트 유형 | 예시 | 보상 |
|-------------|------|------|
| 처치 퀘스트 | "불타는 전장에서 정예몬 3마리 처치" | 광석 ×10 |
| 수집 퀘스트 | "포션 재료 5개 납품" (기존) | 골드 |
| 클리어 퀘스트 | "챕터1 스테이지3 사망 없이 클리어" | 레어 카드 |
| 장비 퀘스트 | "매직 장비 3개 분해" | 광석 ×20 |
| 세트 퀘스트 | "분노 2세트 보너스 발동 상태에서 보스 처치" | 낙인 |

## B. 중기 개발 (새 시스템 필요하지만 비교적 가벼움)

### 4. 일일 콘텐츠 시스템

> Lost Ark 일일 카오스던전/가디언토벌, Epic Seven 일일미궁

| 콘텐츠 | 횟수 | 설명 |
|--------|------|------|
| 일일 퀘스트 | 3개/일 | 랜덤 3개 배정 → 전부 완료 시 보너스 보상 |
| 출석 보상 | 1회/일 | 7일 주기, 7일차에 대형 보상 |
| 일일 보스 도전 | 1~2회/일 | 일반 스테이지보다 강한 특수 보스, 별도 보상 테이블 |
| 일일 무료 크래프팅 | 1회/일 | 재료 할인 or 무료 크래프팅 1회 |

### 5. 챌린지 던전 (난이도 스케일링)

> Diablo 4 악몽던전, PoE 맵 시스템

- 클리어한 스테이지에 난이도 배율 적용 (×1.5, ×2.0, ×3.0)
- 높은 난이도 → 더 좋은 드롭률, 더 많은 경험치
- 최고 클리어 난이도 기록 → 랭킹 연동

### 6. 랭킹 보드

> Diablo 4 래더, MapleStory 랭킹

| 랭킹 종류 | 기준 |
|-----------|------|
| 전투력 랭킹 | 총 스탯 합산 |
| 스테이지 랭킹 | 최고 클리어 스테이지 × 난이도 |
| 수집가 랭킹 | 도감 완성률 |
| 보스 킬러 랭킹 | 챕터보스 최단시간 클리어 |
| 사망 횟수 랭킹 | (역순) 죽지 않고 가장 먼 스테이지 |

## C. 장기 개발 (큰 공수지만 임팩트 높음)

### 7. 메인 스토리 / 내러티브

> Diablo 4 캠페인, Final Fantasy 스토리

- 각 챕터 시작/클리어 시 컷씬(텍스트+일러스트)
- 챕터별 죄악 관련 NPC와 대화
- 보스 처치 전후 짧은 스토리 이벤트
- "왜 이 던전에 가야 하는지" 동기 부여

### 8. 주간 보스

> Lost Ark 어비스 레이드, MapleStory 주간보스

- 주 1~2회 도전 가능한 고난이도 보스
- 일반 챕터보스보다 훨씬 강력 (특수 패턴)
- 유니크 장비/특수 재화만 여기서 드롭
- 초반엔 쉬운 주간보스 1종부터 시작

### 9. 시즌/이벤트 시스템

> Diablo 4 시즌, PoE 리그

- 2~4주 단위 기간 한정 이벤트
- 특별 보상/한정 장비
- "지금 안 하면 못 얻는다" 심리로 접속률 향상

## 핵심 원칙 (성공한 RPG 공통)

1. **빠른 보상 주기** — 초반 1시간은 1~2분마다 뭔가 얻거나 해금
2. **점진적 복잡성** — 시스템을 한꺼번에 열지 않고 진행도에 맞춰 하나씩
3. **다중 진행 축** — 파밍 외에 수집/제작/퀘스트/업적 등 여러 방향의 "할 일"
4. **일일 리듬** — 매일 접속할 이유를 만드는 시간 제한 콘텐츠
5. **명확한 단기+장기 목표** — "지금 당장"과 "이번 주 안에"가 항상 존재

현재 TheSevenRPG에 가장 부족한 건 1, 3, 4번.
코어 루프 위에 **가이드 미션 + 업적 + 일일 콘텐츠**를 얹으면 "할 게 없다" 문제 크게 개선.
