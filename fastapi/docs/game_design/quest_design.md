# 퀘스트 시스템 설계

> 상태: 초안 (2026-03-18)

---

## 확정 사항

### 퀘스트 2계층 구조

| 계층 | 위치 | 특성 |
|------|------|------|
| **메인 퀘스트** | 항상 표시 | 스토리 진행, 순차적, 1회성 |
| **게시판 퀘스트** | 게시판 UI | 플레이어가 골라서 수락, 반복 가능 |

### 해금 시점
- **메인 퀘스트**: 게임 시작부터 존재
- **게시판 퀘스트**: 챕터 1 클리어 후 해금

### 챕터 1 구조
- 챕터 1은 메인 퀘스트만으로 스토리를 따라가는 구간
- 게시판 없이 메인 퀘스트가 진행 동기를 담당

### NPC 시설 해금
- 모든 NPC 시설은 챕터 1 완료 전에 해금
- 해금 순서/시점은 미정

---

## 미결정 사항

- [ ] NPC 시설 해금 순서 및 시점 (챕터 1 내 어느 시점에 어떤 시설)
- [ ] 게시판 퀘스트 동시 수락 수
- [ ] 게시판 퀘스트 갱신 방식 (고정 목록 / 일일·주간 로테이션)
- [ ] 게시판 퀘스트 종류 (납품형, 처치형, 수집형 등)
- [ ] 퀘스트 보상 설계 (골드 / 경험치 / 희귀 재료 / 전용 보상)
- [ ] 퀘스트 재료의 정체성 (범용 토큰 / 몬스터 소재 / 챕터별 재료)
- [ ] 메인 퀘스트 세부 내용 (챕터별 퀘스트 목록)
