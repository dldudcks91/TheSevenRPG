## 1. 게임 개요

- **장르**: 웹 RPG (아이템 파밍 중심)
- **테마**: 7죄종 + 중세신화
- **핵심 재미**: 아이템 파밍과 장착에 대한 밸런싱
- **컨셉**: "전투는 간단하게, 아이템 파밍을 주력으로"
- **핵심 구조**: 전투 **전** 준비(아이템·세트·스탯)가 전투 결과를 결정

## 2. 핵심 게임 루프

### 월드 구조
- **7챕터 × 3스테이지 + 챕터 보스** 구조
- 3스테이지 클리어 → **챕터 보스** 등장 (관문)
- 챕터 보스 클리어 → 다음 챕터 해금
- 스테이지 보스 21마리 + 챕터 보스 7마리 = **보스 28종**

### 스테이지 런 방식 (Story Mode)
모든 진행은 **스토리 모드** 방식으로 이루어진다. 필드 런 / 던전 런 구분 없음.

```
마을 → 스테이지 선택 → 입장 → [일반4+정예1] × 2회 → [일반4+정예1+보스1] → 마을 귀환 → 드롭 확인
```

- **스테이지 구성**: 총 16마리 (일반 12 + 정예 3 + 스테이지 보스 1)
- **웨이브 구조**: [일반4+정예1] → [일반4+정예1] → [일반4+정예1+보스1]

- **최초 입장 (스토리 모드)**: 스테이지를 순서대로 클리어해야 다음 스테이지 해금
  - **포탈 귀환 가능**: 진행 위치 저장 → 귀환 후 재입장 시 해당 몬스터부터 재개 (몬스터 체력 초기화)
  - 귀환 전 처치한 몬스터의 드롭은 이미 인벤에 지급 → 귀환 후 장착하고 복귀 가능
- **재입장 (던전 모드)**: 클리어한 스테이지 언제든 재입장하여 반복 파밍
  - **포탈 귀환 불가**: 클리어 or 사망만 탈출 가능
  - **자동 반복 ON**: 런 완료 후 즉시 재입장 (방치 파밍 효과)
  - **자동 반복 OFF**: 런 1회 후 마을 대기
  - 인벤토리 풀 시 자동 중지 → 인벤 관리가 자연스러운 접속 주기 결정
- **사망 시 (공통)**: 처치한 몬스터의 드롭은 즉시 인벤토리에 지급 (유지) → 스테이지 처음부터 재시작
- **사망 패널티**: 남은 몬스터 드롭 및 보스 드롭 기회 소멸 (체크포인트 없음)

### 챕터 보스
- 해당 챕터 3스테이지 모두 클리어 후 도전 가능
- 첫 클리어: 다음 챕터 해금 + 챕터 전용 유니크 드롭
- 재도전: 챕터 전용 유니크 반복 파밍

### 스폰 등급별 드롭 규칙

| 스폰 등급 | 등장 장소 | 장비 드롭 | 기타 드롭 |
|-----------|-----------|-----------|-----------|
| 일반 | 스테이지 전반 | X | 골드, 강화 재료, 카드(극악 확률) |
| 정예 | 스테이지 중반 | O (매직/레어) | 골드, 강화 재료, 카드 |
| 스테이지 보스 | 스테이지 마지막 | O (레어/유니크) | 골드, 강화 재료 |
| 챕터 보스 | 챕터 클리어 후 | O (챕터 전용 유니크) | 골드 |

### NPC / 거점 — 죄악의 성
중앙 허브. 챕터 클리어마다 시설이 해금된다.

| 해금 조건 | 시설 | 기능 |
|----------|------|------|
| 챕터 1 클리어 | 대장간 | 크래프팅, 강화 |
| 챕터 2 클리어 | 낙인 공방 | 낙인 삽입/분해 |
| 챕터 3 클리어 | 상인 | 재료 구매 (골드 소모처) |
| 챕터 4 클리어 | 흔적 조합소 | 흔적 현황 확인, 슬롯 조정 |

### 골드 소모처

| 소모처 | 역할 |
|--------|------|
| 인벤토리 확장 | 더 오래 방치 파밍 가능, 누적 비용 증가 |
| 스탯 리셋 | 빌드 방향 변경 (레벨 × N골드) |
| 상인 재료 구매 | 드롭 부진 시 보완 수단 |
| 옵션 수치 리롤 | 접사 조합 유지, 수치만 재랜덤 (도박적 소모) |

### 핵심 루프
```
[방치] 클리어한 스테이지 자동 반복 (인벤 풀 시 중지)
    ↓
[능동] 접속 → 드롭 확인 → 장착/크래프팅/빌드 조정
    ↓
[도전] 다음 스테이지 또는 챕터 보스 도전
    ↓
[판단] 실패 시 → 어떤 아이템이 부족한가? → 파밍 스테이지/목표 재설계
    ↓
다음 스테이지/챕터 해금 → 반복
```

## 3. 전투 시스템

### 기본 전투 흐름
- **1:1 완전 자동전투** (플레이어 개입 없음)
- 공격속도 쿨타임마다 자동으로 상대 공격
- HP가 0이 되면 패배
- 전투에는 애니메이션 포함

### 카드 스킬 발동
- 장비 소켓에 카드를 장착하면 전투 중 **조건 충족 시 자동 발동**

| 발동 조건 | 설명 |
|-----------|------|
| 공격 시 n% | 내가 공격 액션을 취할 때 |
| 타격(명중) 시 n% | 내 공격이 실제로 명중했을 때 |
| 피격 시 n% | 내가 공격을 맞았을 때 |

### 상태이상 (7종 — 챕터 테마 연동)

| 상태이상 | 영문 | 효과 | 챕터 | 카운터 대상 |
|---------|------|------|------|------------|
| 화상 | Burn | 체력회복 감소 | 1 분노 | 흡혈(색욕) 빌드 견제 |
| 중독 | Poison | 매 초 고정 데미지 | 2 시기 | 탱커 견제 |
| 스턴 | Stun | 일정 시간 공격 불가 | 3 탐욕 | 고공속 빌드 견제 |
| 빙결 | Freeze | 공격속도 감소 | 4 나태 | 공속형 빌드 견제 |
| 침식 | Corrode | 피격마다 방어력 영구 감소 (스택형) | 5 폭식 | 방어형 빌드 견제 |
| 매혹 | Charm | 명중률 대폭 감소 | 6 색욕 | 고공격 빌드 견제 |
| 심판 | Judge | 카드 스킬 발동 확률 0% (스킬 봉인) | 7 오만 | 스킬 의존 빌드 견제 |

## 4. 스탯 시스템

### 기본 스탯

- **최대 레벨**: 50
- **레벨업 시 스탯 포인트**: 5개
- **총 스탯 포인트**: 49 × 5 = **245개**
- **기본 HP**: 100 (레벨업 시 자동 증가 없음 → 체력 스탯 투자로만 증가)

| 스탯 | 영문 | 1포인트 효과 | 245 올인 시 |
|------|------|-------------|------------|
| 힘 | Strength | 공격력 +0.5% | 무기 기본공격력 × 2.225배 |
| 민첩 | Dexterity | 공격속도 +0.3%, 명중률 +0.5% | 공격속도 1.735배, 명중률 2.225배 |
| 체력 | Life | 최대HP +10 (고정) | 100 + 2,450 = **2,550 HP** |
| 운 | Luck | 치명타확률 +0.1%, 치명타데미지 +0.3%, 회피율 +0.1% | 치명타확률 +24.5%, 치명타데미지 +73.5%, 회피율 +24.5% |
| 코스트 | Cost | 최대코스트 +2 | 레벨당 자동 +1, 스탯 투자 시 +2/포인트 |

### 레벨 구간 설계

| 구간 | 레벨 | 방식 | 비고 |
|------|------|------|------|
| 1챕터 | Lv1 → ~Lv10 | 스토리 진행 | |
| 2챕터 | Lv10 → ~Lv20 | 스토리 진행 | |
| 3챕터 | Lv20 → ~Lv30 | 스토리 진행 | |
| 4챕터 | Lv30 → ~Lv35 | 스토리 진행 | 현재 구현 범위 끝 |
| 그라인드 구간 | Lv35 → Lv50 | 파밍 노가다 | 하드코어 파밍 핵심 구간 |

- 5~7챕터는 Lv35 이상 구간으로 추후 배치
- Lv35 이후 레벨업이 매우 어렵게 설계 → 레벨 1 차이가 PvP에서 체감됨

### 세부 스탯 공식

| 세부 스탯 | 공식 |
|-----------|------|
| 최대HP | `(100 + 체력×10) × (1 + Σ아이템HP%)` |
| 공격력 | `무기기본공격력 × (1 + 힘×0.005) × (1 + Σ아이템공격력%)` |
| 공격속도 | `무기기본공격속도 × (1 + 민첩×0.003) × (1 + Σ아이템공속%)` |
| Acc (명중 수치) | `민첩×0.005 + Σ아이템명중수치` |
| Eva (회피 수치) | `운×0.001 + Σ아이템회피수치` |
| **명중 판정 확률** | `clamp(Acc/(Acc+Eva) + LvDiff×0.01 + K, 0.05, 0.95)` |
| 치명타확률 | `운×0.001 + Σ아이템치확` |
| 치명타데미지 | `1.5 + 운×0.003 + Σ아이템치뎀` |
| 방어력 감소율 | `방어력 / (방어력 + 100)` |
| 마법저항력 감소율 | `마법저항력 / (마법저항력 + 100)` |

> **명중 판정 공식 상수**
> - `LvDiff` = 공격자 레벨 - 방어자 레벨
> - `K` = 0.1 (기본 명중 보정, 추후 밸런싱 시 조정)
> - 동레벨 동스탯(Acc=Eva) 기준 명중률: **60%**
> - 레벨 1 차이 = ±1% 명중률
> - 클램프: 최소 5% ~ 최대 95%

> 계수(0.005 등)는 전투 시뮬레이션 후 튜닝 예정

## 5. 코스트 시스템

장비 착용에 코스트 한도가 필요하며, 좋은 장비일수록 더 많은 코스트를 소비합니다.

### 플레이어 최대코스트
```
최대코스트 = (레벨 × 1) + (코스트스탯 × 2)

예시)
레벨 99, 코스트스탯   0 → 99 코스트
레벨 99, 코스트스탯  50 → 199 코스트
레벨 99, 코스트스탯 100 → 299 코스트
```

### 장비 코스트 공식
```
장비 코스트 = 장비레벨 × 옵션레벨 × 등급배율
```

**등급배율**

| 등급 | 배율 |
|------|------|
| 매직 | × 1.0 |
| 레어 | × 1.5 |
| 크래프트 | × 2.0 |
| 유니크 | × 2.5 |

> 일반 등급 제거됨 (강화 재료로 대체)
> 배율 수치는 시뮬레이션 후 튜닝 예정

> **장비레벨**은 플레이어에게 숨겨지며, 레벨에 따라 뜰 수 있는 옵션 종류가 달라짐
> **크래프트 코스트가 레어보다 높은 이유**: 죄종 카테고리 선택권(통제권)에 대한 프리미엄

## 6. 7죄종 시스템

각 죄종은 장비 세트의 테마이자, 접두/접미사 옵션의 방향성을 결정합니다.

| 영문 | 한국어 | 특성 방향 | 설명 |
|------|--------|-----------|------|
| Wrath | 분노 | 강력한 공격 특화 | 주체할 수 없는 분노의 감정, 복수와 증오 |
| Sloth | 나태 | 강력한 옵션 + 패널티(-옵션) 특화 | 흥미 부재, 습관적 무기력 |
| Lust | 색욕 | 아군 강화 (버프·흡혈) | 돈·힘·성에 대한 강렬한 열망 |
| Envy | 시기 | 상대 능력 약화·탈취 (디버프) | 타인의 소유·특성에 대한 분노와 갈망 |
| Pride | 오만 | 우월함 과시 | 인간을 신처럼 만들어주는 능력에 대한 도착 |
| Gluttony | 폭식 | 스택형 스탯 특화 | 지나친 탐닉과 과소비 |
| Greed | 탐욕 | 확률·운 기반 도박 | 탐욕스러운 물질 소유 추구 |

## 7. 아이템 시스템

### 아이템 종류

| 종류 | 설명 | 드롭 대상 | 상태 |
|------|------|-----------|------|
| 장비 | 영웅에 장착하는 장비 | 정예, 보스 | 설계 중 |
| 카드 | 캐릭터 슬롯에 장착 → 스킬 발동 용도 | 모든 몬스터 (극악 확률) | 설계 중 |
| 강화 재료 | 장비 강화/크래프팅 소모품 (A/B 2종) | 모든 몬스터 | 설계 중 |
| 포션 | 체력 회복 및 특수효과 | 미정 | 미기획 |

### 카드 시스템

#### 기본 구조
- 모든 몬스터는 고유한 카드 1종 보유
- 몬스터 처치 시 극악의 확률로 카드 드랍 (장기 파밍 목표)
- 카드는 **캐릭터 전용 슬롯 4개**에만 장착 가능 (장비 소켓 방식 아님)
- 카드 자체가 캐릭터 특성 레이어 → 코스트 시스템에 영향 없음

#### 각인 레벨 시스템
같은 카드를 중복 획득할수록 각인 레벨 상승.

| 각인 레벨 | 필요 장수 | 효과 |
|-----------|----------|------|
| Lv1 | 1장 | 기본 옵션 활성화 |
| Lv2 | 3장 | 옵션 수치 강화 |
| Lv3 | 10장 | 최대 효과 + 추가 효과 언락 |

> 중복 카드를 합산하는 방식 (인벤 관리 간소화)

#### 카드 옵션 방향
- **아이템에서 얻을 수 없는 고유 옵션**: 경험치 획득량, 골드 드롭률, 특정 속성 저항, 부활 등
- 카드가 장비를 보완하는 역할 → 장비와 카드 풀이 서로 겹치지 않음

#### 활성화 슬롯
- **4슬롯 고정**: 보유한 카드 중 4종만 동시 효과 발동
- "어떤 카드 4개를 선택할 것인가" 자체가 빌드 요소


### 아이템 등급

| 등급 | 획득처 | 접사 구성 | 세트포인트 | 특징 |
|------|--------|-----------|-----------|------|
| 매직 (Magic) | 필드 | 접두 OR 접미 1개 | 1 | 크래프팅 베이스. 코스트 낮음 |
| 레어 (Rare) | 필드/보스 | 접두 + 접미 2개 | 2 | 드롭 완성품. 코스트 중간 |
| 크래프트 (Craft) | 제작 | 접두 + 접미 2개 | 2 | 죄종 선택 가능 + 전용 옵션. 코스트 높음 |
| 유니크 (Unique) | 보스 전용 | 고정 1개 | 1 | 고정 효과 + 수치 범위 1개. 코스트 최고 |

> **일반 등급 제거**: 강화 재료 드롭으로 대체. 필드에서 드롭되는 모든 아이템이 의미를 가짐
> **유니크 능력치**: 완전 랜덤이 아닌 고정 효과 + 수치 범위 1개 → 희소성과 예측 가능성 동시 확보

### 등급별 역할 분담

- **매직**: 크래프팅 베이스 탐색 / 저비용 세트포인트 확보 수단
- **레어**: 메인 파밍 대상. 운이 좋으면 최고의 조합 / 코스트 대비 세트포인트 효율 좋음
- **크래프트**: 원하는 죄종 조합을 설계. 레어로 운에 맡기지 않고 통제하는 루트
- **유니크**: 고유 효과로 빌드 방향을 정의. 코스트가 높아 전 슬롯 유니크는 불가능

### 파밍 흐름

```
일반몹 처치 (필드/던전 공통)
  ├─ 골드             → 재화 축적
  ├─ 강화 재료 A/B    → 장비 강화, 크래프팅 소모
  └─ 카드 (극악 확률)  → 장기 수집 목표

정예몹 처치 (필드/던전 공통)
  ├─ 위 일반몹 드롭 +
  ├─ 매직 장비        → 좋은 베이스면 크래프팅, 아니면 분해
  └─ 레어 장비        → 장착 검토 / 불필요하면 분해

던전 보스 처치
  ├─ 레어 장비        → 고품질 완성품
  └─ 유니크 장비      → 빌드 핵심 슬롯
```

### 타겟 파밍 구조 (2축)

유저의 파밍 의사결정을 두 축으로 분리한다.

| 유저의 질문 | 결정 축 | 결정 요소 |
|------------|---------|-----------|
| "어떤 죄종 접사가 필요하지?" | **지역(필드)** | 필드마다 특정 죄종 접사 확률 편향 |
| "어떤 장비 부위가 필요하지?" | **몬스터 베이스** | 몬스터 종족별 장비 부위 드롭 가중치 |

#### 지역 → 죄종 편향
- 각 필드는 7죄종 중 하나의 테마를 가짐 (예: "분노의 숲", "나태의 늪")
- 해당 필드에서 장비 드롭 시, 테마 죄종의 접사가 높은 확률로 부여됨
- **편향이지 독점은 아님**: 테마 죄종 확률 상승, 나머지도 낮은 확률로 등장
- 한 스테이지 안에 여러 필드가 존재하여, 초반부터 복수의 죄종에 접근 가능

#### 몬스터 베이스 → 부위 편향
- 몬스터 종족(Humanoid, Beast, Skeleton 등)에 따라 특정 장비 부위 드롭률 상승
- `monster_drop_equipment.csv`에서 부위별 가중치 관리
- 유저가 체감할 수 있는 수준의 편향 필요 (균등 배분 지양)

#### 몬스터 타입(Normal/Demon/Undead)은 드롭에 영향 없음
- 몬스터 타입은 공통 접사 옵션(vs_Normal/vs_Demon/vs_Undead 피해)의 대상 분류 역할만 담당
- 드롭 편향과 전투 속성 역할을 분리하여 시스템 복잡도 관리

### 아이템 드랍 로직

```
1. 몬스터 처치
   └─ 정보: level, field-level, field-sin(지역 죄종), monster-base, grade, spawn-grade(일반/정예)
   └─ → monster-score 계산 (공식 미정)

2. 드랍 여부 결정
   └─ spawn-grade별 판정 횟수 (일반: 1회, 정예: 3회)

3. 아이템 종류 결정
   └─ grade로 드랍율 결정
   └─ 종류: gold, equip, card, material 등 (구상 중)

4. (장비) 장비 부위 결정
   └─ monster-base별 부위 가중치 참조 (타겟 파밍 축 2)

5. 베이스 품질 결정
   └─ item-base 정보 + 몬스터 점수 → 베타분포로 결정

6. 아이템 등급 결정
   └─ 몬스터 점수 기반 랜덤
   └─ 일반 등급 제거 → 해당 구간은 강화 재료 드랍으로 대체
   └─ 유니크: 특정 구역의 특정 몬스터만 드랍

7. 접두/접미사 결정
   └─ field-sin(지역 죄종) 기반 확률 편향 적용 (타겟 파밍 축 1)
   └─ 매직: 접두 OR 접미 1개, 레어: 접두 + 접미 2개, 유니크: 고정

8. 옵션 개수 결정
   └─ 매직: 1~3개, 레어: 2~6개, 유니크: 고정

9. 옵션 수치 결정
   └─ 몬스터 점수 기반 베타분포

10. 최종 아이템 생성
```

## 9. 크래프팅 시스템

### 개념

매직 아이템(접두 OR 접미 하나)을 베이스로, 비어있는 쪽의 **죄종 카테고리를 직접 선택**해서 접사를 추가하는 시스템.

```
매직 아이템 [접두: 분노] 보유
  + 재료 소모 (아이템 레벨에 비례)
  + 접미사 죄종 선택: [오만]
  → 크래프트 아이템 [접두: 분노] + [접미: 오만의 랜덤 옵션]
```

### 핵심 규칙

- **베이스 매직 옵션 유지**: 기존 접두/접미는 변경되지 않음
- **죄종 카테고리 선택**: 7죄종 중 하나 선택 → 해당 카테고리 내 옵션은 랜덤
- **재료 비용**: 아이템 레벨에 비례. 고레벨 크래프팅일수록 재료 많이 소모
- **결과물 코스트**: 레어보다 높음 (통제권에 대한 프리미엄)
- **크래프트 전용 옵션**: 레어/유니크에서 얻을 수 없는 옵션 존재 (세부 미정)

### 레어 vs 크래프트 비교

| | 레어 | 크래프트 |
|---|---|---|
| 획득 방법 | 드롭 | 제작 |
| 접두사 | 완전 RNG | 매직 베이스에서 선택 가능 |
| 접미사 | 완전 RNG | 죄종 카테고리 선택 (옵션은 RNG) |
| 코스트 | 낮음 | 높음 |
| 전용 옵션 | 없음 | 있음 (세부 미정) |
| 세트포인트 | 2 | 2 |

> 레어: 저렴하지만 두 옵션 모두 운에 맡김
> 크래프트: 비싸지만 한쪽 죄종을 내가 선택

### 크래프팅 재료

- **획득처**: 필드 드롭 (강화 재료와 구분 필요 여부 검토 중)
- **소모량**: 아이템 레벨 기반 공식 (수치 미정)

## 10. 파밍 레이어 교차 구조

> 각 파밍 레이어가 서로 독립적으로 동작하는 "병렬 구조"의 한계를 극복하기 위한 설계 방향.
> 레이어 A의 완성이 레이어 B의 전제조건이 되는 "교차 구조"를 목표로 한다.

### 교차 체인 1 — 던전 티어 → 접사 티어 (1순위, 핵심)

```
던전 티어 1~3  → 매직 장비만 드롭 (접두 OR 접미 1개)
던전 티어 4~6  → 레어 드롭 가능 (접두 + 접미)
던전 티어 7~9  → 레어 + 접사 수치 상위 티어 가능
던전 티어 10+  → 크래프트/유니크 드롭 가능
```

- 상위 던전 클리어 = 상위 접사의 전제조건
- 크래프팅 베이스 자체가 상위 던전에서만 드롭 → 하위에서 크래프팅 시도 불가
- PoE의 iLvl 구조와 동일한 역할

### 교차 체인 2 — 카드 각인 Lv3 → 세트 포인트 +1 (2순위)

```
특정 죄종 몬스터 카드 각인 Lv3 달성
    ↓
해당 카드 장착 시 → 해당 죄종 세트 포인트 +1 추가 인정
    ↓
세트 레벨이 평소보다 1 높아짐 → 세트 보너스 강화
```

- 카드 파밍 이유: "Lv3 각인이 세트 레벨에 직접 기여"
- "어떤 카드 4개를 선택할 것인가"가 세트 빌드와 연동됨

### 교차 구조 우선순위

| 순위 | 교차 체인 | 설계 난이도 | 임팩트 |
|-----|----------|------------|--------|
| 1 | 던전 티어 → 접사 티어 | 낮음 (드롭 테이블 수정) | 높음 |
| 2 | 카드 각인 Lv3 → 세트+1 | 중간 (세트 계산 로직 수정) | 높음 |

## 11. 세트 보너스 시스템

### 세트포인트 구조

장비의 접두/접미사 각각이 해당 죄종에 세트포인트 1씩 기여.

| 등급 | 세트포인트/슬롯 | 비고 |
|------|---------------|------|
| 매직 | 1 | 접두 OR 접미 하나 |
| 레어 | 2 | 접두 + 접미 각각 1 |
| 크래프트 | 2 | 접두 + 접미 각각 1 |
| 유니크 | 1 | 고정 접사 하나 |

- **5 슬롯** 기준 최대 세트포인트: 레어/크래프트 풀착용 시 **10포인트**
- **죄종 레벨 상한**: 7레벨 (초기 구현은 5레벨)
- **빌드 예시**: `3분노 + 4교만` = 7포인트 소모 (10 중 7 사용)
- **유니크 선택 시 트레이드**: 해당 슬롯 세트포인트 -1 + 코스트 증가 → 고유 효과 획득

### 빌드 전략 갈래

```
7레벨 극대화 빌드: 한 죄종 7포인트 집중 → 각성 효과 확보 + 나머지 3포인트 분배
4+3 분산 빌드:    두 죄종의 중간 단계 보너스를 동시 활용 (각성 없음)
유니크 활용 빌드: 유니크 1~2개 + 세트포인트 희생 → 고유 효과와 세트 조합
```

> 각성(6레벨) 보너스의 강도가 "집중 vs 분산" 메타를 결정하는 핵심 변수

죄종별 장비를 모을수록 레벨별 보너스 획득 (1~7레벨, 초기 구현 1~5레벨)

### 몬스터 풀 방식
- 각 스테이지는 챕터 풀에서 몬스터를 **런마다 랜덤 조합**으로 소환
- 풀에 6~8종 배치 시 런마다 조합이 달라져 반복 파밍 신선도 유지
- 베이스당 3마리(약/중/강) → **총 48마리** (`fastapi/meta_data/monster_info.csv`)

### 몬스터 데이터 구조

| 레이어 | 컬럼 | 역할 |
|--------|------|------|
| monster_type | Normal / Demon / Undead | 헌터 아이템 대상 분류 |
| monster_base | Wolf / Yeti / Troll... | 장비 부위 드롭 분포 결정 |
| monster_grade | 1~4 | 드롭 테이블 품질 결정 |

- **monster_type** → 공통 접사 옵션 `vs_[type]_damage` 대상
- **monster_base** → `monster_drop_equipment.csv`에서 부위별 드롭 가중치 참조
- **monster_grade** → `monster_drop_config.csv`에서 드롭 종류별 확률 참조

## 15. 세계 구조 & 챕터 설계

### 전체 구조
```
7챕터 × 3스테이지 = 21스테이지
각 챕터 = 스테이지 1~3 + 챕터 보스 (관문)
스테이지당 예상 소요: 2~5일
챕터 클리어 예상: 10~20일
전체 클리어 예상: 70~140일
```

### 챕터별 지역 설계

| 챕터 | 죄종 | 지역명 | 영문 | 상태이상 | 스테이지 타입 순서 |
|------|------|--------|------|---------|------------------|
| 1 | 분노 / Wrath | 불타는 전장 | Burning Battlefield | 화상 | Demon → Normal → Undead |
| 2 | 시기 / Envy | 뒤틀린 숲 | Twisted Forest | 중독 | Normal → Demon → Undead |
| 3 | 탐욕 / Greed | 황금의 사막 | Golden Desert | 스턴 | Normal → Undead → Demon |
| 4 | 나태 / Sloth | 망각의 동토 | Frozen Oblivion | 빙결 | Undead → Demon → Normal |
| 5 | 폭식 / Gluttony | 심연의 동굴 | Caverns of the Abyss | 침식 | Undead → Normal → Demon |
| 6 | 색욕 / Lust | 타락한 궁전 | Corrupted Palace | 매혹 | Demon → Undead → Normal |
| 7 | 오만 / Pride | 신의 폐허 | Ruins of God | 심판 | Demon → Normal → Undead |

> 챕터마다 스테이지 타입 순서를 다르게 설정 → 패턴 고정 방지
> 각 챕터의 상태이상이 해당 지역 테마와 직결 — 환경 자체가 위협

### 챕터별 몬스터 풀 & 챕터 보스

**챕터 1 — 분노 / 불타는 전장 (상태이상: 화상)**
- 스테이지: 파멸의 진영(Demon) → 핏빛 교전지대(Normal) → 원한의 묘지(Undead)
- 몬스터 풀: Goblin · Orc · Imp / Human · Troll / Skeleton
- 분위기: 끝나지 않는 전쟁이 대지를 갈아엎은 곳. 불길과 연기, 쓰러진 병사들의 원한
- 챕터 보스: 전장의 군주 — 분노에 물든 대장군

**챕터 2 — 시기 / 뒤틀린 숲 (상태이상: 중독)**
- 스테이지: 변형의 경계(Normal) → 독무의 심림(Demon) → 부패한 뿌리(Undead)
- 몬스터 풀: Wolf · Lizardman / Succubus · Orc / Ghost · Zombie
- 분위기: 독성비가 내리는 뒤틀린 숲. 다른 존재의 힘을 탐내는 그림자. 변형된 생물들의 땅
- 챕터 보스: 숲의 거울 — 침입자의 모습을 복사하는 시기의 화신

**챕터 3 — 탐욕 / 황금의 사막 (상태이상: 스턴)**
- 스테이지: 모래에 묻힌 폐허(Normal) → 저주받은 지하묘지(Undead) → 황금 보물고(Demon)
- 몬스터 풀: Lizardman · Golem / Skeleton · Vampire / Imp · Goblin
- 분위기: 사막에 묻힌 고대 왕국. 모래폭풍 속에 황금이 번쩍이지만 모두 저주가 걸려있음
- 챕터 보스: 황금 수호자 — 보물에 집착한 나머지 보물이 된 왕

**챕터 4 — 나태 / 망각의 동토 (상태이상: 빙결)**
- 스테이지: 얼어붙은 평원(Undead) → 빙하 요새(Demon) → 영구동결의 심부(Normal)
- 몬스터 풀: Zombie · Ghost / Goblin · Imp / Yeti · Troll
- 분위기: 모든 것이 얼어붙어 멈춘 동토. 시간이 정지한 듯 눈보라만 영원히 몰아치는 망각의 땅
- 챕터 보스: 동토의 지배자 — 움직임을 잃어버린 고대의 정령

**챕터 5 — 폭식 / 심연의 동굴 (상태이상: 침식)**
- 스테이지: 탐식의 입구(Undead) → 맥동하는 미로(Normal) → 소화의 심연(Demon)
- 몬스터 풀: Zombie · Lich / Yeti · Golem / Gargoyle · Orc
- 분위기: 끝없이 이어지는 동굴. 모든 것을 삼키는 살아있는 미로
- 챕터 보스: 심연의 입 — 모든 것을 집어삼키는 폭식의 화신

**챕터 6 — 색욕 / 타락한 궁전 (상태이상: 매혹)**
- 스테이지: 부패한 정원(Demon) → 유혹의 회랑(Undead) → 욕망의 왕좌(Normal)
- 몬스터 풀: Succubus · Imp / Vampire · Ghost / Lizardman · Troll
- 분위기: 화려하지만 부패한 궁전. 유혹과 함정, 욕망에 잠식된 하수인들
- 챕터 보스: 궁전의 여왕 — 욕망을 먹고 신이 된 색욕의 화신

**챕터 7 — 오만 / 신의 폐허 (상태이상: 심판)**
- 스테이지: 천상의 계단(Demon) → 무너진 신전(Normal) → 오만의 왕좌(Undead)
- 몬스터 풀: Gargoyle · Orc / Golem · Yeti / Lich · Vampire
- 분위기: 하얀 대리석으로 지어진 신의 궁전이 무너진 폐허. 신성한 빛이 오염되어 차갑게 빛나는 최후의 땅
- 챕터 보스: 타락한 신 — 스스로를 신이라 칭하다 추락한 오만의 끝

## 17. 아트 디자인 — 던전 색채 아이덴티티

### 기본 방향
- 계절 요소 없음. 모든 챕터는 기본적으로 **어두운 톤**을 유지.
- 단, 각 챕터는 **죄악의 색이 오염원으로 깔리는 "빛의 오염" 개념**으로 개성을 표현.
- 밝은 색채가 아닌, 어둠 위에 깔리는 특정 색조로 챕터 구분.

### 챕터별 색채 팔레트

| 챕터 | 죄종 | 주조색 | 오염 색채 |
|------|------|--------|----------|
| 1 | 분노 | 深黑 + 재 | 붉은 불길 · 잿빛 연기 |
| 2 | 시기 | 深黑 + 초록 | 독성 초록 · 뒤틀린 발광 균류 · 산성비 |
| 3 | 탐욕 | 어두운 황갈색 | 부패한 황금빛 · 모래폭풍 · 저주받은 광채 |
| 4 | 나태 | 차가운 백색 + 회색 | 눈보라 · 얼어붙은 정적 · 빙하의 푸른빛 |
| 5 | 폭식 | 흑자색 | 유기물 보라 · 붉은 맥동 균열 |
| 6 | 색욕 | 진홍 + 어두운 보라 | 퇴폐적 붉은빛 · 부패한 화려함 |
| 7 | 오만 | 순백 + 금빛 | 하얀 대리석 · 오염된 신성한 빛 · 부서진 신상 |

### 챕터 4 — 겨울 요소
- 나태 테마의 "모든 것이 멈춘 정지"를 겨울/동토로 표현.
- 유일하게 **겨울/눈 덮인** 환경을 갖는 챕터.
- 팔레트: cold white / ice blue / ash gray / deep void black

### 챕터 7 — 하얀 신전 요소
- 오만 테마의 "신을 자칭한 존재의 거처"를 하얀 대리석 궁전 폐허로 표현.
- 순백의 신성함이 오염되어 차갑게 빛나는 최후의 땅.
- 팔레트: pure white marble / corrupted gold / pale divine light / shadow black
