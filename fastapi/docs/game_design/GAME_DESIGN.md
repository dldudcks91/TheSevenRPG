## 1. 게임 개요

- **장르**: 웹 RPG (아이템 파밍 중심)
- **테마**: 7죄종 + 중세신화
- **핵심 재미**: 아이템 파밍과 장착에 대한 밸런싱
- **컨셉**: "전투는 간단하게, 아이템 파밍을 주력으로"
- **핵심 구조**: 전투 **전** 준비(아이템·세트·스탯)가 전투 결과를 결정

## 2. 핵심 게임 루프

### 월드 구조
- **7챕터 × 3스테이지 + 챕터 보스** 구조
- 3스테이지 클리어 → **챕터 보스** 등장 (관문)
- 챕터 보스 클리어 → 다음 챕터 해금
- 스테이지 보스 21마리 + 챕터 보스 7마리 = **보스 28종**

### 스테이지 런 방식
모든 진행은 단일 방식으로 이루어진다. 필드 런 / 던전 런 / 파밍 모드 구분 없음.

```
마을 → 스테이지 선택 → 입장 → 웨이브1 → 웨이브2 → 웨이브3 → 웨이브4(보스) → 마을 귀환 → 드롭 확인
```

- **스테이지 구성**: 총 13마리 (일반 9 + 정예 3 + 스테이지 보스 1)
- **웨이브 구조**: [일반3+정예1] → [일반3+정예1] → [일반3+정예1] → 보스1 (보스 = 웨이브4)
- **스폰 순서**: 종별 집중 (AAA→BBB→CCC) (상세: [monster_guide.md](monster_guide.md))
- **전투 방식**: 서버 유닛 단위 시뮬레이션 (1:1 순차 처리)

#### 체크포인트
- **웨이브 클리어마다 체크포인트 저장**
- 귀환 or 사망 시 → **현재 웨이브 처음부터 재시작**
- 웨이브 클리어 후 포션 사용 가능 (개수 제한, 수치 추후 밸런스)

#### 귀환
- **귀환 버튼**: 누르면 현재 유닛 전투가 끝난 후 마을 이동
  - 전투 중 사망 시 귀환 버튼 무효 → 그냥 사망 처리
- **HP 보존**: 귀환 시 현재 HP 그대로 유지 (battle_session 저장)
- **재입장**: 저장된 웨이브 체크포인트부터 재시작 — **웨이브 내부 진행은 리셋** (해당 웨이브 1번 몬스터부터)

#### 사망
- **드롭 유지**: 해당 웨이브 내 처치한 몬스터 드롭은 인벤토리 지급 유지
- **사망 패널티**: 현재 레벨 내 누적 경험치의 **10% 차감** (0% 하한, 레벨다운 없음, 연속 사망 누적 없음)
- **재시작**: 현재 웨이브 처음부터 재시작 (스테이지 전체 초기화 없음)

### 챕터 보스
- 해당 챕터 3스테이지 모두 클리어 후 도전 가능
- 첫 클리어: 다음 챕터 해금 + 챕터 전용 유니크 드롭
- 재도전: 챕터 전용 유니크 반복 파밍 + **낙인** 드롭 (희귀)
- 낙인: 해당 챕터 죄종의 크래프팅 재료 (예: Ch1 → 분노의 낙인)

### NPC / 거점 — 죄악의 성
중앙 허브. 챕터 클리어마다 시설이 해금된다.

| 해금 조건 | 시설 | 기능 |
|----------|------|------|
| 챕터 1 클리어 | 대장간 | 크래프팅 (매직 + 낙인 + 광석 → 크래프트), 장비 분해 |
| 챕터 2 클리어 | 퀘스트 게시판 | 퀘스트 재료 납품 → 보상 교환 |
| 챕터 3 클리어 | 상인 | 광석/포션 구매 (골드 소모처) |
| 챕터 4 클리어 | 흔적 조합소 | 흔적 현황 확인, 슬롯 조정 |

### 골드 소모처

| 소모처 | 역할 |
|--------|------|
| 스탯 리셋 | 빌드 방향 변경 (레벨 × N골드) |
| 스킬트리 리셋 | 죄종 트리 초기화 (비용 미확정) |
| 크래프팅 | 매직 + 낙인 + 광석 + 골드 → 크래프트 장비 |
| 상인 구매 | 광석/포션 구매 (드롭 부진 시 보완 수단) |
| 인벤 확장 | 인벤토리 슬롯 추가 |

### 스킬트리 (7죄종 패시브 트리)
7죄종 각각이 하나의 스킬트리 갈래. 레벨업 시 획득하는 **스킬포인트**를 원하는 죄종 트리에 투자한다.

- **트리 구조**: 7죄종 × N단계 (단계별 선행 조건)
- **스킬포인트 수급**: 레벨업 시 획득 (수치 미확정)
- **리셋**: 골드 소비로 리셋 가능 → 빌드 실험 + 골드 소모처
- **세트와의 관계**: 세트 = 장비로 맞추는 것 (파밍 의존), 트리 = 포인트로 찍는 것 (선택 의존). 같은 죄종 세트+트리 = 특화, 다른 죄종 혼합 = 하이브리드
- **효과 방향**: 단순 수치 보너스 + 규칙 변경형 혼합 (상세 미확정)

### 핵심 루프
```
[파밍] 클리어한 스테이지 재입장 → 런 → 드롭 확인
    ↓
[성장] 장착/크래프팅/스킬트리/빌드 조정
    ↓
[도전] 다음 스테이지 또는 챕터 보스 도전
    ↓
[판단] 실패 시 → 어떤 아이템이 부족한가? → 파밍 스테이지/목표 재설계
    ↓
다음 스테이지/챕터 해금 → 반복
```

## 3. 7죄종 시스템

각 죄종은 장비 세트의 테마이자, 접두/접미사 옵션의 방향성을 결정합니다.

| 영문 | 한국어 | 특성 방향 | 설명 |
|------|--------|-----------|------|
| Wrath | 분노 | 강력한 공격 특화 | 주체할 수 없는 분노의 감정, 복수와 증오 |
| Sloth | 나태 | 강력한 옵션 + 패널티(-옵션) 특화 | 흥미 부재, 습관적 무기력 |
| Lust | 색욕 | 아군 강화 (버프·흡혈) | 돈·힘·성에 대한 강렬한 열망 |
| Envy | 시기 | 상대 능력 약화·탈취 (디버프) | 타인의 소유·특성에 대한 분노와 갈망 |
| Pride | 오만 | 우월함 과시 | 인간을 신처럼 만들어주는 능력에 대한 도착 |
| Gluttony | 폭식 | 스택형 스탯 특화 | 지나친 탐닉과 과소비 |
| Greed | 탐욕 | 확률·운 기반 도박 | 탐욕스러운 물질 소유 추구 |

## 4. 파밍 레이어 교차 구조

> 각 파밍 레이어가 서로 독립적으로 동작하는 "병렬 구조"의 한계를 극복하기 위한 설계 방향.
> 레이어 A의 완성이 레이어 B의 전제조건이 되는 "교차 구조"를 목표로 한다.

### 교차 체인 1 — 몬스터 레벨 → 드롭 품질 (1순위, 핵심) ✅확정

**드롭 등급 하드 제한 폐기** → 몬스터 레벨(mlvl) 기반 확률 방식으로 변경

```
D2 구조 채용: dlvl(던전 레벨) → mlvl(몬스터 레벨) → ilvl(장비 레벨)

mlvl 보정:
  일반몹      mlvl = dlvl
  정예        mlvl = dlvl + 2
  스테이지보스 mlvl = dlvl + 3
  챕터보스    mlvl = 고정값 (cap 50)

ilvl = 드롭한 몬스터의 mlvl
```

- 모든 등급의 장비가 어느 던전에서든 드롭 가능
- mlvl이 높을수록 상위 등급(레어/크래프트/유니크) 드롭 확률 상승
- 접사 수치 상한도 ilvl 기반 qlvl 체계로 관리 (낮은 ilvl = 낮은 수치 범주만 가능)
- 상위 던전 파밍 = 상위 등급 + 상위 접사 수치 양쪽 확률 모두 상승

### 교차 구조 우선순위

| 순위 | 교차 체인 | 설계 난이도 | 임팩트 |
|-----|----------|------------|--------|
| 1 | 몬스터 레벨 → 드롭 확률 + 접사 수치 | 중간 (확률 테이블 설계) | 높음 |

## 5. 미결정 사항

### 기획 완료 ✅
- [x] 전투 방식: 1:1 완전 자동전투
- [x] 월드 구조: 필드 + 던전, 스테이지 방식
- [x] 패배 처리: 필드/던전 밖으로 이동, 아이템 유지
- [x] 최대 레벨: 50 (Lv40 이후 하드코어 그라인드 구간)
- [x] 기본 스탯 5종: 힘, 민첩, 체력, 운, 코스트
- [x] 기본 HP: 100 (스탯 투자로만 증가)
- [x] 방어력 공식: `방어력 / (방어력 + 100)`
- [x] 상태이상 7종: 화상/중독/스턴/빙결/침식/매혹/심판 (챕터 테마 연동)
- [x] 코스트 공식: `장비레벨 × 옵션레벨 × 등급배율`
- [x] 카드 시스템 확정: 카드 = 인벤토리 아이템 (카드 전용 탭), 도감 자동 등록 + 스킬 해금
- [x] 카드 수치 범주: 드롭 시 랜덤 수치, 같은 카드도 수치 다름, 더 좋은 수치 교체 가능
- [x] 카드 레벨업: 같은 카드 N장 + 카드 영혼 N개 소모 (수치는 밸런싱 단계 확정)
- [x] 카드 영혼: 카드 분해 시 획득, 범용 재료 (카드 종류 무관)
- [x] 카드 거래: 거래소 거래 가능 (추후 도입)
- [x] 스킬 시스템: 카드 장착 → 트리거 조건 충족 시 자동 발동 (7종 트리거)
- [x] 스킬 트리거 7종: on_attack / on_hit / on_being_hit / on_cooldown_ready / every_n_hit / on_evade / every_n_seconds
- [x] 종족별 스킬 유형 고정: Demon=공격스킬(딜러), Normal=자기버프(탱커/지속전), Undead=디버프/방해(컨트롤)
- [x] 스킬슬롯 4개 점진적 해금: Lv1→1슬롯, Lv5→2, Lv15→3, Lv30→4 (같은 카드 1개만 장착)
- [x] 도감: 첫 획득 시 자동 등록 + 스킬 해금 (레벨업 시스템 없음, 수집 기록 역할만)
- [x] 도감 그룹: ⬜ 재설계 필요 (기존 도감 레벨 합산 방식 폐기)
- [x] 레벨업 보상: 스탯포인트 + 마일스톤 슬롯 해금 (SP 시스템 없음, 자동성장 없음)
- [x] 카드 옵션 방향: 아이템에서 얻을 수 없는 고유 옵션 (경험치, 골드, 속성저항 등)
- [x] 아이템 등급 구조: 일반 제거 → 매직/레어/크래프트/유니크 4등급
- [x] 크래프팅 방식: 매직 베이스 + 낙인(죄종 선택) + 광석 + 골드
- [x] 세트포인트 구조: 접두/접미 각 1포인트, 5슬롯 최대 10포인트
- [x] 죄종 세트 레벨 상한: 7 (초기 구현 5)
- [x] 몬스터 타입 3종 확정: Normal(일반) / Demon(데몬) / Undead(언데드)
- [x] 타겟 파밍 3축: 지역→죄종 편향, 몬스터 베이스→부위 편향, 챕터 보스→낙인(크래프팅 죄종)
- [x] 스폰 등급별 드롭 분리: 일반몹(골드/포션/광석/퀘스트재료/카드), 정예/보스(+장비), 챕터 보스(+낙인)
- [x] 스테이지 구조: 스토리 모드 단일 진행, 클리어 후 재입장 파밍 가능
- [x] 드롭 아이템 7종: 골드/장비/카드/포션/광석/낙인/퀘스트재료
- [x] 광석: 크래프팅 범용 재료 (등급 계층형), 모든 몬스터 드롭
- [x] 낙인: 7죄종별 1종, 챕터 보스 희귀 드롭 → 크래프팅 죄종 선택 재료
- [x] 강화 시스템: 보류 (엔드 콘텐츠 부족 시 추후 추가, 광석 재료로 확장 가능)
- [x] 부적(Charm) 시스템: 제외 (기존 커스터마이징 축과 중복)
- [x] 전투 리포트: 중간 수준 (통계만 표시, 힌트 없음) — 피해량/명중률/회피율/치명타/경직/상태이상/적 잔여HP
- [x] 스테이지 진행 UI: 횡스크롤 필드 진행 + 다음 몬스터 실루엣 + 하단 진행 바 + 드롭 로그
