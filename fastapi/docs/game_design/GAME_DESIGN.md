## 1. 게임 개요

- **장르**: 웹 RPG (아이템 파밍 중심)
- **테마**: 7죄종 + 중세신화
- **핵심 재미**: 아이템 파밍과 장착에 대한 밸런싱
- **컨셉**: "전투는 간단하게, 아이템 파밍을 주력으로"
- **핵심 구조**: 전투 **전** 준비(아이템·세트·스탯)가 전투 결과를 결정

## 2. 핵심 게임 루프

### 월드 구조
- **스테이지 방식**: 던전 보스 클리어 → 다음 스테이지 해금
- 각 스테이지는 **필드**와 **던전**으로 구성

### 필드
- 지역 클릭 → 입장 → 몹 **연속 자동 스폰** (방치 파밍)
- 중간중간 **엘리트(정예) 몬스터** 등장 → 드랍 판정 3회
- 드랍: 일반 / 매직 등급 아이템
- 사망 시: 필드 밖으로 이동, **획득 아이템 유지**

### 던전
- 지역 클릭 → 입장 → 몹 **순서대로 1:1 자동전투**
- 모든 몹 처치 → **보스** 등장
- 보스 클리어 → 다음 스테이지 해금
- 드랍: 레어 / 유니크 등급 아이템
- 사망 시: 던전 밖으로 이동, **획득 아이템 유지**

### 핵심 루프
```
필드 파밍 (일반/매직 아이템)
    ↓
아이템 장착 및 빌드 최적화
    ↓
던전 도전 → 보스 클리어 (레어/유니크 아이템)
    ↓
다음 스테이지 해금 → 반복
```

## 3. 전투 시스템

### 기본 전투 흐름
- **1:1 완전 자동전투** (플레이어 개입 없음)
- 공격속도 쿨타임마다 자동으로 상대 공격
- HP가 0이 되면 패배
- 전투에는 애니메이션 포함

### 카드 스킬 발동
- 장비 소켓에 카드를 장착하면 전투 중 **조건 충족 시 자동 발동**

| 발동 조건 | 설명 |
|-----------|------|
| 공격 시 n% | 내가 공격 액션을 취할 때 |
| 타격(명중) 시 n% | 내 공격이 실제로 명중했을 때 |
| 피격 시 n% | 내가 공격을 맞았을 때 |

### 상태이상

| 상태이상 | 효과 | 카운터 대상 |
|---------|------|------------|
| 스턴 | 일정 시간 공격 불가 | 고공속 적 |
| 빙결 | 공격속도 감소 | 공속형 빌드 |
| 중독 | 매 초 고정 데미지 | 탱커 |
| 화상 | 체력회복 감소 | 흡혈(색욕 세트) |

## 4. 스탯 시스템

### 기본 스탯

- **최대 레벨**: 99
- **레벨업 시 스탯 포인트**: 5개
- **총 스탯 포인트**: 98 × 5 = **490개**
- **기본 HP**: 100 (레벨업 시 자동 증가 없음 → 체력 스탯 투자로만 증가)

| 스탯 | 영문 | 1포인트 효과 | 490 올인 시 |
|------|------|-------------|------------|
| 힘 | Strength | 공격력 +0.5% | 무기 기본공격력 × 3.45배 |
| 민첩 | Dexterity | 공격속도 +0.3%, 명중률 +0.5% | 공격속도 2.47배, 명중률 3.45배 |
| 체력 | Life | 최대HP +10 (고정) | 100 + 4,900 = **5,000 HP** |
| 운 | Luck | 치명타확률 +0.1%, 치명타데미지 +0.3%, 회피율 +0.1% | 치명타확률 +49%, 치명타데미지 +147%, 회피율 +49% |
| 코스트 | Cost | 최대코스트 +2 | 레벨당 자동 +1, 스탯 투자 시 +2/포인트 |

### 세부 스탯 공식

| 세부 스탯 | 공식 |
|-----------|------|
| 최대HP | `(100 + 체력×10) × (1 + Σ아이템HP%)` |
| 공격력 | `무기기본공격력 × (1 + 힘×0.005) × (1 + Σ아이템공격력%)` |
| 공격속도 | `무기기본공격속도 × (1 + 민첩×0.003) × (1 + Σ아이템공속%)` |
| 명중률 | `(1 + 민첩×0.005) × (1 + Σ아이템명중%)` |
| 치명타확률 | `운×0.001 + Σ아이템치확` |
| 치명타데미지 | `1.5 + 운×0.003 + Σ아이템치뎀` |
| 회피율 | `운×0.001 + Σ아이템회피` |
| 방어력 감소율 | `방어력 / (방어력 + 100)` |
| 마법저항력 감소율 | `마법저항력 / (마법저항력 + 100)` |

> 계수(0.005 등)는 전투 시뮬레이션 후 튜닝 예정

## 5. 코스트 시스템

장비 착용에 코스트 한도가 필요하며, 좋은 장비일수록 더 많은 코스트를 소비합니다.

### 플레이어 최대코스트
```
최대코스트 = (레벨 × 1) + (코스트스탯 × 2)

예시)
레벨 99, 코스트스탯   0 → 99 코스트
레벨 99, 코스트스탯  50 → 199 코스트
레벨 99, 코스트스탯 100 → 299 코스트
```

### 장비 코스트 공식
```
장비 코스트 = 베이스코스트 × 등급배율 × 레벨배율
```

**등급배율**

| 등급 | 배율 |
|------|------|
| 일반 | × 1.0 |
| 매직 | × 1.5 |
| 레어 | × 2.0 |
| 유니크 | × 3.0 |

**레벨배율**
```
레벨배율 = 1 + (장비레벨 × 0.1)
장비레벨 1  → × 1.1
장비레벨 5  → × 1.5
장비레벨 10 → × 2.0
```

> **장비레벨**은 플레이어에게 숨겨지며, 레벨에 따라 뜰 수 있는 옵션 종류가 달라짐
> **베이스코스트**(장비 종류별) 수치는 장비 설계 단계에서 확정

## 6. 7죄종 시스템

각 죄종은 장비 세트의 테마이자, 접두/접미사 옵션의 방향성을 결정합니다.

| 영문 | 한국어 | 특성 방향 | 설명 |
|------|--------|-----------|------|
| Wrath | 분노 | 강력한 공격 특화 | 주체할 수 없는 분노의 감정, 복수와 증오 |
| Sloth | 나태 | 강력한 옵션 + 패널티(-옵션) 특화 | 흥미 부재, 습관적 무기력 |
| Lust | 색욕 | 아군 강화 (버프·흡혈) | 돈·힘·성에 대한 강렬한 열망 |
| Envy | 시기 | 상대 능력 약화·탈취 (디버프) | 타인의 소유·특성에 대한 분노와 갈망 |
| Pride | 오만 | 우월함 과시 | 인간을 신처럼 만들어주는 능력에 대한 도착 |
| Gluttony | 폭식 | 스택형 스탯 특화 | 지나친 탐닉과 과소비 |
| Greed | 탐욕 | 확률·운 기반 도박 | 탐욕스러운 물질 소유 추구 |

## 7. 아이템 시스템

### 아이템 종류

| 종류 | 설명 | 상태 |
|------|------|------|
| 장비 | 영웅에 장착하는 장비 | 설계 중 |
| 카드 | 장비 소켓에 장착 → 스킬 발동 용도 | 설계 중 |
| 포션 | 체력 회복 및 특수효과 | 미기획 |
| 재료 | 장비 업그레이드 재료 | 미기획 |

### 카드 시스템
- 몬스터마다 고유한 카드 보유
- 몬스터 처치 시 일정 확률로 카드 드랍
- 드랍된 카드는 장비의 **소켓**에 장착 가능
- 발동: 공격 시 / 타격(명중) 시 / 피격 시 → n% 확률로 스킬 발동

### 아이템 등급

| 등급 | 획득처 | 특징 |
|------|--------|------|
| 일반 (Normal) | 필드 | 재료용 |
| 매직 (Magic) | 필드 | 접두사 또는 접미사 중 1개 |
| 레어 (Rare) | 필드, 보스 | 접두사 + 접미사 |
| 유니크 (Unique) | 보스 | 보스별 1종, 특성 고유 / 능력치 랜덤 |
| 크래프톤 (Crafton) | 생성 | 직접 제작 |

### 아이템 드랍 로직

```
1. 몬스터 처치
   └─ 정보: level, field-level, category, grade, spawn-grade(일반/정예)
   └─ → monster-score 계산 (공식 미정)
   └─ category는 아이템 종류에만 영향

2. 드랍 여부 결정
   └─ spawn-grade별 판정 횟수 (일반: 1회, 정예: 3회)

3. 아이템 종류 결정
   └─ category + grade로 드랍율 결정
   └─ 종류: gold, equip, card, jewel 등 (구상 중)

4. (장비) 장비 종류 결정
   └─ category + grade로 결정

5. 베이스 품질 결정
   └─ item-base 정보 + 몬스터 점수 → 베타분포로 결정

6. 아이템 등급 결정
   └─ 몬스터 점수 기반 랜덤
   └─ 유니크: 특정 구역의 특정 몬스터만 드랍

7. 접두/접미사 결정
   └─ 랜덤 (매직: 1개, 레어: 2개, 유니크: 고정)

8. 옵션 개수 결정
   └─ 매직: 1~3개, 레어: 2~6개, 유니크: 고정

9. 옵션 수치 결정
   └─ 몬스터 점수 기반 베타분포

10. 최종 아이템 생성
```

## 8. 장비 시스템

### 장비 슬롯

| 장비 | 기본 능력치 | 종류 옵션 |
|------|------------|----------|
| 무기 | 공격력 | 대형/중형/소형 |
| 갑옷 | 방어력 | 대형/중형/소형 |
| 투구 | 방어력+ | - |
| 장갑 | 명중률+ | - |
| 신발 | 회피율+ | - |

**무기 종류별 효과**
- 대형: 대형유닛 +10% 데미지 / 소형유닛 -10% 데미지
- 중형: 없음
- 소형: 대형유닛 -10% 데미지 / 소형유닛 +10% 데미지

### 접두/접미사 (7죄종별)

| 장비 | 분노 | 나태 | 색욕 | 시기 | 오만 | 폭식 | 탐욕 |
|------|------|------|------|------|------|------|------|
| 무기 | 공격력증가(%) | 공격력 대폭증가/공격속도 감소 | 흡혈(%) | 방어력 %무시 | 강타(%) | 레벨당 공격력 | 앵벌 |
| 갑옷 | 방어력증가(%) | 방어력 대폭증가/회피율 감소 | 체력회복(%) | 피격 시 효과 | 데미지감소(%) | 레벨당 방어력 | 앵벌 |
| 투구 | 체력증가(%) | 체력 대폭증가/회피율 감소 | 마법저항력 | 피격 시 효과 | 모든 스킬레벨 | 레벨당 체력 | 앵벌 |
| 장갑 | 공격속도증가(%) | 공격속도 대폭증가/공격력 감소 | 치명타 확률 | 상태이상 | 스탯 | 레벨당 공격속도 | 앵벌 |
| 신발 | 회피율증가(%) | 회피율 대폭증가/방어력(체력) 감소 | 치명타 확률 | 상태이상 | 스탯 | 레벨당 회피율 | 앵벌 |

**공통 능력치**
- 최대/최소 피해 증가
- 대/중/소형 추가 피해
- 언데드/데몬 추가 피해
- 골드 획득 증가
- 아이템 드랍 확률 증가
- 소켓

### 죄종 각성 능력 (세트 레벨 6)

| 죄종 | 각성 능력 |
|------|----------|
| 분노 | **최후의 저항**: 사망에 이르는 피해 입었을 때 1회에 한해 체력 1로 생존 + 1턴 무적 + 공격력 2배 |
| 나태 | **얻고 잃고** (상세 미정) |
| 색욕 | 미정 |
| 시기 | **약자멸시**: 상대 체력 30% 이하일 때 방어력 무시 |
| 오만 | **완결무결**: 모든 스킬 레벨 +2, 상태이상 면역 |
| 폭식 | **철벽의 인내**: 전투가 길어질수록 받는 피해 영구 감소 |
| 탐욕 | 미정 |
