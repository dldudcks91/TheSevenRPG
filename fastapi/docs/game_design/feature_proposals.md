## 1. 장비 분해 시스템

### 배경
파밍 흐름도에 "불필요하면 분해"라고 기술되어 있으나, 분해의 결과물과 규칙이 미설계.

### 제안 (업데이트: 2026-03-16)
- **분해 결과물**: 광석 + 소량의 골드
- **등급별 분해 수익 차등**: 매직 < 레어 < 크래프트 < 유니크
- **죄종 선택은 낙인 시스템으로 분리됨** (챕터 보스 희귀 드롭)

### 기대 효과
- 드롭된 장비가 항상 가치를 가짐 (장착 불가해도 분해 재료로 순환)
- 크래프팅 재료 루트: 파밍 → 분해 → 크래프팅으로 자연스럽게 연결
- 인벤토리 관리 동기 부여

## 2. 전투 리포트 ✅확정

### 확정 내용
- **수준**: 중간 (통계만, 힌트 없음)
- **표시 항목**: 총 피해량/받은 피해량, 명중률, 회피율, 치명타 횟수, 경직 횟수, 상태이상 적중, 적 잔여 HP
- **서버 측**: 전투 시뮬레이션 결과에 통계 데이터 포함하여 반환
- **상세**: [battle_guide.md](battle_guide.md) 전투 리포트 섹션 참조

## 3. 정예 몬스터 접두사 시스템 ✅확정

### 확정 내용
- **구조**: 죄종 고유 특성 1개 + 공통 특성 2개 = 3개
- **고유 특성 7종**: 격분(분노)/태만(나태)/유혹(색욕)/박탈(시기)/불가침(오만)/탐식(폭식)/도박(탐욕)
- **공통 특성 16종**: 스탯 8종 + 전투 규칙 8종
- **총 변형**: 840가지
- **상세**: [monster_design.md](monster_design.md) 정예 몬스터 특성 시스템 섹션 참조

## 4. 장비 잠금 / 비교 UI

### 배경
파밍 RPG에서 인벤토리가 차면 실수로 좋은 장비를 분해/판매하는 사고 빈발.
코스트 시스템까지 고려한 장비 교체 판단이 복잡.

### 제안
- **잠금**: 장비에 잠금 표시 → 분해/판매 시 차단
- **즐겨찾기**: 인벤토리 상단 고정 + 크래프팅 베이스 선택 시 필터링
- **비교**: 드롭 장비 vs 현재 착용 장비 — 스탯 변화량 + 코스트 변화량 + 세트포인트 영향 표시
- **경고**: "장착하면 코스트 초과" 알림

### 기대 효과
- 장비 관리 실수 방지
- 코스트 시스템의 전략적 깊이가 UI로 전달

## 5. 스테이지 난이도 선택 (상위 티어)

### 배경
"교차 체인: 던전 티어 → 접사 티어"가 설계되어 있으나, 같은 스테이지를 높은 난이도로 재도전하는 구조가 명시되지 않음.

### 제안
- 클리어한 스테이지를 Tier 2/3으로 재입장 → 몬스터 강화 + 접사 티어 상승
- Lv40~50 그라인드 구간에서 이전 챕터를 높은 티어로 돌아가 파밍
- 티어별 해금 조건: 해당 스테이지 N회 클리어 또는 특정 레벨 도달

### 기대 효과
- 21스테이지가 티어별로 63+ 스테이지로 확장
- "교차 체인 1순위"의 구체적 구현 수단
- 엔드게임 파밍 콘텐츠 자연스럽게 확보

## 6. 일일/주간 보스 재도전 보너스

### 배경
방치 파밍 없이 수동 반복만 가능한 구조에서, 접속 동기를 높이는 보상 사이클 필요.

### 제안
- **일일**: 첫 챕터 보스 클리어 시 추가 보상 (재료, 골드 보너스)
- **주간**: 특정 챕터 보스 N회 이상 클리어 → 보너스 유니크 드롭률 증가
- 보너스 상태를 UI에서 확인 가능

### 기대 효과
- 하드코어 파밍 RPG에서도 로그인 리듬 형성 → 장기 리텐션 향상
- "오늘 뭘 해야 하지?" 판단 보조

## 7. 도전 과제 / 업적 시스템

### 배경
카드 도감과 별도로, 행동 기반의 장기 목표 시스템.

### 제안
- **전투 업적**: 특정 챕터 무피격 클리어, N연속 명중, 경직 없이 보스 처치
- **수집 업적**: 특정 죄종 세트 6레벨 달성, 모든 무기군 장착 경험
- **파밍 업적**: 특정 스테이지 N회 클리어, 유니크 첫 획득
- **보상**: 칭호 (시각적 표시), 소량의 골드/재료, 특수 외형 효과

### 기대 효과
- 자동전투 특성상 "돌리는 동안의 작은 성취감" 제공
- 다양한 플레이 스타일 유도 (한 세트만 파는 게 아니라 여러 도전)

## 8. PvP 아레나

### 배경
기획 필요에 "PVP 규칙 확정" 항목 존재하나, PvP 콘텐츠 프레임 없음.

### 제안
- **랭크 매치**: 자동전투 기반 비동기 PvP (상대의 장비/스탯 스냅샷과 대결)
- 사이즈 상성(무기 vs 갑옷)이 PvP에서 실질적 의미
- **보상**: 특수 외형, 칭호, 한정 크래프팅 재료
- **시즌제**: 주기적 리셋으로 메타 변화 유도

### 기대 효과
- 빌드 다양성의 최종 검증 무대
- 코스트 시스템의 전략적 의미 극대화
- 장기 엔드게임 콘텐츠

## 9. 장비 외형 시스템 (트랜스모그)

### 배경
39종 장비 베이스가 외형에 반영된다면, 성능과 외형 분리 수요 발생.

### 제안
- 한 번 획득한 베이스의 외형을 도감에 등록
- 등록된 외형을 다른 장비에 씌울 수 있음 (골드 소모)
- 외형 도감 수집률 표시

### 기대 효과
- 39종 베이스가 수집 콘텐츠로도 기능
- "성능은 Sacred Armor지만 외형은 Ghost Armor" 커스터마이징
- 추가 골드 소모처
