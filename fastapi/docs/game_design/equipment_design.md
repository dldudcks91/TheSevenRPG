## 무기군별 Implicit

| 무기군 | Implicit | 효과 |
|--------|----------|------|
| 한손검 | 찌르기 | 치명타 확률 +10% |
| 양손검 | 관통베기 | 방어력 20% 무시 (방어력 감소율과 별도 계산) |
| 도끼 | 없음 | 기본 데미지가 다른 무기군보다 높음 |
| 창 | 선제 | 전투 시작 시 100% 명중 선제 1타 |
| 둔기 | 경직 | 경직 시간 추가 +N초 (기본 경직에 가산) |

### 경직 시스템 ✅확정
- **경직 발동 조건**: 단타 피해 > 최대 HP ÷ 10
- **둔기 Implicit**: 데미지 조건 무시하고 강제 경직 발동 + 경직 시간 추가 +N초
- **타격회복속도(FHR)**: `실제 경직 시간 = 기본 경직 시간 × (1 - FHR%)` (수확체감)
  - 갑옷 나태 죄종 옵션으로 획득 가능
  - 공격 측(경직 시간 추가) vs 방어 측(타격회복속도) 공방 축
- 기본 경직 시간, 둔기 추가 경직 시간: 밸런싱 단계에서 확정


## 무기 베이스 (15종) ✅확정

> 수치 데이터: `fastapi/meta_data/equipment_base.csv`
> (베이스명, 무기군, 사이즈, min/max 데미지, 공격속도, Implicit 포함)

빠른 순서: 한손검 > 창 > 양손검 > 도끼 > 둔기 (무기군 간 0.15, 사이즈 간 ±0.15).
도끼 제외 4무기군 DPS ≈ 12.0 균형, 도끼는 Implicit 없는 대신 DPS +30%.

#### 무기군별 성격

| 무기군 | 컨셉 |
|--------|------|
| 한손검 | 빠르고 안정적, 치명타로 터뜨림 |
| 양손검 | 중간 속도, 방어력 무시로 탱커 상대에 강함 |
| 도끼 | 느리지만 최대 데미지 폭발, 운에 의존 |
| 창 | 안정적 + 선제 1타로 초반 유리 |
| 둔기 | 가장 느리지만 경직으로 상대 템포 붕괴 |

## 방어구 베이스 스탯 규칙 ✅확정

> **베이스 방어력은 고정** (레벨 무관). 장비 레벨은 기본옵션(%)의 범위를 결정.
> 기본옵션(%)은 **Local** — 해당 장비의 베이스 스탯에만 곱연산.
> 장갑/신발은 방어력 소량 고정, 방어력%는 없음 (스케일링 안 됨).

### 방어구 타입 구조 (2타입 × 3티어 = 6종)

| 슬롯 | 베이스 방어력 | 타입1 기본옵션 | 타입2 기본옵션 |
|------|-------------|-------------|-------------|
| 갑옷 | 높음 | 방어력% | 방어력% + 마법저항력+ |
| 투구 | 중간 | 체력% | 데미지% |
| 장갑 | 낮음 (고정) | 명중률% | 공격속도% |
| 신발 | 낮음 (고정) | 회피율% | 마법저항력% |

> 갑옷: 중갑(방어력 특화) vs 경갑(방어력+마법저항)
> 투구: 생존형(체력%) vs 공격형(데미지%)
> 장갑: 명중형(명중률%) vs 속도형(공격속도%)
> 신발: 회피형(회피율%) vs 마저항형(마법저항력%)

## 세트 시스템 구조

### 기본 규칙
- **브레이크포인트**: 2/4/6 통일 (폭식, 오만 제외)
- **2세트 = 챕터 상태이상**: 해당 죄종의 챕터 상태이상을 부여하는 효과
- **6세트 = 각성**: 스킬로 얻을 수 없는 강력한 고유 효과
- **세트레벨 상한**: 6
- **한 장비에서 같은 죄종 중복 불가**: 접두/접미에 동일 죄종 금지 (장비당 최대 죄종 1포인트)
- **유니크 접사**: 고정 접사 1개, 해당 죄종 세트포인트 +1 기여

### 6세트 도달 방법
- 장비만으로는 한 죄종 최대 5포인트 (슬롯당 1포인트, 5슬롯)
- **베이직 죄종 선택**: 플레이어가 죄종 1개를 선택하면 해당 죄종 세트포인트 +1
- **변경 조건**: 마을에서만 변경 가능
- 이를 통해 5 + 1 = **6세트 도달** (한 죄종 최대 = 상한과 일치, 초과 없음)

### 빌드 패턴 예시
| 패턴 | 세트 효과 | 예시 |
|------|----------|------|
| 6+4 | 6세트 1개 + 4세트 1개 | 6분노 + 4색욕 |
| 6+2+2 | 6세트 1개 + 2세트 2개 | 6분노 + 2시기 + 2탐욕 |
| 4+4+2 | 4세트 2개 + 2세트 1개 | 4분노 + 4색욕 + 2탐욕 |
| 4+2+2+2 | 4세트 1개 + 2세트 3개 | 4분노 + 2색욕 + 2시기 + 2탐욕 |
| 2+2+2+2+2 | 2세트 5개 (상태이상 5종) | 5개 죄종 상태이상 동시 적용 |

## 죄종별 세트 보너스

> 수치 데이터: `fastapi/meta_data/equipment_set_bonus.csv`

### 구조 요약
- **분노** (2/4/6): 화상 부여 / 잃은 체력 비례 공격력 / **최후의 저항** (사망 1회 생존 + 40% 회복) ✅확정
- **나태** (2/4/6): 빙결 20% 부여 / 미확정 / 미확정
- **색욕** (2/4/6): 매혹 부여 / **갈취** (적 공격력·방어력 흡수) / **지배** (피해 일부 마법화) ✅확정
- **시기** (2/4/6): 중독 부여 / 적 버프 1개 제거 / **약자멸시** (적 체력 30% 이하 방어력 무시) ✅확정
- **오만** (별도 구조): 6세트 **완전무결** (상태이상 면역 + 걸린 적에게 20% 추가 피해). 다른 죄종 2세트와 조합이 핵심 시너지.
- **폭식** (독립형 패널티): 세트포인트 쌓일수록 전 스탯 감소 (-10%/-20%/-35%). 아이템 자체는 1.5~2배 수치. ✅확정
- **탐욕** (2/4/6): 스턴 부여 / 드롭 접사 상위 굴림 확률 증가 / **약탈왕** (보스 처치 시 아이템 2개 드롭) ✅확정
