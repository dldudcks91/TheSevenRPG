# 스킬 풀 (종족별 분류)

> **목적**: 48장 카드(=48 고유 스킬) 설계를 위한 후보 스킬 풀
> **출처**: D2/D3/D4 스킬트리, DNF 스킬, TFT 메커니즘, 오리지널
> **트리거 타입 (7종)**: 공격 시(on_attack) / 타격 시(on_hit) / 피격 시(on_being_hit) / 쿨타임 후 첫 공격(on_cooldown_ready) / n번째 타격마다(every_n_hit) / 회피 시(on_evade) / n초마다(every_n_seconds)
> **전제**: 1:1 자동전투, AoE 없음, 쿨다운 기반(마나 없음)

---

## 카드 시스템 ✅ 확정

### 기본 구조

```
카드 드롭 (랜덤 수치 범주)
  ├→ 도감 자동 등록 (첫 획득 시 스킬 해금)
  ├→ 카드 전용 인벤토리 탭에 아이템으로 보관
  └→ 슬롯 장착 / 카드 레벨업 재료 / 분해 / 거래소 판매
```

### 도감
- 첫 획득 시 자동 등록 + 스킬 해금
- 도감 레벨업 시스템 없음 (수집 기록 역할만)
- 도감 그룹 보너스: 재설계 필요 (⬜ 미확정)

### 카드 수치 (범주)
- 드롭 시 수치가 랜덤 범주 내에서 결정
- 같은 카드라도 수치가 다를 수 있음 (더 좋은 수치로 교체 가능)
- 세부 범주는 밸런싱 단계에서 확정

### 카드 슬롯
- 4슬롯 점진적 해금 (Lv1/5/15/30)
- 같은 카드 1슬롯에만 장착 가능 (중복 장착 불가)

### 카드 레벨업
- 재료: **같은 카드 N장 + 카드 영혼 N개**
- 레벨업 시 카드 수치(범주)는 유지, 추가 보너스 획득
- 세부 수치는 밸런싱 단계에서 확정

### 카드 영혼
- 카드 분해 시 획득 (카드 종류 무관, 범용 재료)
- 거래소 거래 가능 여부 추후 결정

### 거래
- 카드 아이템 거래소 거래 가능 (추후 도입)

## 도감 그룹 시스템 ✅ 확정

### 그룹 구성

같은 스테이지에 등장하는 노말 몬스터 3종 + 스테이지 보스 1종 = 4종을 하나의 도감 그룹으로 묶는다.

```
[도감 그룹] = 노말 3종 + 스테이지 보스 1종
예) S1-1 파멸의 진영 = 고블린 척후병 + 고블린 전사 + 오크 전사 + 아바돈
```

- 7챕터 × 3스테이지 = 21개 그룹 + 보스 스테이지 7개 = **총 28개 도감 그룹**
- 챕터 보스는 그룹에 포함하지 않음 (개별 도감 관리)
- 메타데이터: `collection_group.csv`, `collection_group_bonus.csv`

### 그룹 레벨 판정

**합산 레벨**: 그룹 내 4종(노말3 + 보스1)의 도감 레벨 합산으로 판정.

```
그룹 레벨 = sum(노말A Lv + 노말B Lv + 노말C Lv + 보스 Lv)
미수집 몬스터 = 0으로 계산

합산 3  → 그룹 Lv1  (예: 1+1+1+0, 보스 없이 가능)
합산 5  → 그룹 Lv2  (예: 2+2+1+0, 보스 없이 가능)
합산 7  → 그룹 Lv3  (예: 3+2+2+0, 보스 없이 가능)
합산 10 → 그룹 Lv4  (예: 3+3+3+1, 보스 카드 필수)
```

| 그룹 레벨 | 필요 합산 | 보스 카드 | 비고 |
|----------|----------|----------|------|
| Lv1 | 3 이상 | 불필요 | 노말 3종 수집으로 달성 가능 |
| Lv2 | 5 이상 | 불필요 | 노말 레벨업으로 달성 가능 |
| Lv3 | 7 이상 | 불필요 | 노말 올 Lv3(=9) 내 달성 가능 |
| Lv4 | 10 이상 | **필수** | 노말 최대 합산 9 → 보스 카드 없이 도달 불가 |

### 설계 의도

- **합산 레벨**: 편식 파밍도 어느 정도 허용 → 진행이 막히지 않음
- **Lv4 보스 필수**: 노말 최대 합산 9 → 보스 카드가 "최종 열쇠" 역할
- **종족 연동 보상**: Demon→공격력, Normal→HP, Undead→명중률 (해당 종족 빌드에 시너지)
- **Lv4 보상 전환**: 쌓아온 주력 스탯 + 다른 축의 보너스 → 완성 보상의 차별감

## 종족별 스킬 설계 원칙

| 종족 | 카드 수 | 스킬 유형 | 빌드 방향 |
|------|---------|----------|----------|
| **Demon** (5 베이스) | 15장 | 공격 스킬 — 직접 피해, 추가 타격, 처형 | 딜러 빌드 |
| **Normal** (6 베이스) | 18장 | 자기 버프 — 스탯 강화, 방어, 회복 | 탱커/지속전 빌드 |
| **Undead** (5 베이스) | 15장 | 디버프/방해 — 적 약화, 상태이상, 견제 | 컨트롤 빌드 |

- 스테이지당 단일 종족 등장 → 원하는 스킬 유형을 위한 방향성 있는 파밍
- 스토리 모드에서는 3 스테이지 모두 진행 → 자연스럽게 3종족 모두 경험
