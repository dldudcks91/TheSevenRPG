# 공모전용 게임 콘셉트 아이디어 노트

> 작성일: 2026-03-20
> 목적: 기획 공모전 출품용 독창적 메커니즘 탐색
> 상태: 아이디어 단계 (미확정)

---

## 공모전 조건

- **기획만 보는 공모전** (구현 불필요)
- **독창성 배점 70%** — "이런 게임 본 적 없는데?"가 나와야 함
- 인디게임 공모전 수상작은 대부분 오프라인 싱글 패키지 (Steam/Unity)

---

## 현재 기획의 문제점

### 가져갈 것 (테마 자산)
| 요소 | 이유 |
|------|------|
| 7대 죄악 x 성경 모티프 | 테마 기반 자체는 강함 |
| 배신의 밤 서사 | 7명의 역할이 다 다르고, 기억 왜곡이 있음 |
| 죄악 빌드 철학 | "어떤 죄를 선택하고 어떤 죄를 버릴 것인가" |
| 코스트 딜레마 | "강해지려면 뭔가를 포기해야 한다"는 구조 |
| 세계관 (아우레아, 바알, 7악마) | 캐릭터와 관계가 잘 짜여있음 |

### 버릴 것
| 요소 | 이유 |
|------|------|
| 자동전투 파밍 RPG 루프 | farm->equip->auto->repeat는 가장 흔한 모바일 공식 |
| 웹 RPG 장르 | "저예산 모바일" 인상 |
| 속성 = 죄악 | 분노=불, 시기=독이면 스킨만 바꾼 것 |

### 핵심 지적
- 현재 기획서는 **개발 스펙 문서**이지 공모전 기획서가 아님
- "7대 죄악 테마 RPG"는 이미 수십 개 존재 → 테마만으로 독창성 불가
- 자동전투 파밍 RPG는 장르 자체가 독창성 0

## 7대 죄악의 본질 분석

죄악은 외부의 적이 아니라 **내 안의 욕망**.

- 분노 — 참으면 되는데 못 참음
- 탐욕 — 충분한데 더 원함
- 나태 — 해야 하는데 안 함
- 폭식 — 그만 먹으면 되는데 못 멈춤
- 시기 — 남의 것이 내 것보다 좋아 보임
- 색욕 — 이성을 잃게 만드는 끌림
- 오만 — 내가 맞다는 확신

> **공통점: "하지 말아야 하는 걸 알면서도 하는 것"**
> **핵심 현상: 중독 — 한 번 하면 또 하게 됨**

이게 게임 메커니즘이 되려면, 플레이어 본인이 "하지 말아야 하는 걸 알면서도 하게 되는" 구조여야 함.

---

## 확정 방향: 오토배틀 유지

오토배틀을 좋아하는 이유 3가지:
1. 전투 조작이 귀찮아서 (편의성)
2. 전투 전 준비가 진짜 게임이라고 생각해서 (빌드 중심 철학)
3. 전투 연출을 감상하는 게 좋아서 (시각적 만족)

### 오토배틀인데 독창적이었던 게임: Loop Hero
- 전투: 완전 오토배틀, 플레이어 개입 0
- 핵심: 플레이어가 하는 일이 "전투"가 아니라 "세계를 만드는 것"
- **교훈: 오토배틀은 결과 확인 수단일 뿐, 진짜 게임은 그 바깥에 있어야 함**

### 목표 구조
```
[파티] → 자동 이동 + 자동 전투 (알아서 돌아다님)
[나(바알)] → 그 동안 내 할 일을 함 (이게 진짜 게임)
```

## 메커니즘 아이디어 목록

### 아이디어 A: 매 턴 7개의 죄 중 하나를 선택

- 모든 행동이 죄악. 매 턴 "어떤 죄를 지을 것인가" 선택
- 전부 강하지만 전부 대가 있음. 안전한 선택 없음
- 같은 죄 반복 → 중독 → 통제 불능
- 처음엔 내가 죄를 선택, 나중엔 죄가 나를 선택

### 아이디어 B: 세계를 만들수록 나를 삼킨다 (Loop Hero + DD 변형)

- 죄악 타일 배치로 세계 생성 → 파티가 자동 탐험
- 내가 놓은 타일이 내 파티를 오염시킴
- 타일 안 놓으면 진행 불가, 많이 놓으면 파티 타락
- 독창적 지점: "내가 만든 세계가 내 팀을 타락시킨다"
- 문제: 타일 배치가 Loop Hero 냄새 강함

### 아이디어 C: 죄악은 전염된다

- 바알이 힘을 쓰면 지역이 해당 죄악에 감염
- 악마를 잡으려면 죄악을 써야 하는데, 세계가 망함
- "영웅이 세계를 구하면서 동시에 망친다"

### 아이디어 D: 죄악은 거래다

- 악마를 잡으려면 다른 악마의 힘을 빌려야 함
- 거래가 꼬이고 빚이 쌓이고 자유를 잃음
- "악마에게 빌려서 악마를 잡는다"

### 아이디어 E: 죄악은 나 자신이다 (소멸 게임)

- 바알의 능력치 = 7개 죄악. 스킬 쓰면 영구 감소
- 게임이 진행될수록 약해짐 (성장 게임의 반대)
- "7번째 악마를 잡을 때 나는 무엇이 남아있는가"

### 아이디어 F: 속삭임 (유혹자 플레이)

- 바알은 싸울 수 없음. 파티원에게 죄악을 속삭여서 강화
- 플레이어 = 유혹자. 내가 직접 내 파티원을 타락시킴

### 아이디어 G: 계약서

- 런 시작 전 악마들과 조건부 계약 체결
- 많이 걸수록 강하지만 조건이 꼬임

### 아이디어 H: 7게이지 밸런싱 (Reigns형)

- 영웅이 7개 죄종에 대한 게이지를 가짐
- 어느 하나라도 100 되면 해당 죄악에 지배당함
- 전부 0이면 힘이 없어서 진행 불가
- 높을수록 강하지만 100 찍으면 끝
- 파티가 싸우는 동안 나는 7개 게이지를 조율하는 게 "내 할 일"

## 삼국지6 전투 구조 참조

- 명령 → 자동 실행 → 결과 확인 → 다음 명령
- 오토배틀 + 턴 사이 개입 구조
- 죄악 명령을 내리면 강해지지만 오염도에 따라 명령대로 안 움직일 수 있음
- 가능성: "명령을 내리는 게임"이 아니라 "언제까지 이 녀석을 쓸 수 있는가"를 관리하는 게임

---

## 현재 유력 조합

```
오토배틀 (유지)
+ 7게이지 밸런싱 (Reigns형, "내 할 일")
+ 중독 메커니즘 (같은 죄 반복 시 통제 불능)
+ 배신의 밤 서사 (런 반복으로 진실 해금)
```

### 미해결 과제
- "파티가 싸우는 동안 내가 하는 일"의 구체적 형태
- 단순 버프 카드 써주기는 뻔함 → 더 신박한 개입 방식 필요
- 7게이지 관리가 게임으로서 충분히 재미있는가?
- 독창성 70% 기준을 충족할 수 있는가?
