# TheSevenRPG — 화면 기획서

> 최초 작성: 2026-03-17
> 최종 업데이트: 2026-03-17 (좌우 분할 5탭 구조 확정)
> 디자인 컨셉: 레트로 도트 RPG (CRT 다크 테마, 픽셀 폰트 Galmuri, 각진 보더)
> 타겟: PC 브라우저 (모바일 대응은 추후)

---

## 1. 화면 구조 개요

게임은 **2개 화면**으로 구성된다: Login → Main.

```
┌──────────┐         ┌──────────────────────────────────────────────┐
│  Login   │ ──────→ │                   Main                       │
│          │  로그인  │  ┌──────────┬───────────────────────────────┐│
│          │  성공   │  │ 좌측패널  │         우측 메인뷰            ││
│          │         │  │ (5탭전환) │     (마을 ↔ 전투 전환)        ││
│          │         │  │ 350px    │         나머지 flex            ││
│          │         │  └──────────┴───────────────────────────────┘│
└──────────┘         └──────────────────────────────────────────────┘
```

### Main 화면 — 좌우 분할 레이아웃

```
┌───────────────────────────────────────────────────────────────────────────────┐
│  상단 바: 닉네임 Lv.15  STR DEX VIT LUK COST  ████ EXP  3,200G  [로그아웃]   │
├────────────────┬──────────────────────────────────────────────────────────────┤
│ 📊 ⚔ 🎒 ⚡ 📖 │                                                              │
│────────────────│                                                              │
│                │                 우측 메인뷰                                   │
│  좌측 패널      │            (마을 ↔ 전투 전환)                                 │
│  (350px 고정)  │                                                              │
│                │   마을 모드: 마을 허브 + 스테이지 선택                          │
│  탭에 따라      │   전투 모드: HUD + Phaser 아레나 + 전투 로그                   │
│  내용 전환      │                                                              │
│                │                                                              │
│                │                                                              │
└────────────────┴──────────────────────────────────────────────────────────────┘
```

**핵심 원칙:**
- 좌측 패널 350px 고정폭, 우측은 나머지 flex
- 좌측 패널은 항상 표시 (5탭으로 내용 전환)
- 우측 메인뷰는 마을 ↔ 전투 상황에 따라 전환
- 화면 이동 없이 한 화면에서 대부분의 조작을 수행
- 팝업은 좌측 패널 위에 오버레이 (position: absolute)

## 2. 공통 요소

### 2-1. 로딩 오버레이

| 항목 | 설명 |
|------|------|
| 트리거 | `showLoading()` / `hideLoading()` |
| 위치 | 화면 전체 덮음 (fixed, z-index: 9998) |
| 스타일 | 검정 70% 투명도, 3도트 점멸 애니메이션 |

### 2-2. 토스트 알림

| 타입 | 보더 색 | 배경 | 용도 |
|------|---------|------|------|
| success | 초록 | 어두운 초록 | 성공 알림 |
| warning | 노랑 | 어두운 노랑 | 경고 |
| error | 빨강 | 어두운 빨강 | 에러 |
| info | 파랑 | 어두운 파랑 | 안내 |

### 2-3. 팝업 공통 동작

- **호버**: 마우스 올리면 팝업 표시, 빼면 사라짐
- **클릭**: 팝업 고정 (마우스 빼도 유지)
- **닫기**: 팝업 외부 클릭 시 닫힘
- **위치**: 해당 아이콘 근처에 오버레이 (position: absolute)

## 3. Login 화면

**목적**: 게임 진입점. 닉네임 + 비밀번호로 로그인 또는 회원가입.

```
┌─────────────────────────────────────────────────────────────┐
│                                                             │
│                        THE SEVEN                            │
│              7대 죄악의 성에 오신 것을 환영합니다              │
│                                                             │
│              ╔═══════════════════════════╗                   │
│              ║  ┌───────────────────┐    ║                   │
│              ║  │     닉네임         │    ║                   │
│              ║  └───────────────────┘    ║                   │
│              ║  ┌───────────────────┐    ║                   │
│              ║  │     비밀번호       │    ║                   │
│              ║  └───────────────────┘    ║                   │
│              ║  (에러 메시지)             ║                   │
│              ║  ┌──────────┐┌──────────┐ ║                   │
│              ║  │  로그인   ││ 회원가입  │ ║                   │
│              ║  └──────────┘└──────────┘ ║                   │
│              ╚═══════════════════════════╝                   │
│                    (CRT 스캔라인 효과)                        │
└─────────────────────────────────────────────────────────────┘
```

| 요소 | 설명 |
|------|------|
| 타이틀 | "THE SEVEN" 36px 볼드 빨강, 3D 쉐도우 |
| 닉네임 입력 | maxlength=20, 중앙정렬 |
| 비밀번호 입력 | maxlength=30, type=password, Enter 키 → 로그인 |
| 에러 메시지 | 미입력/2자미만/4자미만/서버 에러 조건부 표시 |
| 로그인 버튼 | API 1007 호출 (빨강, 메인 버튼) |
| 회원가입 버튼 | API 1003 호출 (회색, 서브 버튼) |

**연동 API:** `1003` (회원가입), `1007` (로그인)

## 4. Main 화면

---

### 4-1. 상단 바

전체 너비, 좌우 패널 위에 걸쳐있는 캐릭터 요약 바.

```
┌─────────────────────────────────────────────────────────────────┐
│  닉네임  Lv.15   STR 10  DEX 12  VIT 8  LUK 5  COST 10        │
│  ████████████░░░░░░░░ EXP                    3,200 G  [로그아웃]│
└─────────────────────────────────────────────────────────────────┘
```

| 요소 | 위치 | 설명 |
|------|------|------|
| 닉네임 | 좌측 | 볼드 16px |
| 레벨 | 닉네임 옆 | "Lv.N" 노랑 볼드 |
| 스탯 요약 | 중앙 | STR/DEX/VIT/LUK/COST 인라인 칩 |
| EXP 바 | 하단 좌측 | 가로 바, 노랑 |
| 골드 | 우측 | "N,NNN G" 노랑 |
| 로그아웃 | 우상단 | 작은 텍스트 버튼 |

---

### 4-2. 좌측 패널 — 5탭 전환

350px 고정폭. 상단에 탭 아이콘 5개, 하단에 탭 내용 (세로 스크롤).

```
┌───────────────────────────────┐
│  📊   ⚔   🎒   ⚡   📖      │ ← 탭 아이콘 (5개)
│ 스탯  장비 아이템 스킬  도감    │
├───────────────────────────────┤
│                               │
│         탭 내용 영역           │ ← 세로 스크롤 가능
│                               │
└───────────────────────────────┘
```

| 탭 | 아이콘 | 역할 |
|----|--------|------|
| 스탯 | 📊 | 캐릭터 정보, 스탯 배분, 전투 스탯, 세트 보너스 상세 |
| 장비 | ⚔ | 장착 슬롯 5개, 코스트, 세트 요약, 미장착 장비 아이콘 그리드, 비교 팝업 |
| 아이템 | 🎒 | 포션/광석/낙인/퀘스트재료 (스택), Phase 18 전까지 "준비 중" |
| 스킬 | ⚡ | 스킬 슬롯 4개, 미장착 스킬 아이콘 그리드, 비교 팝업 |
| 도감 | 📖 | 챕터별 도감 그룹, 카드 수집 현황, 그룹 패시브 (열람 전용) |

#### 4-2-A. 스탯 탭

**목적**: 캐릭터 스탯 확인/배분 + 전투 스탯 + 세트 보너스 상세.

```
┌───────────────────────────────┐
│  📊   ⚔   🎒   ⚡   📖      │
├═══════════════════════════════┤
│  캐릭터 정보                   │
│  ─────────────────────────── │
│  레벨: 15                     │
│  경험치: 4,200 / 5,000        │
│  골드: 3,200 G                │
│                               │
│  스탯 배분 (SP: 5)            │
│  ─────────────────────────── │
│  STR  10   [+]               │
│  DEX  12   [+]               │
│  VIT   8   [+]               │
│  LUK   5   [+]               │
│  COST 10   [+]               │
│                               │
│  전투 스탯                     │
│  ─────────────────────────── │
│  HP: 180 / 200               │
│  공격력: 45~62                │
│  공격속도: 1.2                │
│  명중률: 62%                  │
│  회피율: 8%                   │
│  치명타: 12%                  │
│                               │
│  세트 보너스                   │
│  ─────────────────────────── │
│  분노 ██░░░░░ 2/7            │
│   2세트: 화상 부여        ✓   │
│   4세트: 잃은HP 비례 공격력↑  │
│   6세트: 최후의 저항          │
│  시기 █░░░░░░ 1/7            │
│   (미활성)                    │
│  탐욕 ░░░░░░░ 0/7            │
│   (미활성)                    │
│                               │
│  [스탯 리셋]                   │
└───────────────────────────────┘
```

| 영역 | 요소 | 설명 |
|------|------|------|
| 캐릭터 정보 | 레벨/경험치/골드 | 기본 정보 |
| 스탯 배분 | 5종 + [+] 버튼 | SP > 0일 때 활성, 클릭 시 +1 |
| 전투 스탯 | 최종 전투력 | 장비 + 기본 스탯 반영 |
| 세트 보너스 | 죄종별 바 + 효과 | 활성=초록✓, 미활성=회색, 0포인트=축약 |
| 스탯 리셋 | 버튼 | API 1005, 골드 소비 |

| 사용자 액션 | 결과 |
|-------------|------|
| [+] 클릭 | 스탯 +1, SP -1, 전투 스탯 실시간 갱신 |
| [스탯 리셋] 클릭 | 확인 팝업 → API 1005 → 전체 초기화 |
| 세트 효과 행 호버 | 효과 상세 툴팁 |

**연동 API:** `1004` (유저 정보), `1005` (스탯 리셋)

#### 4-2-B. 장비 탭

**목적**: 장비 장착/해제, 인벤토리 관리, 판매. 비교 팝업으로 장비 비교.

```
┌───────────────────────────────┐
│  📊   ⚔   🎒   ⚡   📖      │
├═══════════════════════════════┤
│  장착 장비                     │
│  ┌──┐┌──┐┌──┐┌──┐┌──┐       │
│  │⚔ ││🛡││🪖││🧤││👢│       │ ← 5슬롯 일자, 등급 보더
│  └──┘└──┘└──┘└──┘└──┘       │
│                               │
│  코스트: 8 / 30               │
│  ███████░░░░░░░░░░░           │
│                               │
│  세트: 분노2 시기1            │ ← 세트 포인트 요약
│                               │
│  미장착 장비  24개              │
│  [전체][무기][방어구]           │ ← 필터 탭
│  ┌──┐┌──┐┌──┐┌──┐┌──┐       │
│  │⚔ ││⚔ ││🛡││🪖││👢│       │ ← 정사각형 아이콘 그리드
│  └──┘└──┘└──┘└──┘└──┘       │    등급별 보더, 부위 아이콘
│  ┌──┐┌──┐┌──┐┌──┐┌──┐       │    세로 스크롤
│  │🛡││🧤││⚔ ││👢││🪖│       │
│  └──┘└──┘└──┘└──┘└──┘       │
│  ┌──┐┌──┐┌──┐               │
│  │⚔ ││🛡││🧤│               │
│  └──┘└──┘└──┘               │
│                               │
│  24 / 100          [인벤확장] │
└───────────────────────────────┘
```

**비교 팝업 (미장착 아이콘 호버 시):**
```
┌─────────────────┐ ┌─────────────────┐
│ ◀ 착용중         │ │ 미착용 ▶        │
│─────────────────│ │─────────────────│
│ 분노의 대검      │ │ 접두 매직 단검   │
│ 레어 · Lv.5     │ │ 매직 · Lv.3     │
│ Cost 3          │ │ Cost 2          │
│─────────────────│ │─────────────────│
│ 공격력    +18   │ │ 공격력    +12   │
│ 공격속도   +3   │ │ 공격속도   +8   │
│ 명중      +5   │ │ 치명타     +4   │
│                 │ │                 │
│                 │ │ [장착]  [판매]   │
└─────────────────┘ └─────────────────┘
```
- 수치 상승=**초록**, 하락=**빨강**
- 해당 부위 비착용 시 좌측 "비어있음"

**단독 팝업 (장착 슬롯 클릭 시):**
```
┌─────────────────┐
│ 착용중           │
│─────────────────│
│ 분노의 대검      │
│ 레어 · Lv.5     │
│ Cost 3          │
│─────────────────│
│ 공격력    +18   │
│ 공격속도   +3   │
│ 명중      +5   │
│                 │
│ [해제]  [판매]   │
└─────────────────┘
```

**아이콘 표시 규칙:**
- 정사각형 48×48 또는 52×52px
- 부위 아이콘: ⚔(무기), 🛡(갑옷), 🪖(투구), 🧤(장갑), 👢(신발)
- 등급별 보더: normal=회색, magic=파랑, rare=노랑, craft=초록, unique=보라
- 아이콘 내부에 부위 아이콘 + 레벨 (예: ⚔ +3)

| 영역 | 요소 | 설명 |
|------|------|------|
| 장착 슬롯 | 5개 일자 | weapon/armor/helmet/gloves/boots |
| 코스트 바 | 게이지 | 현재/최대, 초과 시 장착 불가 |
| 세트 요약 | 텍스트 | "분노2 시기1" (상세는 스탯 탭) |
| 필터 탭 | 3탭 | 전체 / 무기 / 방어구 |
| 아이콘 그리드 | 정사각형 | 미장착 장비만, 등급 보더, 부위 아이콘 |
| 비교 팝업 | 호버/클릭 | 착용중 vs 미착용 나란히, 수치 차이 컬러 |
| 단독 팝업 | 슬롯 클릭 | 착용 장비 상세 + 해제/판매 |
| 푸터 | 카운트 + 확장 | "24 / 100" + [인벤 확장] |

| 사용자 액션 | 결과 |
|-------------|------|
| 미장착 아이콘 호버 | 비교 팝업 |
| 미장착 아이콘 클릭 | 팝업 고정 |
| 장착 슬롯 클릭 | 단독 팝업 |
| [장착] 클릭 | API 2001 → 슬롯 갱신, 세트 요약 갱신 |
| [해제] 클릭 | API 2002 → 슬롯 비움 |
| [판매] 클릭 | API 2004 → 아이템 제거 + 골드 갱신 |
| [인벤 확장] 클릭 | API 2005 → 골드 차감 + 최대 칸 증가 |
| 필터 탭 클릭 | 아이콘 그리드 필터링 |

**연동 API:** `2001`~`2005`

#### 4-2-C. 아이템 탭

**목적**: 스택형 아이템(포션/광석/낙인/퀘스트재료) 관리.

> **구현 시기:** Phase 18 이후. 그 전까지 "준비 중" 표시.

```
┌───────────────────────────────┐
│  📊   ⚔   🎒   ⚡   📖      │
├═══════════════════════════════┤
│  [포션] [광석] [낙인] [퀘재]   │ ← 카테고리 필터
│  ─────────────────────────── │
│  ┌──┐┌──┐┌──┐┌──┐┌──┐       │
│  │🧪││🧪││🧪││  ││  │       │ ← 정사각형 아이콘
│  │×3││×1││×5││  ││  │       │    우하단 수량 뱃지
│  └──┘└──┘└──┘└──┘└──┘       │
│  ┌──┐┌──┐┌──┐               │
│  │💎││💎││💎│               │
│  │×12│×8││×2│               │
│  └──┘└──┘└──┘               │
│  ┌──┐                       │
│  │🔥│                       │
│  │×1│                       │
│  └──┘                       │
└───────────────────────────────┘
```

**아이콘 호버 시 팝업:**
```
┌─────────────────┐
│ HP 포션 (소)     │
│─────────────────│
```

#### 4-3-A. 마을 모드 (기본 상태)

**목적**: 마을 허브. 배경 이미지 위에 NPC 오브젝트를 배치하여 클릭으로 시설 진입.

```
┌──────────────────────────────────────────────────────┐
│  ████ 성벽 ██████   ⚔ 출정문   ██████ 성벽 ████     │ ← 상단: 성벽 + 출정문
│                                                      │
│              (마을 배경 이미지 — 전체 채움)             │
│                                                      │
│     🔨                💰                 📋          │ ← 중단: NPC 안뜰
│    대장간             상인              게시판         │
│                                                      │
│              (성 안뜰 코블스톤 바닥)                   │
│                                                      │
└──────────────────────────────────────────────────────┘
```

**NPC 오브젝트 표시 규칙:**
- 배경 이미지 위에 절대좌표(%)로 클릭 영역 배치
- 출정문: 상단 중앙 (50%, 12%) — 성벽 위 큰 아이콘, 호버 시 빨강 발광
- NPC 시설: 중하단 안뜰에 좌/중/우 배치 (20%/50%/80%, 55~60%)
- 각 NPC는 아이콘 + 이름 라벨
- 호버 시: 이름 라벨 밝아짐 + 테두리 발광
- 미해금 NPC: 반투명 (opacity 0.3) + 🔒 아이콘 + 호버 시 "Ch.N 클리어 후 해금" 표시
- 해금 조건: 대장간=Ch1, 게시판=Ch2, 상인=Ch3

| NPC | 아이콘 | 해금 조건 | 클릭 시 |
|-----|--------|----------|---------|
| 대장간 | 🔨 | Ch1 클리어 | 우측 뷰 → 크래프팅 화면 (Phase 19) |
| 상인 | 💰 | Ch3 클리어 | 우측 뷰 → 상점 화면 (Phase 20) |
| 게시판 | 📋 | Ch2 클리어 | 우측 뷰 → 퀘스트 화면 (Phase 20) |
| 출정문 | ⚔ | 항상 해금 | 우측 뷰 → 스테이지 선택 모드 |

| 사용자 액션 | 결과 |
|-------------|------|
| NPC 호버 | 이름 라벨 강조 + 테두리 발광 |
| NPC 클릭 (해금) | 우측 뷰 전환 |
| NPC 클릭 (미해금) | "Ch.N 클리어 후 해금됩니다" 토스트 |
| 출정문 클릭 | 우측 뷰 → 스테이지 선택 |
