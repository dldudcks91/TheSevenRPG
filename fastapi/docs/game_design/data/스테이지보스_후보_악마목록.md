# 스테이지 보스 후보 목록 (임시)

> 성경/신화/고에티아/민속/문학에서 조사한 스테이지 보스 후보 전체 목록
> 챕터 보스(사탄/레비아탄/맘몬/벨페고르/바알제붑/아스모데우스/루시퍼)는 제외

## 평가 기준
- **유명세** ★1~5: 게이머/대중문화에서 얼마나 알려져 있는가
- **연관도** ★1~5: 해당 챕터 보스(죄종)와 얼마나 잘 맞는가

## 타입별 분류 원칙
- **Demon**: 지옥·마법 계열 악마 (아바돈, 릴리스, 메피스토펠레스 등)
- **Normal**: 인간류·자연 생물·고대 신 (베헤모스, 레기온, 다곤 등)
- **Undead**: 죽어서 되살아난 존재, 사령, 망자 (드라큘라, 밴시, 리치 등)
