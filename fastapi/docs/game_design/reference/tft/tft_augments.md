## Set 8 — Monsters Attack!

### 시스템 구조

- **Hero Augments 118개** (챔피언 전용 — 각 챔피언당 2개)
- 일반 Regular Augments 170개 이상
- **총 280개 이상의 Augment**

### 증강체 유형

| 유형 | 설명 |
|------|------|
| Hero Augment | 특정 챔피언 보유 시 선택 가능한 전용 강화 (2종 중 1개 선택) |
| Regular Augment | 기존과 동일한 일반 증강체 |

### Hero Augment 구조

- 챔피언마다 **영웅(Hero)형** 또는 **악당(Villain)형** 중 1개 선택
- 영웅형: 해당 챔피언의 스킬/스탯 강화
- 악당형: 게임 전반 시스템 효과

### 주요 Hero Augment 예시

| 챔피언 | 영웅형 | 악당형 |
|--------|--------|--------|
| Jinx | Jinx 처치 시 연쇄 로켓 발사 | 아군 사망마다 Jinx 공격속도 증가 |
| Vi | Vi 돌진 시 경로 적에게 추가 기절 | 전장 어느 위치에서든 돌진 가능 |
| Thresh | 낫 투척 범위 증가 | 사망한 적 영혼 수집으로 방어력 증가 |
| Zed | 분신 2개 소환 | 분신이 적 처치 시 아이템 드랍 |
| Lucian | The Culling 투사체 수 증가 | 아군 죽을 때마다 이동속도 증가 |
