## Set 1 — Beta Set

- **출시:** 2019년 6월 (패치 V9.13) | **종료:** 2019년 11월
- **테마:** 룬테라 진영(Faction) 정치
- TFT 최초 공개 버전. 공식 명칭은 "Set 1" 또는 "Beta Set"
- 57명의 챔피언, 다양한 룬테라 진영 기반 시너지
- 증강체 시스템 없음
- **주요 메타:** 6 Noble, 6 Glacial, 9 Sorcerer, 6 Blademaster, Void/Assassin 조합

---

## Set 2 — Rise of the Elements

- **출시:** 2019년 11월 (패치 V9.22) | **종료:** 2020년 3월
- **테마:** 원소(Element)
- Origin이 "Element"로 재명명
- **핵심 메카닉 — 원소 헥스 시스템:** 매 게임마다 Inferno/Mountain/Ocean/Wind 4개의 원소 헥스가 랜덤 배치, 위에 있는 유닛에게 보너스 부여
- 7골드짜리 특수 챔피언 Lux 등장 (원소별로 다른 외형/능력)
- Avatar 오리진: 보드 원소에 따라 능력이 바뀌는 챔피언들
- 증강체 시스템 없음
- **주요 메타:** Ocean/Mage, Inferno/Sorcerer, 6 Light, 4 Electric, Poison 조합

---

## Set 3 — Galaxies

- **출시:** 2020년 3월 (패치 10.6) | **종료:** 2020년 9월
- **테마:** 우주/SF (Dark Star, Star Guardian, Mech-Pilot 스킨 세계관)
- **핵심 메카닉 — 갤럭시 시스템:** 매 게임 랜덤 갤럭시가 선택되어 게임 규칙 변경 (예: 초기 골드 2배, 챔피언 풀 변경 등)
- Mech-Pilot: 3개 유닛이 합체하여 초대형 메카 소환
- 58명 챔피언 (Origins 10개 + Classes 13개)
- 증강체 시스템 없음
- **주요 메타:** Mech-Pilot, 6 Cybernetic, 6 Dark Star, Star Guardian/Sorcerer

### 갤럭시 유형 예시

| 갤럭시 | 효과 |
|--------|------|
| Medium Legends | 챔피언 풀 크기 절반 |
| Trade Sector | 1-1 캐러셀에서 챔피언 1개씩 교환 가능 |
| Neekoverse | 모든 플레이어 동일한 2개 Neeko's Help 획득 |
| Star Cluster | 1-1 캐러셀에서 2스타 챔피언 등장 |
| Lilac Nebula | 첫 Spat 공짜 |
| Binary Stars | 아이템 2개 착용 가능 (초과 불가) |
| Superdense Galaxy | 초기 경험치 대폭 감소 |

## Set 4 — Fates

- **출시:** 2020년 9월 (패치 V10.19) | **종료:** 2021년 1월
- **테마:** 동아시아 신화/판타지 (Warring Kingdoms, Blood Moon, Spirit Blossom)
- **핵심 메카닉 — Chosen:** 상점에 가끔 2스타 상태로 등장하는 Chosen 챔피언 구매 가능 (정가 3배). 해당 챔피언의 특성 하나가 2개로 카운트 + 고유 스탯 보너스
- 58명 챔피언, 26개 특성 (Origins 13개 + Classes 13개)
- 증강체 시스템 없음
- **주요 메타:** Dusk/Moonlight, Cultist Galio, Duelist, Warlord

### Chosen 메카닉 상세

| 항목 | 내용 |
|------|------|
| 등장 조건 | 상점에서 낮은 확률로 등장 (일반 챔피언보다 진하게 표시) |
| 구매 비용 | 정가 × 3 (1코스트 Chosen = 3골드) |
| 특성 보너스 | 특성 카운트 +1 (예: 3 Dusk Chosen → 6 Dusk 효과) |
| 스탯 보너스 | Chosen 타입별 고유 스탯 추가 |
| 제한 | 1게임에 Chosen 1명만 보유 가능 |

## Set 5 — Reckoning

- **출시:** 2021년 4월 28일 (패치 V11.9) | **종료:** 2021년 7월
- **테마:** 선악 대립 (Dawnbringer vs Nightbringer)
- **핵심 메카닉 — Shadow Items:** 아이템에 Shadow 버전 존재. 강력한 효과가 있지만 페널티도 동반
- Dawnbringer(빛) vs Nightbringer(어둠) 진영 구분 — 양쪽을 섞으면 패널티
- God-King: Garen(선) vs Darius(악) — 둘이 같은 팀이면 상호 약화
- 60명 챔피언, Origins 14개 + Classes 14개
- 증강체 시스템 없음
- **주요 메타:** Forgotten Shadow Items, Redeemed, 6 Hellion

### Shadow Items 예시

| 일반 아이템 | Shadow 버전 | 페널티 |
|------------|-------------|--------|
| B.F. Sword | Shadow B.F. Sword | 최대 HP -15% |
| Needlessly Large Rod | Shadow Rod | 받는 피해 +10% |
| Chain Vest | Shadow Chain Vest | 이동속도 감소 |

## Set 5.5 — Dawn of Heroes

- **출시:** 2021년 7월 (패치 V11.15) | **종료:** 2021년 11월

- 총 57명 챔피언 (12개 신규 챔피언 추가)
- 증강체 시스템 없음
- **주요 메타:** 8 Sentinel, Cannoneer, Forgotten (Shadow 없이)

### Radiant Items 상세

| 항목 | 내용 |
|------|------|
| 획득 방법 | Armory (황금 캐러셀에서 선택) 또는 특수 이벤트 |
| 효과 | 완성 아이템의 강화 버전 + 추가 보너스 효과 |
| 예시 | Radiant Warmog's Armor = 일반 효과 + 전체 아군 HP 회복 |

  - 총 140개 이상의 Hextech Augment

- **주요 메타:** Mutant, Innovator, Socialite Carry, Yordle

## Set 14 — Cyber City

- **출시:** 2025년 4월 2일 (패치 15.6) | **현재 진행 중**
- **테마:** 사이버펑크 도시 "Cyber City" — 네온, 해킹, 미래 기술
- **핵심 메카닉 — Hacks:** 게임 중 3회 등장하는 Hacks가 Augment/상점/전리품/만남 등 핵심 시스템을 변형
  - Black Market Augments: 과거 세트 Augment가 해킹된 슬롯에서만 등장
  - Hacked 상점: 2스타 챔피언 제공
- Anima Squad 부활 + Garen(God of the Net), Zac(Virus) 신규 단독 유닛
- 27개 특성 (Origins 14개 + Classes 13개)
- **주요 메타:** Street Demon, Divinicorp, Anima Squad

### Hacks 시스템

| 항목 | 내용 |
|------|------|
| 등장 횟수 | 게임 중 3회 |
| Hack 유형 | Black Market Augment / Hacked Shop / Loot Hack / Encounter Hack |
| Black Market | 해당 슬롯에 과거 세트 증강체 등장 (Set 6~13 중 랜덤) |
| Hacked Shop | 해당 라운드 상점에 2스타 챔피언 등장 |
| 증강체 선택 | 1개의 강한 증강체 vs 2개의 약한 증강체 |
