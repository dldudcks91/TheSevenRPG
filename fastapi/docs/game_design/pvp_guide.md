## 1. 기본 PVP 전투

| 항목 | 내용 |
|------|------|
| 방식 | PVE 동일 1:1 자동전투 (서버 시뮬레이션) |
| 선공 | 없음 — 양측 동시 공격 (공격속도 쿨타임 각자 독립) |
| 시간 제한 | 60초, 초과 시 HP% 비교하여 높은 쪽 승리 |
| 레벨 보정 | 없음 — 명중 공식에서 LvDiff 항 0 고정 |
| 포션 | 사용 불가 |
| 연맹 버프 | 적용 |

### 명중 공식 (PVP)

```
hit_rate = clamp(Acc/(Acc+Eva) + 0 + K, 0.05, 0.95)
```

- PVE 공식과 동일하되 `LvDiff×0.01` 항만 제거
- K = 0.1 (기본 보정) 유지
- 레벨 차이는 명중에 영향 없음 — 순수 스탯/장비 차이로만 판정

### 동시 공격 구조

- 양 플레이어가 각자의 공격속도 쿨타임에 따라 독립적으로 공격
- 선공 우위 없음 — 공격속도가 빠르면 "더 자주 때림" (먼저 때리는 이점 아님)
- 전투 시작 시 양측 쿨타임 0에서 동시 시작

## 2. 전투 전 전략

### 기본 아레나

- 금기/카드 교체 등 추가 시스템 없음
- 매칭 즉시 전투 진행
- 정찰 정보 (전적/관전 조회용): 레벨, 무기군, 세트 포인트 분포

### 복수전

- 아레나 패배 후 24시간 내 1회 도전 가능
- 추가 정보 공개: 상대의 무기 베이스 이름 + 갑옷 타입(중갑/경갑)
- 카드 교체 등 추가 시스템 없음 (기본 아레나와 동일)

> 복수전의 이점은 "추가 정보를 아는 상태에서 내 빌드를 조정할 시간이 있음"

## 3. PVP 모드

### 아레나 (1v1 랭크)

- **참가 조건**: Lv20 이상
- **등급제**: 청동 → 은 → 금 → 백금 → 다이아 → 챔피언
- **시즌제**: 주기 및 세부 보상 추후 확정

### 복수전

- 아레나 패배 후 24시간 내 1회
- 세부 보상 추후 확정
