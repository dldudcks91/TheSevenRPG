## v0.8 — 2026-03-09 (8차)

### 확정
- **직업 시스템 없음**: 직업 도입 시 카드 기반 스킬 시스템과 역할 충돌 → 제거
- **캐릭터 개성 3축**: 스탯 투자 방향 + 카드 4개 선택 + 죄종 세트 방향
- **전직 개념 추후 추가 예정**: 특정 조건 달성 후 7죄종 기반 전직 선택, 해당 죄종 스탯 계수 강화

### 기각
- 직업별 스탯 계수 차등화 (전사/도적/마검사): 전직 시스템으로 흡수 예정
- 직업별 고유 스킬: 카드 시스템 정체성 희석 우려로 기각

## v0.7 — 2026-03-09 (7차)

### 확정
- **낙인(Brand) 시스템 1차 확정**
  - 소켓 구조: 기본 1소켓(접두 죄종 전용) + 드롭 랜덤 옵션으로 2소켓(접미 죄종 전용)
  - 타입 매칭: 낙인 죄종 = 장비 접사 죄종 일치 시에만 삽입 가능
  - 낙인 스탯: 완전 랜덤 (죄종 스탯과 분리)
- **파밍 레이어 교차 구조 4종 확정**
  - 체인 1 (1순위): 던전 티어 → 접사 티어 (iLvl 구조)
  - 체인 2 (2순위): 낙인 분해 → 죄종 전용 파편 → 접사 고정 크래프팅
  - 체인 3: 죄종 세트 레벨 X 달성 → 해당 죄종 낙인 추가 효과 활성화

### 기각
- 흔적 게이지 × 낙인 직접 게이팅: 관리 피로도 과다 → 세트 레벨×낙인으로 대체

## v0.6 — 2026-03-06 (6차 이전)

### 확정
- **흔적 시스템 1차 확정**
  - 7종류 (7죄종 대응)
  - 게이지 방식 (처치 시 소모, 시간 경과로 감소)
  - 파밍 효율 전용 (전투 스탯 직접 영향 없음)
- **크래프팅 시스템 (PoE 커런시 참조)**
  - 장비 분해 → 파편 획득 (등급별 파편 수 차등)
  - 파편 소모: 옵션 재주사위 / 등급 업그레이드 / 옵션 추가
- **카드 각인 레벨 시스템 (Lost Ark 참조)**
  - Lv1(3장), Lv2(5장), Lv3(10장)
  - 각인 레벨에 따라 발동 확률 및 효과 강화

## v0.5 — 이전 (5차)

### 확정
- **아이템 등급 구조 확정**: 일반 제거 → 매직 / 레어 / 크래프트 / 유니크 4등급
- **크래프팅 방식**: 매직 베이스 + 죄종 카테고리 선택 + 재료 소모
- **강화 재료 2종(A/B)**: 강화/크래프팅 공용, 등급 계층형
- **스폰 등급별 드롭 분리**: 일반몹(골드/재료/카드만), 정예/보스(장비 드롭)
- **던전 구조**: 일반4+정예1 세트 × 3~4회 → 던전 보스

## v0.4 — 이전 (4차)

### 확정
- **카드 시스템 구조**: 수집 + 각인 레벨(Lv1/2/3) + 캐릭터 슬롯 4개 활성화
- **카드 옵션 방향**: 아이템에서 얻을 수 없는 고유 옵션 (경험치, 골드, 속성저항 등)
- **타겟 파밍 2축**: 지역→죄종 편향, 몬스터 베이스→부위 편향
- **몬스터 타입 3종**: Normal(일반) / Demon(데몬) / Undead(언데드)

## v0.3 — 이전 (3차)

### 확정
- **7죄종 시스템 옵션 방향 확정** (각 죄종별 특화 스탯)
  - 분노: 강력한 공격 특화
  - 나태: 강력한 옵션 + -옵션
  - 색욕: 아군 강화(버프)
  - 시기: 상대 약화(디버프)
  - 오만: 우월함 과시
  - 폭식: 스택형 스탯
  - 탐욕: 확률/운 도박
- **세트포인트 구조**: 접두/접미 각 1포인트, 5슬롯 최대 10포인트
- **죄종 세트 레벨 상한**: 7 (초기 구현 5)
- **코스트 시스템**: `장비레벨 × 옵션레벨 × 등급배율`

## v0.1 — 초기 기획 (1차)

### 확정
- **장르**: 웹 RPG (아이템 파밍 중심), 테마: 7죄종 + 중세신화
- **핵심 루프**: 몬스터 파밍 → 장비 드롭 → 스탯 강화 → 더 강한 몬스터
- **전투 방식**: 1:1 완전 자동전투, 공속 기반 쿨타임
- **기본 스탯 5종**: 힘(STR) / 민첩(DEX) / 체력(VIT) / 운(LCK) / 코스트(COST)
- **최대 레벨**: 99
- **기본 HP**: 100 (스탯 투자로만 증가)
- **방어력 공식**: `방어력 / (방어력 + 100)`
- **상태이상 4종**: 스턴 / 빙결 / 중독 / 화상
- **월드 구조**: 필드 + 던전, 스테이지 방식
- **패배 처리**: 필드/던전 밖으로 이동, 아이템 유지
