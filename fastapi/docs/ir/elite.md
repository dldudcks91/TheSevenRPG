## 목적
정예 몬스터에 죄종 고유 특성 1개 + 공통 특성 2개를 런타임 부여하고, 전투에 반영.

---

## 동작 규칙

**정예 구성**: 노말 유닛 스탯 기반 + 죄종 고유 특성 1개 + 공통 특성 2개 (3개 조합)

**죄종 고유 특성 (7종)**
| 죄종 | 특성명 | 효과 |
|------|--------|------|
| 분노 | 격분 | HP 30% 이하 시: 공격력×2, 공속+50% |
| 나태 | 태만 | 공속-50%, 공격력×2 |
| 색욕 | 유혹 | 타격 시 플레이어 공격력 10% 흡수 (정예+10%, 플레이어-10%, 스택) |
| 시기 | 박탈 | 치명타/회피 발동 시 무효화 (일반 피해로 전환) |
| 오만 | 불가침 | 모든 상태이상 면역 |
| 폭식 | 탐식 | 매 타격 공격력 +2% 영구 누적 |
| 탐욕 | 도박 | 피해 0.2~1.8배 랜덤, 처치 시 드롭+1 |

**공통 특성 풀 (16종, 2개 랜덤 선택)**

스탯 강화 (8종):
| 특성 | 효과 |
|------|------|
| 강인한 | HP +50% |
| 단단한 | DEF +50% |
| 거대한 | 사이즈 +1단계, HP +20% |
| 날랜 | 공속 +30% |
| 흉포한 | ATK +30% |
| 민첩한 | 회피율 +30% |
| 정확한 | 명중률 +30% |
| 치명적인 | 치확 +20%, 치뎀 +50% |

전투 규칙 (8종):
| 특성 | 효과 |
|------|------|
| 재생하는 | 매초 최대HP의 1% 회복 |
| 가시의 | 받은 피해의 15% 반사 |
| 경화의 | 피격마다 DEF +1% 영구 누적 |
| 선제의 | 전투 시작 시 선공 1회 (100% 명중) |
| 보복의 | 피격 시 30% 확률 즉시 반격 1회 |
| 폭발하는 | 사망 시 최대HP 50% 범위 폭발 |
| 저주받은 | 타격 시 30% 확률 랜덤 디버프 |
| 흡혈의 | 피해의 20% HP 회복 |

**특성 배정 규칙**
- 죄종 고유: 스테이지가 속한 챕터의 죄종 1개 고정
- 공통: 16종 중 2개 랜덤 (중복 불가)
- 총 조합: 7 × C(16,2) = 840종

---

## 입력 / 출력

API 없음. EliteManager는 내부 헬퍼로만 사용.

**BattleManager에서 호출**
- 입력: stage_id, monster_idx, spawn_type="정예"
- 출력: 특성이 적용된 몬스터 스탯 dict + traits 목록

## 상태 변화

**DB**: 없음 (특성은 런타임 생성, 저장하지 않음)
**Redis**: 없음

---

## 구현 제약
- 특성 효과는 BattleManager._simulate_with_hp 내에서 적용
- 스탯 강화 특성: 전투 시작 전 몬스터 스탯에 곱연산
- 전투 규칙 특성: 시뮬레이션 루프 내에서 조건별 처리
- 현재 Phase 16(상태이상) 미구현이므로 일부 특성은 Phase 16까지 보류:
  - 오만(불가침) — 상태이상 면역이나 상태이상 시스템 자체가 Phase 16
  - 저주받은 — 랜덤 디버프 부여이나 디버프 시스템이 Phase 16
