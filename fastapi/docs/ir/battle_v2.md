# 카드 스킬 발동(Phase 18.5), 세트 보너스(Phase 22) 스텁 대기
---

## 목적
확정된 전투 공식 전체를 BattleManager에 반영. 경직/사이즈보정/마저항/상태이상 7종/전투 리포트.

---

## Phase 16 구현 범위

### 구현
- 경직 시스템 (기본경직 + FHR)
- 사이즈 보정 (대↔대 +10%, 대↔소 -10%)
- 마법 저항 별도 감소율
- 상태이상 7종 (화상/중독/스턴/빙결/침식/매혹/심판)
- 전투 리포트 (통계)

### 스텁 (후속 Phase에서 활성화)
- 카드 스킬 발동 (Phase 18.5 카드 인벤토리 후)
- 세트 보너스 전투 적용 (Phase 22 세트 보너스 후)

---

## 핵심 공식

**경직**: 단타 피해 > 최대HP÷10 → 경직 발동, 실제 시간 = 기본 × (1 - FHR%)
**사이즈**: 대형↔대형 ×1.10, 대형↔소형 ×0.90, 중형 ×1.0
**마저항**: MR / (MR + 100)
**상태이상**: 화상4s(회복감소) / 중독4s(초당고정뎀) / 스턴1s(공격불가) / 빙결3s(공속감소) / 침식(영구DEF감소스택) / 매혹4s(명중감소) / 심판2s(스킬봉인)
