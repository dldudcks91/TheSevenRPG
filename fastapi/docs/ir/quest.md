## 목적
재료 납품 → 보상 교환. Ch2 클리어 해금. 반복 가능 퀘스트.

---

## 동작 규칙

### 퀘스트 조회 (API 4004)
- 시설 해금 검증 (비트 1)
- QUEST_LIST 상수 기반 목록 + 유저 재료 보유량 대비 완료 가능 여부

### 퀘스트 납품 (API 4003)
- quest_id 입력
- 시설 해금 검증
- 재료 차감 → 보상(골드/재료) 지급

### 퀘스트 목록 (폴백 상수)
| quest_id | 이름 | 필요 재료 | 보상 |
|----------|------|-----------|------|
| 1 | 광석 납품 (소) | ore/1 ×10 | 500G |
| 2 | 광석 납품 (대) | ore/1 ×30 | 2000G |
| 3 | 포션 공급 | potion/1 ×5 | 800G + ore/1 ×10 |
