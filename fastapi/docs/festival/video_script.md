# TheSevenRPG — 게임 소개 영상 스크립트

> 목표 길이: 2~3분
> 형식: 나레이션 + 게임 화면 캡처/목업 + 텍스트 오버레이
> 톤: 어둡고 긴장감 있는 도입 → 시스템 설명은 명확하고 간결하게

---

## 구성 (5파트)

| 파트 | 시간 | 내용 |
|------|------|------|
| 1. 훅 | 0:00~0:20 | 세계관 도입 — 배신의 밤 |
| 2. 콘셉트 | 0:20~0:40 | 한 줄 콘셉트 + 차별점 3축 |
| 3. 핵심 플레이 | 0:40~1:30 | 파밍→빌드→도전 루프 시연 |
| 4. 시스템 하이라이트 | 1:30~2:20 | 죄악 세트, 코스트, 타겟파밍 |
| 5. 클로징 | 2:20~2:40 | 개발 현황 + 마무리 |

## 파트 1. 훅 (0:00~0:20)

### 화면
- 검은 화면. 텍스트만 하나씩 나타남.
- 마지막에 붉은 낙인이 타오르며 등장.

### 나레이션
> 지옥에는 왕이 있었다.
> 7개의 감정을 나누어 부하에게 깃들게 한 대악마, 바알.
>
> 그 밤, 7개의 칼이 등에 꽂혔다.
> 뒤를 돌아봤다. 전부 아는 얼굴이었다.

### 텍스트 오버레이
```
"배신당했다."
```

---

## 파트 2. 콘셉트 (0:20~0:40)

### 화면
- 게임 타이틀 로고 등장
- 3축 차별점을 카드 형태로 하나씩 플립

### 나레이션
> TheSevenRPG.
> 7대 죄악으로 빌드하는 파밍 RPG.
>
> 테마가 장식이 아닙니다. 장비의 접사, 세트 효과, 상태이상, 보스의 서사 —
> 모든 시스템이 죄악이라는 하나의 축 위에 서 있습니다.

### 텍스트 오버레이
```
테마 = 속성 = 세트 = 서사
"분노를 얻고 탐욕을 버려라"
```

---

## 파트 3. 핵심 플레이 (0:40~1:30)

### 화면
- 스테이지 선택 화면 → 전투 진입 → 자동전투 씬 → 드롭 결과 → 인벤토리 → 장비 장착

### 나레이션
> 스테이지에 입장하면, 전투는 자동으로 진행됩니다.
> 플레이어가 개입할 수 없습니다. 승패는 이미 결정되었으니까요.
>
> 그래서 중요한 건 전투 전 준비입니다.
>
> 몬스터를 쓰러뜨리면 장비가 드롭됩니다.
> 장비에는 죄악 접사가 붙어 있고, 같은 죄종을 모으면 세트 효과가 발동합니다.
>
> 파밍하고, 조합하고, 도전하고, 실패하면 다시 파밍합니다.
> 이것이 핵심 루프입니다.

### 텍스트 오버레이 (루프 순서대로)
```
파밍 → 빌드 → 도전 → 판단 → 반복
```

---

## 파트 4. 시스템 하이라이트 (1:30~2:20)

### 4-1. 죄악 세트 (1:30~1:50)

#### 화면
- 인벤토리에서 접사 확인 → 세트 포인트 게이지 채워지는 연출

#### 나레이션
> 7개의 죄종마다 고유한 전투 특성이 있습니다.
> 분노는 공격, 나태는 강력하지만 페널티가 따르는 양날의 검.
> 2세트에서 상태이상, 4세트에서 고유 효과, 6세트에서 각성.
> 어떤 죄악을 선택할지가 곧 빌드입니다.

### 4-2. 코스트 딜레마 (1:50~2:05)

#### 화면
- 코스트 바 UI. 장비를 끼면 코스트가 차오르고, 한도 초과 시 빨간색 경고

#### 나레이션
> 좋은 장비에는 높은 코스트가 필요합니다.
> 코스트에 스탯을 투자하면 강한 장비를 낄 수 있지만, 기본 스탯이 약해집니다.
> 뭘 포기할 것인가. 이것이 빌드의 핵심 딜레마입니다.

### 4-3. 타겟 파밍 (2:05~2:20)

#### 화면
- 월드맵에서 지역 선택 → 해당 지역 죄종 하이라이트 → 몬스터 타입별 장비 부위 표시

#### 나레이션
> 파밍에도 전략이 있습니다.
> 지역을 고르면 죄종이 정해지고, 몬스터를 고르면 장비 부위가 정해집니다.
> 무엇을 파밍할지, 플레이어가 직접 설계합니다.

## 파트 5. 클로징 (2:20~2:40)

### 화면
- 7챕터 보스 실루엣이 하나씩 등장
- 마지막에 루시퍼 실루엣 + "나도 거기 없었다" 텍스트
- 게임 타이틀 + 개발 현황

### 나레이션
> 7명의 악마. 하나의 사건. 누가 배신했는가.
> 보스를 쓰러뜨릴 때마다, 진실이 한 조각씩 맞춰집니다.
>
> TheSevenRPG. 서버와 클라이언트 핵심 루프가 구동 중입니다.

### 텍스트 오버레이
```
서버 17개 API 완성
클라이언트 전투 연출 + UI 완성

TheSevenRPG
"분노를 얻고 탐욕을 버려라"
```

## 제작 노트

### 필요 소재
- 게임 플레이 캡처 (전투 씬, 인벤토리, 스테이지 선택)
- 텍스트 타이포그래피 애니메이션 (파트 1, 5)
- BGM: 어두운 오케스트라 (도입) → 긴장감 있는 일렉트로닉 (시스템 설명)

### 대안 (플레이 영상이 부족할 경우)
- 목업/와이어프레임 화면 + 나레이션으로 구성
- 기획서 다이어그램을 애니메이션화하여 활용
