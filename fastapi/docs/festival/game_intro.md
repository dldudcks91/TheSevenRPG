# TheSevenRPG

> "분노를 얻고 탐욕을 버려라 — 7대 죄악으로 빌드하는 파밍 RPG"

---

## 게임 정보

| 항목 | 내용 |
|------|------|
| 장르 | 웹 RPG (아이템 파밍 중심) |
| 플랫폼 | 웹 브라우저 (PC/모바일) |
| 타겟 | 빌드 설계와 파밍을 즐기는 하드코어 RPG 유저 |
| 개발 상태 | 서버·클라이언트 핵심 루프 구동 (19개 API 완성) |

---

## 이 게임이 다른 이유

대부분의 RPG에서 테마는 세계관을 꾸미는 배경이다. TheSevenRPG에서 **7대 죄악은 게임의 뼈대**다.

| 축 | 차별점 |
|---|--------|
| **테마** | 장비 접사, 세트 효과, 상태이상, 스테이지, 보스 서사 — 모든 시스템이 죄악 위에 서 있다 |
| **플레이** | 전투는 자동. 재미는 조작이 아니라 빌드 설계에 있다. 코스트 한도 안에서 "뭘 포기할 것인가"가 빌드의 핵심 |
| **서사** | 보스를 쓰러뜨려야 배신의 진실이 밝혀진다. 파밍의 동기가 곧 서사의 동기 |

## 세계관

지옥에는 왕이 있었다.

**바알**. 7개의 감정을 나누어 부하에게 깃들게 한 대악마.
분노, 시기, 탐욕, 나태, 폭식, 색욕, 오만 — 그것은 지옥을 유지하는 균형이었다.

그 밤, **7개의 칼이 등에 꽂혔다.**
뒤를 돌아봤다. 전부 아는 얼굴이었다.

바알은 추락했다. 힘을 잃고, 기억이 뒤섞인 채, 인간의 몸에 봉인되었다.
확실한 건 하나. **배신당했다.**

몸에 새겨진 낙인 — 저주이자, 죄악의 힘을 되찾을 수 있는 유일한 무기.
바알은 일어선다. 7명의 악마를 쫓고, 그들의 죄악을 빼앗아 자신을 무장하기 위해.

---

## 핵심 루프

```
[파밍] 스테이지를 돌며 장비를 모은다
    ↓
[빌드] 죄악 접사를 조합하고, 코스트 한도 내에서 최적의 세트를 만든다
    ↓
[도전] 다음 스테이지 또는 챕터 보스에 도전한다
    ↓
[판단] 실패 시 — 어떤 죄악이 부족한가? 어떤 장비를 파밍할 것인가?
    ↓
    반복
```

## 챕터 구조

7개의 챕터. 7명의 악마. 하나의 사건.

| 챕터 | 죄악 | 지역 | 보스 | 상태이상 |
|------|------|------|------|---------|
| 1 | 분노 | 불타는 전장 | 사탄 | 화상 |
| 2 | 시기 | 뒤틀린 숲 | 레비아탄 | 중독 |
| 3 | 탐욕 | 황금의 사막 | 맘몬 | 스턴 |
| 4 | 나태 | 망각의 동토 | 벨페고르 | 빙결 |
| 5 | 폭식 | 심연의 동굴 | 바알제붑 | 침식 |
| 6 | 색욕 | 타락한 궁전 | 아스모데우스 | 매혹 |
| 7 | 오만 | 신의 폐허 | 루시퍼 | 심판 |

보스를 쓰러뜨릴 때마다, 배신의 밤의 진실이 한 조각씩 드러난다.

> *"나는 말만 했다. 실제로 거래한 놈은 따로 있다."* — 레비아탄의 유언

## 시스템 요약

| 시스템 | 독창적 설계 |
|--------|-----------|
| **죄악 세트** | 장비 접사가 7죄종에 속하고, 2/4/6세트 브레이크포인트로 전투 효과 발동 |
| **코스트 딜레마** | 좋은 장비 = 높은 코스트. 스탯과 장비 사이에서 선택 |
| **타겟 파밍 2축** | 지역 = 죄종 편향, 몬스터 = 장비 부위 편향. 뭘 파밍할지 스스로 설계 |
| **도감 스킬** | 몬스터 카드 수집 → 스킬 해금 → 전투 빌드 확장 |
| **사이즈 상성** | 무기·갑옷 크기 조합에 따라 데미지 보정 (대형 무기 vs 소형 갑옷 = +10%) |
