## 한 줄 콘셉트 (확정)
> "분노를 얻고 탐욕을 버려라 — 7대 죄악으로 빌드하는 파밍 RPG"

## 핵심 차별화 포인트 (확정 — 3축 구조)

> 핵심: 테마(7대 죄악)가 장식이 아니라, 속성·빌드·서사를 관통하는 게임의 뼈대다.

### 테마
7대 죄악이 게임의 모든 것과 연결된다. 전투 속성, 장비 옵션, 빌드 방향, 세계관 — 전부 죄악에서 시작한다. 분노 장비를 모아 분노의 악마를 쓰러뜨린다. 테마가 장식이 아니라, 게임의 뼈대인 RPG.

### 플레이
전투는 자동. 승패는 전투 전에 결정된다. 제한된 코스트 안에서 죄악 세트를 조합하고, 무엇을 포기할지 선택한다.

### 서사
7명의 악마, 하나의 밤. 누가 배신했는가. 보스를 쓰러뜨릴 때마다 답이 한 조각씩 맞춰진다. 진실을 찾기 위해 파밍하는 RPG.
