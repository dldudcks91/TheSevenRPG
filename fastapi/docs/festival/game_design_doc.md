# TheSevenRPG — 게임 기획서

---

## 1. 콘셉트

> "분노를 얻고 탐욕을 버려라 — 7대 죄악으로 빌드하는 파밍 RPG"

### 핵심 차별화

**테마** — 7대 죄악이 게임의 모든 것과 연결된다. 전투 속성, 장비 옵션, 빌드 방향, 세계관 — 전부 죄악에서 시작한다. 분노 장비를 모아 분노의 악마를 쓰러뜨린다. 테마가 장식이 아니라, 게임의 뼈대인 RPG.

**플레이** — 전투는 자동. 승패는 전투 전에 결정된다. 제한된 코스트 안에서 죄악 세트를 조합하고, 무엇을 포기할지 선택한다.

**서사** — 7명의 악마, 하나의 밤. 누가 배신했는가. 보스를 쓰러뜨릴 때마다 답이 한 조각씩 맞춰진다. 진실을 찾기 위해 파밍하는 RPG.

## 2. 게임 개요

| 항목 | 내용 |
|------|------|
| 장르 | 웹 RPG (아이템 파밍 중심) |
| 플랫폼 | 웹 브라우저 (PC/모바일) |
| 타겟 | 빌드 설계와 파밍을 즐기는 하드코어 RPG 유저 |
| 최대 레벨 | 50 |
| 전투 방식 | 1:1 완전 자동전투 |

### 핵심 루프

```
[파밍] 스테이지를 돌며 장비를 모은다
    ↓
[빌드] 죄악 접사를 조합하고, 코스트 한도 내에서 최적의 세트를 만든다
    ↓
[도전] 다음 스테이지 또는 챕터 보스에 도전한다
    ↓
[판단] 실패 시 — 어떤 죄악이 부족한가? 어떤 장비를 파밍할 것인가?
    ↓
    반복
```

### 진행 모드

| 모드 | 설명 |
|------|------|
| 스토리 모드 | 스테이지를 순서대로 클리어. 포탈 귀환 가능 (진행 위치 저장) |
| 던전 모드 | 클리어한 스테이지 재입장. 자동 반복 ON/OFF. 인벤 풀 시 자동 중지 |

- **사망 시**: 처치한 몬스터의 드롭은 유지. 남은 몬스터/보스 드롭 기회 소멸
- **방치 파밍**: 자동 반복 ON → 런 완료 후 즉시 재입장. 인벤토리가 가득 차면 자동 중지되어 자연스러운 접속 주기를 만듦

## 3. 세계관 & 스토리

### 세계 — 아우레아 (Aurea)

**바알**은 지옥의 왕이었다. 자신의 힘을 7개의 감정으로 나누어 부하에게 깃들게 하고, 지옥의 균형을 유지했다. 분노, 시기, 탐욕, 나태, 폭식, 색욕, 오만 — 이 일곱은 바알의 왕권을 떠받치는 기둥이었다.

그러나 그 균형은 깨졌고, 7개의 감정은 세계 각지에 흩어져 **죄악의 화신**이 되었다. 그 땅은 각 죄종의 색채로 물들었다. 아우레아 — 한때 신이 거닐던 이 세계는, 이제 7개의 죄악이 지배하는 땅이다.

### 주인공 — 바알

그 밤, **7개의 칼이 등에 꽂혔다.** 뒤를 돌아봤다. 전부 아는 얼굴이었다.

바알은 추락했다. 힘을 잃고, 기억이 뒤섞인 채, 인간의 몸에 봉인되었다.
몸에 새겨진 **낙인** — 저주이자, 죄악의 힘을 흡수할 수 있는 유일한 무기.

바알은 일어선다. 7명의 악마를 쫓고, 그들의 죄악을 빼앗아 자신을 무장하기 위해.

### 배신의 밤

7명의 악마가 하나의 사건에 관련되어 있다. 그러나 바알의 기억은 불완전하다.

- 바알은 "7개의 칼"로 기억하지만, **실제 배신 참여자는 5명**
- 사탄은 이미 추방 상태였고, 루시퍼는 그 자리에 없었다
- 챕터를 진행할수록 기억의 왜곡이 풀리며 진실이 드러난다

### 배신의 밤 — 인과 관계

**티리엘** — 천상의 존재. 바알의 왕권을 무너뜨리기 위해 배신을 설계한 흑막. 플레이어가 직접 만나는 것은 Ch3 이후.

```
[훨씬 이전] 티리엘이 아스모데우스를 스파이로 심음
    → 바알의 약점 정보를 지속 전달
    ↓
[배신 직전] 티리엘이 바알제붑에게 접근, 충동적 거래 유도
    → 맹약 획득 → 악마들이 자유로워짐
    ↓
[배신의 밤] 자유로워진 악마들에게 선택을 제안
    → 레비아탄: "추락시켜라" 제안
    → 맘몬: 대가를 챙기고 거래 확정
    → 벨페고르: 알면서 방관
    → 루시퍼: 불참 (바알을 살리려 다른 방법을 찾고 있었음)
    ↓
바알 추락 → 낙인이 새겨짐 → 인간의 몸에 봉인
```

### 7악마 — 배신의 밤 역할

| 악마 | 죄악 | 배신의 밤 역할 | 성경 모티프 |
|------|------|--------------|------------|
| 사탄 | 분노 | 참여 안 함 (이미 추방) | 카인과 아벨 |
| 레비아탄 | 시기 | 배신의 말을 꺼냄 | 요셉과 형제들 |
| 맘몬 | 탐욕 | 대가를 챙기고 거래 확정 | 유다의 배신 |
| 벨페고르 | 나태 | 알면서 방관 | 요나의 도주 |
| 바알제붑 | 폭식 | 충동적 거래로 맹약을 넘김 | 에서의 장자권 |
| 아스모데우스 | 색욕 | 처음부터 스파이 | 삼손과 들릴라 |
| 루시퍼 | 오만 | 불참. 바알을 살리려 했음 | 바벨탑 + 루시퍼의 추락 |

### 챕터 간 서사 연결

각 챕터 보스를 쓰러뜨릴 때마다, 다음 악마로 이어지는 단서가 드러난다.

```
Ch1 사탄 → "나는 거기 없었다" → 7개의 칼 중 하나가 빠짐
Ch2 레비아탄 → "나는 말만 했다, 실제로 거래한 놈은 따로 있다" → Ch3 맘몬
Ch3 맘몬 → "다 알면서 눈감아준 놈이 있다" → Ch4 벨페고르
Ch4 벨페고르 → "맹약을 넘긴 놈이 시작이었다" → Ch5 바알제붑
Ch5 바알제붑 → "내가 문을 열었지만 조종한 놈이 있다" → Ch6 아스모데우스
Ch6 아스모데우스 → "마지막 한 명, 루시퍼" → Ch7 루시퍼
Ch7 루시퍼 → "나도 거기 없었다" → 배신의 밤의 진실 완성
```

> 파밍의 동기가 곧 서사의 동기다. 다음 보스를 왜 죽여야 하는지 이유가 있는 RPG.

## 4. 챕터 & 스테이지 구조

### 월드 구조

```
7챕터 × 3스테이지 = 21스테이지
각 챕터 = 스테이지 1~3 + 챕터 보스 (관문)
스테이지 보스 21마리 + 챕터 보스 7마리 = 보스 28종
```

### 7챕터 구성

| 챕터 | 죄악 | 지역 | 챕터 보스 | 상태이상 |
|------|------|------|-----------|---------|
| 1 | 분노 (Wrath) | 불타는 전장 | 사탄 | 화상 |
| 2 | 시기 (Envy) | 뒤틀린 숲 | 레비아탄 | 중독 |
| 3 | 탐욕 (Greed) | 황금의 사막 | 맘몬 | 스턴 |
| 4 | 나태 (Sloth) | 망각의 동토 | 벨페고르 | 빙결 |
| 5 | 폭식 (Gluttony) | 심연의 동굴 | 바알제붑 | 침식 |
| 6 | 색욕 (Lust) | 타락한 궁전 | 아스모데우스 | 매혹 |
| 7 | 오만 (Pride) | 신의 폐허 | 루시퍼 | 심판 |

### 스테이지 런 구조

```
마을 → 스테이지 선택 → 입장 → [일반3+정예1] × 3웨이브 → 보스1 → 마을 귀환
```

- **스테이지당 몬스터**: 13마리 (일반 9 + 정예 3 + 스테이지 보스 1)
- **스테이지 단일 타입**: 1스테이지 = 1몬스터 타입 (Normal / Demon / Undead)
- 몬스터 타입은 장비 접사 `vs_[type]_damage`의 대상 분류 → 타입을 읽고 장비를 맞추는 전략 레이어
- 챕터마다 타입 순서를 다르게 배치 → 패턴 고정 방지
- **스테이지 보스 예시**: Ch1 아바돈(Abaddon) / 레기온(Legion) / 둘라한(Dullahan)

### 챕터 보스

- 3스테이지 전부 클리어 후 도전 가능 (관문)
- 첫 클리어: 다음 챕터 해금 + 챕터 전용 유니크 드롭
- 재도전: 유니크 반복 파밍

### 거점 — 죄악의 성

중앙 허브. 챕터 클리어마다 시설이 해금된다.

| 해금 조건 | 시설 | 기능 |
|----------|------|------|
| Ch1 클리어 | 대장간 | 크래프팅, 강화 |
| Ch2 클리어 | 낙인 공방 | 낙인 삽입/분해 |
| Ch3 클리어 | 상인 | 재료 구매 (골드 소모처) |
| Ch4 클리어 | 흔적 조합소 | 흔적 현황 확인, 슬롯 조정 |

## 5. 전투 시스템

### 전투 방식

- **1:1 완전 자동전투** — 플레이어 개입 없음
- 공격속도 쿨타임마다 자동 공격. HP가 0이 되면 패배
- 승패는 전투 **전** 준비(장비·세트·스탯)로 결정됨

### 카드 스킬 발동

장비 소켓에 카드를 장착하면 전투 중 조건 충족 시 자동 발동.

| 발동 조건 | 설명 |
|-----------|------|
| 공격 시 n% | 내가 공격 액션을 취할 때 |
| 타격(명중) 시 n% | 내 공격이 실제로 명중했을 때 |
| 피격 시 n% | 내가 공격을 맞았을 때 |

### 스탯 체계

**기본 스탯 5종** — 레벨업 시 스탯 포인트 5개 획득 (총 245개)

| 스탯 | 역할 |
|------|------|
| 힘 (Str) | 공격력 증가 |
| 민첩 (Dex) | 공격속도·명중 증가 |
| 체력 (Life) | 최대 HP 증가 |
| 운 (Luck) | 치명타 확률·데미지·회피 증가 |
| 코스트 (Cost) | 장비 장착 한도 증가 |

### 주요 공식

| 세부 스탯 | 공식 |
|-----------|------|
| 최대 HP | `(100 + 체력×10) × (1 + Σ아이템HP%)` |
| 공격력 | `무기기본공격력 × (1 + 힘×0.005) × (1 + Σ아이템공격력%)` |
| 공격속도 | `무기기본공격속도 × (1 + 민첩×0.003) × (1 + Σ아이템공속%)` |
| 방어력 감소율 | `방어력 / (방어력 + 100)` |

### 명중/회피 판정

```
명중 확률 = clamp(Acc/(Acc+Eva) + LvDiff×0.01 + 0.1, 0.05, 0.95)
```

- 동레벨 동스탯 기준 명중률: **60%**
- 레벨 1차이 = ±1%
- 클램프: 5% ~ 95%

### 상태이상 (7종)

각 챕터 테마와 1:1 연동. 특정 빌드를 카운터하는 역할.

| 상태이상 | 챕터 | 효과 |
|---------|------|------|
| 화상 | Ch1 분노 | 매 턴 최대HP 비례 지속 피해 |
| 중독 | Ch2 시기 | 지속 피해 + 회복량 감소 |
| 스턴 | Ch3 탐욕 | 일정 시간 행동 불능 |
| 빙결 | Ch4 나태 | 행동 불능 + 빙결 중 피격 시 추가 피해 |
| 침식 | Ch5 폭식 | 방어력 지속 감소 (스택) |
| 매혹 | Ch6 색욕 | 일정 확률로 자신을 공격 |
| 심판 | Ch7 오만 | HP 일정 이하 시 즉사 판정 |

### 코스트 시스템

장비 착용에 코스트 한도가 필요. 좋은 장비일수록 더 많은 코스트를 소비.

```
플레이어 최대코스트 = (레벨 × 1) + (코스트스탯 × 2)
장비 코스트 = 장비레벨 × 옵션레벨 × 등급배율
```

- 코스트 투자가 많으면 → 강한 장비 착용 가능, 스탯 약함
- 코스트 투자가 적으면 → 스탯 강함, 장비 제한
- **"뭘 포기할 것인가"** 가 빌드의 핵심 딜레마

## 6. 장비 & 빌드

### 장비 슬롯 (5종)

| 슬롯 | 기본 능력치 | 사이즈 구분 | 베이스 수 |
|------|------------|-----------|----------|
| 무기 | 공격력 | 소형/중형/대형 | 15개 (5무기군 × 3사이즈) |
| 갑옷 | 방어력 | 소형/중형/대형 | 6개 |
| 투구 | 방어력+ | 없음 | 6개 |
| 장갑 | 명중률+ | 없음 | 6개 |
| 신발 | 회피율+ | 없음 | 6개 |

> 총 39개 장비 베이스

### 무기군 (5종)

각 무기군은 고유한 Implicit(전투 규칙 변경)을 가짐.

| 무기군 | Implicit | 특징 |
|--------|----------|------|
| 한손검 | 찌르기 (치확 +10%) | 크리티컬 빌드 |
| 양손검 | 관통베기 (방깎 20%) | 방어 무시 |
| 도끼 | 없음 (높은 기본 데미지) | 순수 화력 |
| 창 | 선제 (1타 100% 명중) | 안정적 선타 |
| 둔기 | 경직 (딜레이 +0.1초) | 속도 제어 |

### 사이즈 상성 (무기 vs 갑옷)

| | 대형 갑옷 | 소형 갑옷 |
|---|---------|---------|
| 대형 무기 | +10% 데미지 | -10% 데미지 |
| 소형 무기 | -10% 데미지 | +10% 데미지 |

> 중형은 보정 없음

### 아이템 등급 (4종)

| 등급 | 접사 구성 | 세트포인트 | 핵심 역할 |
|------|-----------|-----------|----------|
| 매직 | 접두 OR 접미 1개 | 1 | 크래프팅 베이스. 저비용 세트포인트 |
| 레어 | 접두 + 접미 2개 | 2 | 메인 파밍 대상. 운이 좋으면 최고 조합 |
| 크래프트 | 접두 + 접미 2개 | 2 | 죄종 카테고리 선택 가능. 통제하는 루트 |
| 유니크 | 고정 1개 | 1 | 고유 효과로 빌드 방향 정의. 보스 전용 |

### 크래프팅

매직 아이템을 베이스로, 비어있는 쪽의 죄종 카테고리를 직접 선택.

```
매직 [접두: 분노] + 재료 + 접미 죄종 선택: [오만]
→ 크래프트 [접두: 분노] + [접미: 오만의 랜덤 옵션]
```

- 기존 옵션 유지 + 선택한 죄종 내 랜덤 옵션 추가
- 레어보다 코스트 높음 (통제권에 대한 프리미엄)
- 크래프트 전용 옵션 존재 (레어/유니크에서 획득 불가)

### 7죄종 & 세트 보너스

접두/접미사 각각이 해당 죄종에 세트포인트 1씩 기여.
5슬롯 기준 최대 10포인트. 죄종 레벨 상한 7.

| 죄종 | 특성 방향 | 2세트 (상태이상) |
|------|-----------|-----------------|
| 분노 (Wrath) | 공격 특화 | 화상 |
| 시기 (Envy) | 디버프·탈취 | 중독 |
| 탐욕 (Greed) | 확률·운 기반 | 스턴 |
| 나태 (Sloth) | 강옵션 + 패널티 | 빙결 |
| 폭식 (Gluttony) | 스택형 스탯 | 침식 |
| 색욕 (Lust) | 버프·흡혈 | 매혹 |
| 오만 (Pride) | 우월함 과시 | 심판 |

### 세트 브레이크포인트

| 포인트 | 보너스 |
|--------|--------|
| 2세트 | 챕터 상태이상 부여 |
| 4세트 | 죄종별 고유 효과 |
| 6세트 | 각성 — 죄종별 궁극 효과 |

### 빌드 전략 갈래

```
7레벨 극대화: 한 죄종 집중 → 각성 확보 + 나머지 3포인트 분배
4+3 분산:    두 죄종의 중간 보너스 동시 활용 (각성 없음)
유니크 활용:  유니크 1~2개 + 세트포인트 희생 → 고유 효과와 세트 조합
```

## 7. 파밍 & 아이템

### 타겟 파밍 (2축)

유저의 파밍 의사결정을 두 축으로 분리.

| 질문 | 결정 축 | 결정 요소 |
|------|---------|-----------|
| "어떤 죄종 접사가 필요하지?" | **지역 (필드)** | 필드마다 특정 죄종 접사 확률 편향 |
| "어떤 장비 부위가 필요하지?" | **몬스터 베이스** | 종족별 장비 부위 드롭 가중치 |

- **지역 → 죄종 편향**: 편향이지 독점은 아님. 테마 죄종 확률 상승, 나머지도 등장
- **몬스터 베이스 → 부위 편향**: 체감 가능한 수준의 편향 설계

### 드롭 등급 분리

| 스폰 등급 | 드롭 |
|-----------|------|
| 일반 몬스터 | 골드, 강화 재료, 카드 (극악 확률) |
| 정예 몬스터 | 장비 드롭 (3회 판정) |
| 보스 | 장비 드롭 + 유니크 가능 |

### 드롭 생성 흐름

```
몬스터 처치 → 드롭 판정 → 부위 결정 (몬스터 베이스 가중치)
                        → 등급 결정 (몬스터 강도 기반)
                        → 접사 결정 (필드 죄종 편향)
                        → 옵션 수치 결정 (몬스터 강도 → 베타분포)
```

- 강한 몬스터일수록 높은 등급·높은 수치의 장비가 생성됨
- 접사 죄종은 필드 테마에 편향되지만 독점은 아님 → 2축 파밍 의사결정과 연동

### 진행 연동 — 던전 티어 → 장비 등급

```
던전 티어 1~3  → 매직 장비만 드롭
던전 티어 4~6  → 레어 드롭 가능
던전 티어 7~9  → 상위 접사 티어 가능
던전 티어 10+  → 크래프트/유니크 드롭 가능
```

상위 던전 클리어가 상위 장비의 전제조건. 파밍 레이어 간 종속성을 만들어 진행감을 강화.

### 카드 & 도감 통합 시스템

- 모든 몬스터는 고유 카드 1종 보유 (48종 = 16베이스 × 3)
- 극악의 확률로 드롭 → **도감에 자동 등록** (인벤토리에 쌓이지 않음)
- 카드 **거래 불가** — 직접 파밍만이 유일한 획득 경로

```
카드 드롭 → 도감에 자동 등록
          → 1장째: 스킬 해금 + 도감 Lv1
          → 2장째~: 도감 경험치 누적 → 도감 레벨업 → 패시브 보너스 강화
```

| 역할 | 설명 |
|------|------|
| 스킬 해금 | 첫 카드 등록 시 액티브 스킬 해금 |
| 도감 레벨업 | 중복 카드 누적 → 패시브 보너스 자동 적용 |

### 스킬 슬롯 (점진적 해금)

| 레벨 | 슬롯 수 |
|------|---------|
| Lv1 | 1슬롯 |
| Lv5 | 2슬롯 |
| Lv15 | 3슬롯 |
| Lv30 | 4슬롯 |

- 장착된 스킬: 적중 시 n% 확률로 자동 발동
