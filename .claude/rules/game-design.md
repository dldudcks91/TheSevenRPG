# 기획 논의 규칙

자세한 가이드는 `game-design` skill 참조.

## 논의 방식
- 분석과 제안만 한다
- "CSV에 반영할까요?", "문서에 추가할까요?" 같은 마무리 질문 금지
- 문서/CSV 수정은 사용자가 명시적으로 요청할 때만 진행

## 확정된 수치 (변경 시 사용자 확인 필요)
- 만렙: **50**
- 명중 공식: `clamp(Acc/(Acc+Eva) + LvDiff×0.01 + 0.1, 0.05, 0.95)`
- 동레벨 동스탯 기준 명중률: **60%**, 레벨 1차이 = ±1%
- 1~4챕터: Lv1→35 스토리 구간 / 5~7챕터: Lv35+ 하드코어 구간

## 챕터-죄악-성경 모티프 매핑 (확정)
| 챕터 | 죄악 | 성경 모티프 |
|------|------|------------|
| 1 | 분노 | 카인과 아벨 |
| 2 | 시기 | 요셉과 형제들 |
| 3 | 탐욕 | 유다의 배신 |
| 4 | 나태 | 요나의 도주 |
| 5 | 폭식 | 에서의 장자권 |
| 6 | 색욕 | 삼손과 들릴라 |
| 7 | 오만 | 바벨탑 + 루시퍼의 추락 |
